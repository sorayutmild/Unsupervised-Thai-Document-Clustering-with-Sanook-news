   สำหรับคนที่อยากเล่นเกม Pokemon Go แต่ไม่อยากลุกออกจากที่นั่งที่นอน ก็ใช่ว่าจะไม่มีวิธี เพราะตอนนี้ เราสามารถหลอก Location บนสมาร์ทโฟนของเราได้ทั้ง iPhone และ Android แม้วิธีการจะซับซ้อนสักหน่อย แต่ก็ไม่เกินความสามารถ แต่ทั้งนี้ การหลอกจับโปเกมอน ด้วยการโกง Location อาจจะทำให้คุณถูกแบนได้ ดังนั้น ถ้าจะใช้วิธีนี้ ต้องใช้อย่างฉลาด

   สิ่งที่คุณจะต้องรู้ในการหลอก Location ของ Pokemon Go ก็คือการ Jailbroken iPhone หรือการ Rooted Android นั่นเอง การทำแบบนี้ทำให้คุณสามารถแตะหรือ Tap เพื่อที่จะเดินไปรอบ ๆ บนจอได้ เนื่องจากเกม Pokemon Go นั้น ใช้ GPSในการบ่งบอกว่า ตัวคุณ ซึ่งเป็นผู้เล่นนั้นอยู่ที่ไหน และบอกคุณว่า มีโปเกมอน อยู่ในบริเวณใกล้ ๆ นั้นหรือไม่ แต่มันจะมีแอพ สำหรับ iPhone และ Andriod ที่ช่วยให้คุณหลอก Location ของตัวเอง ในการใช้พิกัด GPS    วิธีการที่จะบอกต่อไปนี้ จะช่วยให้คุณไม่ต้องวิ่งออกไปหาโปเกมอน แต่สามารถจับสามารถจับมันมาไว้ในคอลเล็กชั่นได้ โดยไม่ต้องลุกออกไปไหน   คุณสามารถใช้ iOS 9.3.3 Jailbreak เพื่อที่จะ Jailbreak ซอฟต์แวร์ ที่จะใช้หลอก Pokemon Go โดยเปลี่ยน Location ของตัวเอง แต่ทั้งนี้ อย่าติดตั้ง iOS 9.3.4 ถ้าคุณต้องการ Hack เกม Pokemon Go คุณสามารถ Jailbreak iPhone ของคุณเอง และติดตั้งแอพเพื่อที่จะปกปิดความจริงที่ว่า คุณได้ Jailbreak เพื่อหลอก Location    สำหรับ Android คุณก็จะต้อง Root โทรศัพท์ และใช้เครื่องมือที่จะช่วยปกปิดว่า โทรศัพท์นั้น ถูก Root    สำหรับสาเหตุ ที่ทำให้ต้องปกปิดการ Jailbreak หรือ Root ก็เพราะ Pokemon Go มีกฏห้ามไม่ให้คุณทำแบบนั้น การเล่นต้องเล่นกันแบบไม่โกง Pokemon Go หมายถึงการเล่นบนอุปกรณ์มือถือ ต้องการให้คุณออกมาสำรวจโลก วิธีการที่ได้บอกไปนั้นถือเป็นการโกง และหากถูกจับได้ ก็จะได้รับการเตือนก่อน หากไม่หยุด ก็จะต้องถูกระงับบัญชีไปสำหรับขั้นตอนดังนี้   1. Jailbreak หรือ Root โทรศัพท์ ซึ่งในขั้นตอนนี้ จะขึ้นอยู่กับซอฟต์แวร์ของโทรศัพท์มือถือด้วย บางรุ่น ก็ไม่สามารถทำได้ แต่โดยทั่ว ๆ ไปแล้ว การ Jailbreak iOS 9.3.3 ของ iPhone และ Root ของ Android รุ่นที่ก่อน Galaxy S7 นั้น สามารถทำได้
   2. ซ่อน Jailbreak หรือ Root เพราะแอพของ Pokemon Go จะตรวจเช็คได้ ในการซ่อน คุณต้องติดตั้ง tsProtector ลงใน iPhone จัดการ Jailbreak ตามคำแนะนำ และดาวน์โหลด Jailbreak Protector ซึ่งจะมีค่าใช้จ่าย 1.99 ดอลล่าร์สหรัฐ สำหรับตัว tsProtector 8+  สำหรับ Andriod ให้ลอง Hide my Root หรืออาจจะหาแอพ หรือ Tweak อื่น ใน XDA
   3. ดาวน์โหลดแอพ Location Faking สำหรับ iPhone และ Android  สำหรับ iPhone นั้น แอพที่ดีที่สุดคือ LocationFaker ซึ่งมีขายที่ Cydia ราคา 2.99 ดอลล่าร์สหรัฐ นอกจากนี้ จะมี PokemonGoAnywhere 
   ซึ่งจะทำให้คุณสามารถ tap หน้าจอเพื่อออกเดินในเกม การจะใช้แอพนี้ จะต้องแอด http://williamcobb.com/repo  เข้าไปที่ Source ใน Cydia จากนั้นก็ค้นหา PokemonGoAnywhere และเมื่อติดตั้งลงไปใน iPhone แล้ว ก็ Restart Springboard เมื่อทุกอย่างเสร็จเรียบร้อย ให้เปิดเกมขึ้นมา คุณจะเห็น Option ในการเดินไปรอบ ๆ แผนที่ ซึ่งจะเดินช้า หรือเดินเร็ว ขึ้นอยู่กับการ Tap 
สำหรับ Android ให้ดาวน์โหลด Fake Location Spoofer Free ลงในเครื่องโทรศัพท์ของคุณ
   4. จับตัวโปเกมอน เมื่อดำเนินการตามขั้นตอนเสร็จแล้ว ก็นั่งจับตัวโปเกมอนได้เลย วิธีการนี้ ได้ผล แต่หากถูกจับได้ก็เท่ากับเสียเงิน และต้องเสียบัญชี Pokemon Go ไปด้วย   นอกจากนี้ยังมีวิธีการหลอก Location โดยไม่ต้อง Jailbreak หรือ Root ได้เช่นกัน แต่จะใช้ได้เพียงไม่กี่ครั้ง สำหรับ Android ให้ใช้ Location Device Only วิธีนี้ จะใช้ GPS ของตัวอุปกรณ์ของคุณเท่านั้น ซึ่งทำให้ Location ของคุณเคลื่อนไปได้ และทำให้ตัวละครในเกมเดินได้ โดยที่คุณไม่ต้องลุกไปไหน วิธีตั้งคือ เข้าไปที่ Settings -> Location -> Mode -> Device only  วิธีการนี้ ทำให้ตัวละคร เดินออกไปไหนต่อไหนได้ เป็นระยะทางประมาณ 1.7 กิโลเมตร แต่เมื่อจัดการจับโปเกมอนเสร็จแล้ว ให้เปลี่ยนกลับมาตามเดิม     สำหรับ iPhone คุณไม่สามารถปรับเปลี่ยน Setting ได้ แต่ถ้าคุณกดปุ่ม Home และใช้แอพอื่นสักสองสามนาที จากนั้น ก็ให้กลับมาที่เกมอีกครั้ง วิธีนี้ จะทำให้ตัวละครของคุณวิ้งไปรอบ ๆ ได้เช่นกัน แม้จะไม่ดีเท่า Andriod แต่ก็ช่วยให้แอบจับโปเกมอนได้บ้าง ในเวลาที่ไม่สะดวกจะออกไปนอกสถานที่
