ผู้เล่น Pokemon GO เพลีย! เจอบั๊กวางไข่กัน Gym ทำให้ตีไม่ได้
Pokemon GO
ยิ่งเป็นเกมฮิตระดับโลก ก็ยิ่งทำให้คนเล่นเจอช่องโหว่ของเกมมากขึ้น นั่นทำให้ตอนนี้ Pokemon GO มีการพบบั๊กและช่องโหว่ของเกมกันเพียบ เป็นปัญหาที่ทำให้ทาง Niantic ปวดหัวมากในตอนนี้ ไม่ว่าจะเป็นบั๊กปั๊มไอเทม ใช้ไอเทมได้ไม่มีวันหมด เพิ่มเงินในเกม และอื่นๆ และที่เป็นบั๊กตัวปัญหาอย่างหนึ่งในตอนนี้ก็คือ บั๊กวางไข่กัน Gym ที่ทำให้ผู้เล่นคนอื่นๆไม่สามารถเข้าไปโจมตี Gym นั้นๆได้
ปัญหาบั๊กไข่ หรือ Egg glitch พบครั้งแรกเมื่อช่วงสุดสัปดาห์ที่ผ่านมา และเริ่มมีคนใช้กันมากขึ้น วิธีก็คือปกติแล้วการป้องกัน Gym นั้นผู้เล่นที่ยึดครองจะต้องเลือกโปเกม่อนมาตัวหนึ่ง ให้ทำหน้าที่ปกป้อง Gym ซึ่งก็ต้องเลือกตัวที่อึดๆเก่งๆ CP เยอะๆ แต่การใช้ไข่มาป้องกันนั้น มันมี CP0 จึงทำให้เหมือนเป็นบั๊กและผู้เล่นคนอื่นๆไม่สามารถตี Gym ได้นั่นเอง โดยทางผู้เล่นได้พบช่องโหว่ว่า Egg ไข่โปเกม่อนนั้นก็มี ID Number เหมือนกับโปเกม่อนตัวอื่นๆ มันจึงมีช่องให้ใช้นำมาวางใน Gym ได้ โดยการใช้โปรแกรมอื่นช่วยนิดหน่อย อย่างเช่นโปรแกรม Python-based API จาก Github หรือพวก Android emulator จากนั้นก็ใส่สคริปต์เพิ่มไป ทำให้เลือกไข่มาวางใน Gym ได้

การทำเช่นนี้ถือเป็นการโกงอย่างหนึ่ง เพราะผู้เล่นที่ยึดครอง Gym ก็จะได้รับ Pokecoins หรือเงินในเกมเพิ่มขึ้นไปเรื่อยๆโดยไม่ต้องกลัวว่าจะมีใครมาตี Gym ได้ ทาง Niantic ก็ยังไม่ได้ออกมาแก้ปัญหานี้ ผู้เล่นจึงทำได้แค่ Report ผู้เล่นที่โกงแจ้งให้ทาง Niantic ทราบ
