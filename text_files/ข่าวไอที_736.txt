    สำหรับเรื่องของเทคโนโลยีการโทรออกผ่าน WiFi หรือคนทั่วไปจะรู้จักคำว่า WiFi Calling ซึ่งเป็นเทคโนโลยีใหม่ล่าสุดที่นำสัญญาณ WiFi มาเป็นคลื่นโทรศัพท์เพื่อโทรออกหรือรับสายได้นั่นเอง โดยตอนนี้ทั้ง 3 ค่ายเปิดให้บริการแล้ว ไหน ๆ ก็มีให้ใช้งาน ทีม Sanook! Hitech เห็นว่ามาดูกันว่าแต่ละค่ายมีเงื่อนไขอะไรบ้าง และประโยชน์ของเทคโนโลยีนี้
WiFi Calling คืออะไร

     WiFi Calling หรืออีกชื่อคือ VoWiFi (Voice Over WiFi) เป็นการคลื่นสัญญาณ WiFi ที่คุณต่อจากอุปกรณ์เชื่อมต่อไร้สาย ไม่ว่าอยู่ที่ทำงาน, ที่บ้าน, หรือจาก Pocket WiFi ในทุกมุมโลก คุณสามารถโทรออกได้เลย ขอแค่คุณใส่ซิมเบอร์ของคุณใส่ในเครื่องที่รองรับ คุณก็สามารถใช้บริการผ่านเบอร์มือถือปกติของคุณได้เลย
    ทั้งนี้ตัวเครื่องจะต้องรองรับให้บริการนี้ด้วย ซึ่งปัจจุบันมีเครื่องที่รองรับบริการนี้ยังไม่มากสักเท่าไหร่
ข้อดีของเทคโนโลยี WiFi Calling
    จุดเด่นของ WiFi Calling คือไม่ว่าคุณจะใช้โทรออกคุยสนทนา หรือ Video Call รวมถึงบางค่ายรองรับการส่ง SMS จะทำได้เหมือนกับใช้งานสัญญาณมือถือทั่วไป และยิ่งใช้งานในต่างประเทศ จะเหมือนกับการใช้งานในประเทศ เพราะคิดค่าบริการไม่ต่างกัน
    รวมไปถึงการเชื่อมต่อสัญญาณที่ติดง่ายขึ้นและคมชัดมากขึ้นอีกด้วย เหมาะกับจุดที่สัญญาณมือถือเข้าไม่ถึง คุณใช้สัญญาณ WiFi แทนคลื่นมือถือได้เลย
ความแตกต่างระหว่าง WiFi Calling กับ Roaming
ในส่วนของเรื่องความแตกต่างนั้นการใช้บริการ Roaming คือการไปใช้สัญญาณในต่างประเทศ ซึ่งผู้ให้บริการได้ไปทำสัญญาใช้บริการเอาไว้

ข้อดีคือ คุณสามารถใช้งานมือถือได้คมชัดและเหมือนกับอยู่ในประเทศได้เลย
ข้อเสียคือค่าบริการ ไม่ว่าจะโทรออกภายในประเทศ หรือ รับสายจากประเทศไทย ล้วนแต่มีค่าใช้จ่ายที่สูงพอสมควรเลยทีเดียว และถ้าไม่เปิดจะไม่สามารถใช้งานโทรออกหรือรับสายในต่างประเทศได้

กรณีนี้ WiFi Calling แค่เราหาสัญญาณ WiFi Hot Spot หรือ คุณเช่า Pocket WiFi ไปในต่างประเทศ ต่อเชื่อมแล้วคุณก็สามารถใช้เครื่องโทรออก หรือรับสายจากในประเทศไทยได้ ซึ่งมีค่าบริการที่ถูกกว่า และสัญญาณคมชัดกว่าเพราะไม่ต้องแออัดกับเครือข่ายมือถือที่แต่ละประเทศไม่เหมือนกัน แต่ต้องให้ WiFi ตัวนั้นสามารถออกอินเทอร์เน็ตได้ถึงจะโทรออกได้นะครับ

การคิดค่าบริการของทั้ง 3 ค่ายไม่ได้แตกต่างกัน โดยมีเงื่อนไขดังนี้

การโทรในประเทศ จะคิดค่าบริการตามโปรโมชั่นเหมือนกับโทรผ่านเครือข่าย โดยรับสายจะไม่เสียเงิน
การใช้งานในต่างประเทศ กรณีที่ใช้โทรกลับไทย จะคิดค่าบริการตามโปรโมชั่นเดิมของคุณ
ในกรณีใช้งานต่างประเทศ แล้วรับสายจากประเทศไทย คุณจะไม่เสียค่าบริการใด ๆ
แต่กรณีที่คุณอยู่ในประเทศหรือต่างประเทศ โทรไปยังประเทศอื่นไม่ใช่ประเทศไทย จะคิดอัตรา Roaming ตามปกติ

    สรุปสั้น ๆ คือ ถ้าโทรเข้าออกของเบอร์ในประเทศไทยไม่ว่าจะอยู่ในประเทศหรือต่างประเทศค่าโทรจะเหมือนกับคุณใช้งานในประเทศ แต่ถ้าคุณใช้โทรไปยังต่างประเทศไม่ว่าจะอยู่ในไทยหรือต่างประเทศก็จะคิดค่าบริการเหมือนการ Roaming ทุกประการ
    ดังนั้นจะใช้โทรออกเบอร์ที่ไม่ใช่ +66 คิดให้ดีและดูแต่ละค่ายว่าคิดค่าบริการเท่าไหร่
ความแตกต่างของผู้ให้บริการ
    แน่นอนว่าเรื่องความแตกต่างของผู้ให้บริการก็จะอยู่ที่เงื่อนไขและการรองรับอุปกรณ์ต่าง ๆ ที่ตอนนี้เรียกได้ว่า แต่ละค่ายก็มีมาตรฐานที่แตกต่างกัน โดยมีสิ่งที่ต้องรู้ดังนี้

   เริ่มจาก AIS เป็นผู้เปิดบริการ WiFi Calling รายหลังสุด จะใช้งานเทคโนโลยีนี้ได้เฉพาะ มือถือ iPhone ตั้งแต่รุ่น 5c ขึ้นไป โดยต้องอัปเดท iOS เป็นเวอร์ชั่น 9.3.2 ขึ้นไป และ อัปเดทสัญญาณให้เป็น AIS 24.2 ถึงจะใช้งานได้

Dtac เรียกได้ว่าเป็นผู้เปิดให้บริการ WiFi Calling เป็นรายแรกของประเทศไทย ตอนนี้รองรับอุปกรณ์มากที่สุด ซึ่งประกอบไปด้วย

    ส่วน Truemove H นั้นจะต้องใข้งานผ่าน Application WiFi Calling ที่สามารถโหลดได้ทั้ง iOS และ Android จุดดืคือไม่ว่าจะเป็นเครื่องไหนก็สามารถใช้งานได้เลย แต่มีข้อจำกัดที่คุณจะต้องมี True ID ถึงจะใช้งาน Apps นี้ได้ แม้อาจจะไม่สะดวกสักนิด แต่ถ้าไม่อยากตั้งค่าให้วุ่น Apps ก็เป็นทางเลือกที่ดี
    สรุปสั้น ๆ เทคโนโลยี WiFi Calling เป็นอีกสิ่งที่อำนวยความสะดวกและช่วยให้คุณไม่พลาดการติดต่อ แม้ว่าจุดที่คุณอยู่จะมีสัญญาณมือถือน้อย หรือการโทรออกในต่างประเทศ กรณีที่ไม่ได้เปิด Roaming ไว้ก็ตาม แต่อย่างไรก็ดีการศีกษาข้อมูลและรู้ผลเสียเกี่ยวกับการโทรข้ามประเทศจะช่วยให้การบริหารจัดการค่าโทรนั้นไม่ต้องจ่ายแพงจนเกินไปแน่นอนครับ
