มันแปลกดีนะ...Facebook เดินเรื่องขึ้นศาลสั่งฟ้อง Mark Zuckerberg!Facebook เว็บไซต์สังคมออนไลน์เบอร์หนึ่งของโลกเตรียมเดินเรื่องสั่งฟ้อง Mark Zuckerberg ในชั้นศาลอย่างถึงที่สุดชนิดที่แฟนๆ TechXcite อาจจะไม่เคยคาดคิดมาก่อน อย่างไรก็ตามนาย Mark Zuckerberg คนที่ว่านี้เราไม่ได้หมายถึงคุณน้องมาร์คผู้ก่อตั้ง Facebook แต่อย่างใด ทว่าเรากำลังพูดถึงกระทาชายชาวอิสราเอลรายหนึ่งที่เปลี่ยนชื่อตัวเองเป็น Mark Zuckerberg ทันทีที่ Facebook สั่งฟ้องเขาและบริษัทเสียอย่างนั้น!

เรื่องของเรื่องมีอยู่ว่าคุณพี่ Mark Zuckerberg คนที่ 2 ที่เรากำลังพูดถึงกันนี้แต่เดิมคือนาย Rotem Guez ชาวอิสราเอลที่ได้เปิดบริษัทชื่อว่า Like Store (แค่ชื่อก็หมิ่นเหม่แล้วพ่อคุณ) ซึ่งเป็นบริษัทการตลาดออนไลน์ซึ่งจะจำหน่ายคะแนนการกด Like บน Facebook ให้กับบรรดาบริษัทต่างๆเพื่อช่วยเพิ่มปริมาณการกด Like ให้กับหน้าผลิตภัณฑ์หรือ Brand Page เหล่านั้นนั่นเอง พูดง่ายๆคือหน้าที่ของนาย Rotem ก็คือการไประดมคนมาช่วยกด Like ให้กับหน้า Facebook ของสินค้าแต่ละแบรนด์นั่นเอง


อย่างไรก็ตามหลังจากนั้นไม่นานทางด้านของบริษัท Facebook ที่ไปได้ยินข่าวของนาย Rotem เข้าก็ได้เริ่มส่งจดหมายเตือนให้นาย Rutem ปิดบริษัท Like Store ของตัวเองไปซะเพราะถือว่าการทำงานของ Like Store นั้นขัดแย้งกับกฏการใช้งานของ Facebook อยู่พอสมควร เมื่อเห็นดังนั้นแล้วนาย Rotem ซึ่งเพิ่งจะถูกทาง Facebook สั่งปิดหน้า Profile ส่วนตัวไปแล้วด้วยจึงตัดสินใจที่จะเปลี่ยนชื่อของตัวเองให้กลายเป็น Mark Zuckerberg เพื่อที่ว่าหาก Facebook ตัดสินใจจะฟ้องเขาแล้วนั่นก็จะหมายความว่า Facebook สั่งฟ้องประธานบริษัทของตัวเองแบบกลายๆไปด้วยเลย...เอากับพ่อเจ้าประคุณรุนช่องเขาสิ

งานนี้ก็เลยยังไม่รู้ว่าคดีความระหว่างนาย Mark Zuckerberg ตัวจริงจากบริษัท Facebook กับนาย Mark Zuckerberg ตัวปลอมจาก Like Store จะไปจบลงตรงไหนกันแน่เหมือนกัน แต่ TechXcite มั่นใจว่า Mark Zuckerberg (คนไหนไม่รู้แหละ) จะต้องเป็นฝ่ายชนะแน่นอน 555+
