     เทียบทุกช็อต Honda Accord 2019 ใหม่ และ Toyota Camry 2019 ใหม่ คันไหนจะสวยเด็ดกว่ากัน
     หลังจากที่ Toyota Camry 2019 ใหม่ ถูกเปิดตัวอย่างเป็นทางไปแล้วก่อนหน้านี้ ล่าสุด ฮอนด้าประเทศไทยก็เตรียมส่ง Accord 2019 ใหม่ มาดักทางที่งานมอเตอร์เอ็กซ์โปปลายเดือนพฤศจิกายนนี้ เราลองไปเทียบกันมุมต่อมุมของทั้งคู่ ว่าในด้านดีไซน์คันไหนสวยลงตัวกว่ากัน
 
ดีไซน์ด้านหน้า

     ดีไซน์ของ Accord 2019 ใหม่ ยังคงเน้นความสปอร์ตมากกว่า Camry 2019 โดยมีดีไซน์ไฟหน้าและกระจังหน้าคล้ายกับ Civic โฉมปัจจุบัน แต่ถูกออกแบบให้ลดความโฉบเฉี่ยวลง ขณะที่ Camry 2019 ก็เน้นกระจังหน้าสีดำขนาดใหญ่พร้อมกับไฟหน้าดีไซน์เฉี่ยว
 
ดีไซน์ด้านหลัง

     ด้านท้ายของ Accord 2019 ถูกออกแบบไฟท้ายให้เป็นรูปตัว C ดูแปลกตากว่ารุ่นที่แล้ว มาพร้อมกระจกโอเปร่าที่ช่วยให้รถดูยาว พร้อมกับแนวหลังคาที่ลาดเทไปทางด้านหลังมากกว่า ส่วน Camry 2019 ก็ดูมีดีไซน์ฉีกไปจากรุ่นที่แล้วพอสมควรเช่นกัน
 
ดีไซน์ด้านข้าง

     ด้านข้างของ Accord 2019 จะเห็นได้ว่ามีแนวหลังคาลาดเทไปทางด้านหลังคล้ายกับรถคูเป้มากกว่า ทำให้ดีไซน์ของแอคคอร์ดดูมีความเป็นสปอร์ตเพิ่มขึ้น ขณะที่ Camry เน้นความหรูหราด้วยเสา C-pillar ขนาดใหญ่ พร้อมกับแนวฝากระโปรงที่เชิดขึ้นช่วยให้ดูภูมิฐานมากกว่า
 
แผงแดชบอร์ด

     ภายในห้องโดยสารของ Accord 2019 ยังเน้นความอนุรักษ์นิยมมากกว่า Camry 2019 แต่แฝงด้วยความทันสมัย เช่น หน้าจออินโฟเทนเม้นท์แบบลอยตัว, ปุ่มเปลี่ยนเกียร์ไฟฟ้า เป็นต้น
 
ระบบอินโฟเทนเม้นท์

     หน้าจออินโฟเทนเม้นท์ของ Accord 2019 ถูกออกแบบให้สูงในระดับเดียวกับสายตา ช่วยให้มองเห็นขณะขับขี่ได้ง่ายกว่า ขณะที่ Camry ถูกออกแบบให้กลืนไปกับแผงแดชบอร์ด ตกแต่งด้วยปุ่มสีเงินเพิ่มความหรูหรา
     ทั้งนี้ Honda Accord 2019 ใหม่ เตรียมอวดโฉมในงาน Motor Expo 2018 ที่จะถึงนี้ หากใครกำลังเล็ง D-Segment คันใหม่อยู่แล้วล่ะก็ห้ามพลาดครับ
