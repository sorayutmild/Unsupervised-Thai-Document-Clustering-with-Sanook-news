สำหรับ การเล่นเกมของใครหลายคนนั้น  แน่นอนว่าล้วนแต่มีเหตุผลที่ไม่เหมือนกันในแต่ละคน แต่สิ่งหนึ่งที่เหมือนๆ  กันก็คือเล่นเกมนั้นให้จบ แต่บางเกมนั้นก็โคตรจะยากแสนสาหัส  กว่าจะจบก็เลือดตาแทบกระเด็น วันนี้ผมก็มี 5 เกม Console  แดนปลาดิบสุดแสนยากมาให้ชมกันจ้า จะมีอะไรบ้างไปดูกันเลย   ซีรี่ย์ Monster Hunter  
 
game





        มาดูเกมกันก่อนเลย ก็คือเจ้าซีรี่ย์ล่าแย้ Monster Hunter นั่นเอง  แค่ช่วงแรกๆ ก็รู้สึกว่ายากใช้ได้แล้วกับการที่ต้องเอาดาบทื่อๆ  ไปไล่เฉาะมอนสเตอร์ แต่ตอนท้ายๆ ที่เป็นเควสต์ระดับ G Rank ในบาร์แล้ว   เรียกว่าโคตรโหดกันเลยทีเดียว โดนทุบทีแทบม่อง  เล่นกันหลายคนอาจจะไม่เท่าไหร่ แต่เล่นคนเดียวนี่เสียวซี๊ดสุดๆ (ยกเว้นภาค  Portable 3rd ที่ทำมาค่อนข้างง่ายเมื่อเทียบกับภาคอื่นๆ ) 
 Mushi Hime Sama Futari 
 
game





เกมยานยิงแนวตั้งกระสุนท่วมจอหรือที่เรียกกันว่า Danmaku Shooting  นั้นก็เรียกได้ว่ามีหลายเกมเลยทีเดียวที่ทำออกมาให้ยอดมนุษย์ได้เล่นกัน  ซึ่งค่ายที่ขึ้นชื่อว่าโหดสัสจริงๆ เลยก็คือค่าย Cave นั่นเอง  ซึ่งก็มีอยู่หลายเกมแต่เกมที่ถูกพูดถึงมากที่สุดก็คือ Mushi Hime Sama  ซึ่งเป็นเกม Arcade มาก่อนแล้วพอร์ตลง Xbox 360 แต่เห็นกระสุนก็สยองแล้ว  เหอะๆ 
  Ninja Gaiden 
 
game





เกม Action ยอดนิยมที่แฟนเกมรุ่นเก่าหลายๆ คนต้องรู้จัก สำหรับเจ้า  Ninja Gaiden ภาคแรก ดูเผินๆ เหมือนจะไม่ค่อยมีอะไรเท่าไหร่  แต่พอเล่นไปได้ซักพักก็จะเริ่มเข้าใจว่าเล่นยังไงก็ไม่จบซักกะที  ม่องเท่งกันรัวๆ ด้วยเหตุนี้เอง ทำให้ Ninja Gaiden  นั้นยังเป็นซีรี่ย์ที่คงความยากมาจนถึงทุกวันนี้  
 Dark Souls 
 
game





ภาคต่อของเกม Open World Action RPG จาก  From Software  ที่ได้รับความนิยมเป็นอย่างมากในเหล่านักเล่นเกมคอนโซล  และคว้ารางวัลยอดเยี่ยมมาหลายสำนัก นอกจากเนื้อเรื่อง,  ระบบเกมและธีมเกมที่ถูกใจเกมเมอร์แล้ว  ยังมีความยากระดับโหดที่ไม่เป็นรองใครอีกด้วย  ใครจะคิดว่าเริ่มเกมมาฉากแรกก็ต้องวิ่งหนีศัตรูกันแล้วเพราะสู้ไม่ได้  ม่องแล้วม่องอีกจนแสนละเหี่ยใจ
  Dai Makai Mura 
 
game





 
      เรียกว่าโคตรยากปราบเซียนมาตั้งแต่สมัยยุคบุกเบิกเลยก็คือ Dai Makai  Mura จาก Capcom หลายคนอาจจะไม่คุ้นชื่อ  แต่ถ้าบอกว่าเป็นเกมอัศวินปาหอกก็จะนึกออกแน่นอน สำหรับภาค Dai หรือภาค 2  นี้เป็นภาคที่ยากสุดติ่งสุดๆ ไม่ว่าจะเป็นกับดักต่างๆ สุดโหด ไหนจะเจอเจ้า  Red Arimer และต้องจบถึง 2 รอบด้วยกันถึงจะเป็นฉากจบที่แท้จริง
