เกร็ดไม่เล็กไม่น้อย 10 ข้อที่ควรรู้ก่อนตัดสินใจซื้อ Notebook
เชื่อว่าช่วงนี้มีหลายๆคนอาจจะตัดสินใจซื้อ notebook ใหม่  เนื่องด้วยหลายๆสาเหตุ ไม่ว่าจะเป็นเสียหายจากน้ำท่วม หรือซื้อรับปีใหม่  หรือซื้อก่อนที่ราคาอาจจะแพงขึ้นเนื่องด้วยการขาดแคลน harddisk ดังนั้น  Notebookspec  เราจึงหาบทความเกี่ยวกับเกร็ดความรู้ที่จะมาช่วยในการตัดสินใจของทุกๆท่านมา ฝากกันครับ ซึ่งก็มี 10 ข้อด้วยกัน เราไปเริ่มกันเลยดีกว่า
1. จะ Windows หรือ Mac ดี
macvwindowsอันนี้เรียกได้ว่าเป็นประเด็นหลักในการแบ่งค่ายเลยทีเดียว เพราะทั้งรูปแบบและ  hardware ก็แตกต่างกันแทบจะสิ้นเชิง เนื่องด้วยทั้งเครื่องและ OS  ที่แตกต่างกัน  โดยสิ่งที่ต่างกันอย่างเห็นได้ชัดก็คือเรื่องของประสบการณ์การใช้งาน  ที่ต้องยอมรับว่า Mac ตอบสนองและสร้างความพึงพอใจกับผู้ใช้ได้ดีกว่า  โดยเฉพาะหน้าตาของ OS และโปรแกรมที่ดูจะหน้าตาดีกว่าใน Windows
ส่วน Windows ก็มีจุดเด่นมากๆที่มี hardware และ software  จากค่ายอื่นรองรับมากมาย  โดยเฉพาะอย่างยิ่งเกมที่มีเป็นล้านๆเกมให้เราได้หามาเล่นกัน  เอาเป็นว่าทั้งคู่ต่างก็มีข้อดีของมัน  อยู่ที่ใครจะนำข้อดีเหล่านั้นมาใช้ให้เกิดประโยชน์กับตัวเองให้มากที่สุด  ถ้าคิดว่าอันไหนมันเหมาะกับเรา ก็ซื้ออันนั้นเลยครับ  จะได้ถูกใจไปตลอดการใช้งาน
2. แรม 4 GB ก็พอแล้วววว
ddr3_00ถ้าเรามองจากสเปกของ notebook ราคากลางๆขึ้นไปแต่ละเครื่อง  จะพบว่าแรมมักจะให้มาที่ 4 GB กันแทบทั้งนั้น  น่าจะเนื่องมาจากความพอดีกับการใช้งานเป็นหลัก  (ไม่แน่อาจจะเป็นการผลักดันให้ผู้ใช้เปลี่ยนไปใช้ Windows แบบ 64-bit  ด้วยก็เป็นได้) ซึ่งถ้าเราจะซื้อมาใช้งานทั่วๆไป แนะนำว่าแค่ 4 GB  ก็เหลือเฟือแล้วครับ สามารถใช้ทำงานได้อย่างสบายๆเลย  แต่กับคนที่ต้องใช้เครื่องเพื่อรันระบบเสมือนเช่นพวก VMWare  ก็คงต้องจัดหนักกันซักหน่อย
ซึ่งการที่เพิ่มแรมให้มีปริมาณมากๆโดยไม่จำเป็นนั้น  นอกจากจะส่งผลโดยตรงกับเงินในกระเป๋าแล้ว  ยังส่งผลกับระยะเวลาการใช้งานแบตเตอรี่ที่สั้นลงอีกด้วย  เพราะไฟต้องไปหล่อเลี้ยงกับหน่วยความจำของแรมทุกๆ bit  ตลอดระยะเวลาการใช้งาน ดังนั้น มีเท่าที่จำเป็นก็พอจ้า
3. ความจุ HDD อาจไม่สำคัญเท่าความเร็วก็เป็นได้
bare_magic_trans_backgrnd-1ผู้ใช้ส่วนใหญ่มักจะมองหา notebook เครื่องที่มีความจุของ HDD ให้มากๆเข้าไว้  เผื่อเอาไว้เก็บนู่นนั่นนี่มากมาย  แต่มีน้อยคนที่จะคำนึงถึงความเร็วของแหล่งเก็บข้อมูลอย่าง HDD  ถ้าให้คิดง่ายๆระหว่างเก็บข้อมูลได้เยอะๆ แต่เรียกข้อมูลมาใช้ได้ช้า  กับเก็บข้อมูลได้มากพอตัวแต่เรียกมาใช้ได้ไว  เชื่อว่าคนอยากได้ในข้อหลังก็มีไม่น้อยทีเดียว  เพราะถ้าต้องการเก็บข้อมูลมากๆ ส่วนตัวคิดว่าซื้อ External HDD  มาดีกว่าครับ อย่าเก็บข้อมูลหลักๆ เยอะๆไว้ใน notebook ดีกว่า  โอกาสเสี่ยงต่างๆมันเยอะใช่เล่นเลยทีเดียว
ซึ่ง HDD ที่น่าจะเริ่มมองหาก็คงเป็นพวกที่มีความเร็วระดับ 7200 RPM  หรือถ้าเงินถึงจริงๆก็ไปเล่นพวก SSD หรือ HDD แบบ Hybrid อย่าง Seagate  Momentus XT ไปเลยก็ได้ครับ  พวกนี้ช่วยเพิ่มความแรงให้เครื่องอย่างเห็นได้ชัด  แถมโอกาสเสี่ยงที่แหล่งเก็วข้อมูลจะพังจาก Bad Sector ก็มีน้อยกว่า HDD  มากทีเดียว
4. เหล่า Pentium มันหมดยุคไปแล้ว…ไปหาพวก Core i เถอะ
comp_cpu1296567698ก็ยังคงอยู่มาได้นะครับ สำหรับ CPU ในตระกูล Pentium ที่เราคุ้นตากันมานาน  ก่อนจะมาถึงยุคของ Core i ที่บูมขึ้นมา พร้อมทั้งเทคโนโลยีใหม่ๆอีกเพียบ  โดยในกลุ่มของ notebook ที่ราคาไม่สูงนัก เราก็จะเห็นเครื่องที่ใช้ CPU  เป็นพวก Pentium กันมากพอสมควร ซึ่งถ้าจะให้พูดตรงๆ จริงๆแล้ว ถ้าเป็นไปได้  เก็บเงินอีกหน่อยกัดฟันซื้อพวก Core i ดีกว่าครับ  ถึงแม้ว่าการนำมาใช้ทำงานทั่วไปมันจะยังทำได้ดีอยู่ก็ตาม แต่สิ่งที่ตามหลัง  CPU เทคโนโลยีใหม่อยู่มากก็คือเรื่องการใช้พลังงาน  โดยเฉพาะระยะเวลาการใช้งานแบตเตอรี่ที่ Core i ทำได้ดีกว่าอย่างเห็นได้ชัด
หรือถ้าจำกัดเรื่องงบจริงๆ ลองย้ายค่ายไป AMD บ้างก็ได้นะครับ ราคาพอกัน แต่ประสิทธิภาพและการใช้พลังงานน่าพอใจกว่า Pentium เยอะ
5. หันไปหา AMD บ้างก็ได้…”บางอย่าง” มันดีกว่า Intel อีกนะ
AMD-Ontario-APUในช่วงปีที่ผ่านมา AMD ทำการผลิกโฉมวงการ CPU บน notebook ไปพอสมควรทีเดียว  กับการเปิดตัว APU จากแพลตฟอร์ม Fusion  ที่มีจุดเด่นในเรื่องของการ์ดจอในตัวและประสิทธิภาพที่น่าพอใจเมื่อเทียบกับ ราคา  รวมไปถึงระยะเวลาการใช้งานแบตเตอรี่ที่ทำได้ดีอย่างกับพลิกหน้ามือเป็นหลัง มือเลย โดยเฉพาะเมื่อเทียบกับเหล่า Pentium ในข้อ 4
6. keyboard กับ touchpad ก็สำคัญนะจ๊ะ
Review Lenovo ThinkPad Edge E125 16การจะมองหา notebook จะให้มองแต่สเปกก็คงไม่ดีเท่าไร  เพราะเราควรจะมององค์ประกอบโดยรวมของเครื่องด้วย โดยเฉพาะ keyboard และ  touchpad ที่เราต้องใช้เป็นส่วน input เข้าไปหาสเปกแรงๆภายใน  ดังนั้นจึงควรจะพิถีพิถันซักเล็กน้อย  ซึ่งจุดที่เราควรทดสอบให้ดีๆก็อย่างเช่นการเด้งของปุ่มต่างๆ ลักษณะของปุ่ม  การจัดวางรูปแบบต่างๆ และอาจดูถึงฟังก์ชันเสริมอย่างพวกไฟ LED  ใต้ปุ่มด้วยก็เป็นได้ ถ้าเราต้องใช้งานในที่ที่มีแสงน้อยเป็นประจำๆ
ส่วน touchpad ก็ต้องดูพวกที่สามารถตอบสนองการใช้งานได้ดี ไม่แล็ก  รวมไปถึงผิวสัมผัสที่ไม่หนืดเกินไป ส่วนฟังกฺชันพวก multi gesture  นั้นก็แทบจะเป็นฟังกฺชันหลักของ touchpad ในปัจจุบันไปเรียบร้อยแล้ว  และอีกส่วนหนึ่งที่หลายๆคนต้องการก็คือปุ่มคลิกซ้าย-ขวาที่ไม่มีเสียง  ซึ่งทั้งหมดทั้งมวลเหล่านี้ทางที่ดีที่สุดคือไปทดลองด้วยตนเองครับ
7. จอกระจก – เลี่ยงได้ก็ดีmacbook-matte-screen1
notebook สมัยนี้กว่า 90% ใช้จอแบบที่มีแสงสะท้อนจากข้างหลังได้  หรือที่เราเรียกว่าจอกระจก ซึ่งข้อดีมันก็มีอยู่  คือทำให้ตัวเครื่องดูใหม่น่าใช้งานและสีสันที่ได้จากจอสวยงาม  คมชัดสมจริงกว่าจอด้าน รวมไปถึงราคาต้นทุนที่ถูกกว่าจอด้าน  (ถ้าจะทำให้จอด้านมีคุณสมบัติด้านสีเทียบเท่ากัน จะต้องใช้ต้นทุนที่สูงมาก)  แต่ก็มีข้อเสียที่ส่งผลเสียต่อสุขภาพได้  ก็คือมันสะท้อนแสงจากข้างหลังผู้ใช้ได้ดีมากๆ  ทำให้เกิดแสงสะท้อนเข้ามาที่ตาเรา  สายตาจึงต้องปรับการทำงานให้สอดคล้องกับสภาพแสงที่มากเช่นนั้นตลอดเวลา  จึงอาจเกิดปัญหากับสายตาได้
 ดังนั้นถ้าเป็นไปได้  แนะนำว่าพยายามหาซื้อรุ่นที่เป็นจอด้านก็จะดีมากครับ แต่อาจจะหายากซักหน่อย  แถมพวกเครื่องสั่งประกอบที่มักจะมีจอด้านให้เลือกก็ไม่บูมในไทยซะด้วย  หรืออีกหนทางหนึ่งสำหรับคนที่ต้องทำงานหน้าจอนานๆ  แนะนำว่าหาซื้อจอแยกที่เป็นจอด้านมาต่อเพิ่มจะดีกว่าครับ ใหญ่กว่า  สบายตากว่าเดิมเยอะจริงๆ
8. ถ้าไม่จำเป็นก็ไม่ต้องซื้อเครื่องแรงๆสำหรับเล่นเกมก็ได้ (ตอนนี้คงยังไม่เหมาะกับในไทยนัก)
Onlive-Logoจากก่อนหน้านี้ที่เวลาเราจะต้องเล่นเกมใน notebook  เราจำต้องมองหาเครื่องสเปกแรงๆ ราคาแพงๆ กัน ต่อมาในปัจจุบัน  เครื่องที่มีราคาระดับกลางๆก็สามารถเล่นเกมได้ดีระดับหนึ่งแล้ว  ซึ่งก็แน่นอนว่ามันยังไม่หยุดแค่นี้แน่ๆ  โดยในต่างประเทศเราจะเริ่มเห็นการให้บริการเล่นเกมผ่านทางเน็ตกันแล้ว  อย่างในรายของ OnLive ที่มีบริการให้สมาชิกสามารถเล่นเกมโหดๆผ่านทางเน็ตได้  ถึงแม้ว่าสเปกเครื่องของผู้ใช้จะไม่แรงมากก็ตาม  ทำให้ฝันของหลายๆคนที่อยากเล่นเกมใหญ่ๆ โหดๆ  แต่งบไม่อำนวยเป็นจริงขึ้นมาได้  แต่ในกรณีอาจจะยังมาใช้กับในประเทศไทยไม่ได้  เนื่องด้วยข้อจำกัดในหลายๆสาเหตุ เช่น การซื้อหาเกมบน OnLive  และความเร็ว/ความเสถียรของเน็ตที่เอาแน่เอานอนไม่ได้มากนัก  คาดว่าวิธีนี้คงต้องรอกันไปยาวๆละนะ
9. ถ้าต้องการแบบที่สุด…ยอมจ่ายไปเหอะ
Samsung-Series-9-Crystal-Studded-Laptop-1สำหรับ กรณีใครที่ต้องการเครื่องที่สุดๆไปเลย แนะนำว่าถ้าจ่ายไหวก็จัดไปเลยครับ  เพราะซื้อครั้งเดียวแล้วน่าจะใช้งานได้อีกนานพอดู  แต่ก่อนจะจ่ายก็วางแผนการใช้เงินดีๆนะครับ จะได้ไม่ลำบากภายหลัง
10. ถ้ายังไม่จำเป็นถึงขั้นต้องซื้อใหม่…อัพเกรดก็พอได้อยู่นะ
upgradechartก่อนจะซื้อคอมเครื่องใหม่ ขอให้ลองถามตัวเองให้แน่ใจก่อนกว่า เราจำเป็นต้องซื้อมั้ย เพราะ บางทีการอัพเกรดเล็กๆน้อยๆ  ก็อาจทำให้ประสิทธิภาพเครื่องเพิ่มขึ้นมาได้มากพอตัว  โดยเฉพาะอย่างยิ่งการเปลี่ยนจาก HDD ไปใช้ SSD  หรืออาจรวมไปถึงการเพิ่มแรมเข้าไปอีกหน่อยก็น่าจะอยู่ได้อีกระยะหนึ่งที เดียว แต่ถ้าเครื่องใครจะพังแหล่มิพังแหล่ จำเป็นต้องซื้อก็จัดไปโลด
ก็หวังว่าทั้ง 10 ข้อคงจะช่วยในการตัดสินใจเลือกซื้อ notebook  ใหม่ของทุกๆคนได้ไม่มากก็น้อยละนะ ขอให้พลังแห่ง notebook  จงสถิตอยู่กับท่านครับ
