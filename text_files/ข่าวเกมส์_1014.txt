 
Football Manager 2013
เกมส์วางแผนคุมทีมฟุตบอลแห่งมวงมนุษย์ชาติ Football Manager 2013 หรือ FM 2013 ภาคใหม่กำหนดวางจำหน่ายออกมาแล้ว หลังจากประกาศเปิดตัวเกมส์ไปเมื่อช่วงเดือนกันยายน 2012 ที่ผ่านมา สำหรับผู้ที่สั่งจองล่วงหน้า จะได้รับ Code สำหรับลงทะเบียนเล่นในช่วง Beta Test ซึ่งข้อมูลต่างๆในการเล่นช่วง Beta จะนำไปเล่นต่อยังเวอร์ชั่นเต็มได้อีกด้วย
 





สิ่งที่ Football Manager 2013 ภูมิใจนำเสนอในภาคนี้ก็คือระบบ 3D match engine ตัวใหม่ล่าสุด ที่จะทำให้เวลาเข้าสู่โหมด View ชมการแข่งขันนั้นแลดูสมจริง และสวยงามมากขึ้น เพิ่มอรรถรสในการเล่นสำหรับผู้ที่ชอบชมการแข่งขันระหว่างคุมทีมมากขึ้น ทำให้เวลาจะรู้สึกเหมือนได้เป็นผู้จัดการคุมทีมฟุตบอลจริงๆ หรือถ้าบางคนอาจจะไม่ชอบเกมส์ผู้จัดการทีมฟุตบอลที่มีระบบเรื่องมากยุ่งยาก ใช้เวลาเล่นนานเกินไป ทาง Football Manager 2013 เค้าก็มีระบบอีกอย่างจะนำเสนอ ก็คือ FMC mode เป็นโหมดการเล่นแบบคลาสสิคที่ไม่เน้นความสมจริงหรือระบบที่ยุ่งยาก เหมาะกับผู้เล่นที่อยากเล่นจบแต่ละฤดูกาลแบบไวๆ เน้นคุมทีมแบบจัดทีม แทคติก และก็ผลการแข่งขันเท่านั้น โดยยกเลิกพวกความสมจริงยุ่งยากอย่างเรื่องการขอใบอนุญาตทำงาน (Work Permit) การทำสัญญาต่างๆออกไป ทำให้ในโหมดนี้สามารถเล่นจบฤดูกาลโดยใช้เวลาประมาณ 10 ชั่วโมงเท่านั้นเอง
 





ส่วนโหมดปกติก็จะมีการเพิ่มความสมจริงเข้าไปอีกอย่างการจ้างโค้ช มาฝึกซ้อมเฉพาะกิจให้นักเตะ หรือโหมดฝึกซ้อมแบบใหม่ๆ และก็ยังมี worldwide leaderboards ให้เล่นแข่งขันทำคะแนนกันกับผู้เล่นคนอื่นๆผ่านระบบออนไลน์ด้วย
