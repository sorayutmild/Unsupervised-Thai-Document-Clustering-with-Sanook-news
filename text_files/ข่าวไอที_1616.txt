Galaxy Note 2 กับ Lumia 920 เลือก?
เมื่อสัปดาห์ก่อน arip ได้จัดทำรายงานพิเศษ "10 เหตุผลที่ทำใหคุณหลงรัก Note 2 ตอนที่ 1" และ ตอน 2 ปรากฎว่า ได้รับความสนใจจากคุณผู้อ่านอย่างท่วมท้น เช้านี้ ทางกองบรรณาธิการขอเสนอรายงานพิเศษเปรียบมวยสมาร์ทโฟนเรืองธงของ Samsung กับวินโดวส์โฟนที่มาแรงอย่าง Nokia Lumia 920 เพื่อเป็แนวทางสำหรับคุณผู้อ่านเว็บไซต์ arip ที่กำลังตัดสินใจอยู่ว่าจะเลือกใครดี?
ซัมซุง (Samsung) กำลังเดิมพันกับ Galaxy Note 2 ในการที่จะปิดช่องว่างของตลาดสมาร์ทโฟนเรือธงรุ่นพี่อย่าง Galaxy S3 อย่างไรก็ตาม คู่แข่งที่โผล่ขึ้นมาในตลาดเป็นรายที่ 3 เพื่อไม่ให้สมรภูมิรบนี้มีแค่ Samsung และ Apple นั่นก็คือ Nokia ที่ส่ง Lumia 920 วินโดวส์โฟนที่ทางบริษัทหวังจะใช้ในการทวงบัลลังก์ผู้ผลิตสมาร์ทโฟนอันดับหนึ่งของโลกคืน อย่างไรก็ตาม Noka เข้าสุ่ตลาดสมาร์ทโฟนช้าเกินไปจนทำให้สมรภูมินี้มีการแข่งขันสูงมาก ทำให้ไม่ใช่เรื่องง่ายเลยที่จะแทรกตัวเข้ามาเบียดคู่ชกยักษ์ใหญ่สองราย (Samsung & Apple) อย่างไรก็ตาม เพื่อให้เห็นภาพความพยายามของ Nokia ในการกลับมาครั้งนี้ ทางกองบรรณาธิการจึงถือโอกาสเปรียบเทียบสมาร์ทโฟนเรือธงจากทั้งสองบริษัทระหว่าง Samsung Galaxy Note 2 ที่ทำงานด้วยระบบปฏิบัติการ Android กับ Nokia Lumia 920 ที่ใช้ระบบปฏิบัติการ Windows Phone 8 เพื่อเป็นแนวทางให้คุณผู้อ่านสามารถนำไปใช้ในการประกอบการตัดสินใจเลือกซื้อสมาร์ทโฟนที่เหมาะกับตัวเองได้อย่างถูกต้อง
ดีไซน์ และเคส: เริ่มต้นกันด้วยขนาด และน้ำหนักของสมาร์ทโฟนสองรุ่นนี้ (เปรียบเทียบกันแบบนักมวยเลย) Galaxy Note 2 จะมีขนาด 151.1×80.5×9.4 มม. หนัก 180 กรัม ส่วน Lumia 920 จะมีขนาดของตัวเครื่อง 130.3×70.8×10.7 มม. และหนัก 185g ชัดเจนว่า Note 2 ใหญ่กว่าพอสมควร บางกว่ามาก ส่วนน้ำหนักต่างกันเล็กน้อย Galaxy Note 2 มีให้เลือก 3 สีด้วยกันคือ น้ำเงิน เทา และขาว แต่ด้วยเคสที่เป็นพลาสติกอาจจะทำให้ดูเหมือนของราคาถูก อย่างไรก็ดี สำหรับสาวก เรื่องนี้ไม่ใช่ประเด็น Galaxy Note 2 มาพร้อมกับ S-Pen และหน้าจอ HD ขนาด 5.5 นิ้วที่ใหญ่ที่สุดในตลาด ส่วนทางด้าน Nokia Lumia 920 จะมีสีสันให้เลือกมากกว่า เหลือง แดง ฟ้า ดำ ฯลฯ เคส Polycarbonate ชิ้นเดียวตลอดทำให้มันดูแข็งแรงทนทาน หรูหราสมราคม ซึ่งทำให้เป็นที่ต้องตาต้องใจจนต้องสัมผัส ซึ่งน่าจะเป็นจุดแข็งภายนอกที่ดูโดดเด่น และเหนือกว่า Note 2 แม้จะมีขนาดตัวเครื่อง และจอเล็กกว่าก็ตาม

จอแสดงผล: อย่างที่เกริ่นไปบ้างแล้วในข้อเปรียบเทียบประเด็นแรก Galaxy Note 2 จะมาพร้อมกับหน้าจอสัมผัส Super AMOLED HD ขนาด 5.5 นิ้ว ความละเอียด 1280 x 720 พิกเซล (267 ppi) กระจกหน้าจอแข็งแกร่งด้วย Gorilla Glass 2 ในขณะที่ Lumia 920 หน้าจอสัมผัส 4.5 นิ้ว IPS LCD ทีมาพร้อมกับเทคโนโลยี PureMotion HD+ ความละเอียด 1280 x 768 พิกเซล (332 ppi) แถมยังมีเทคโนโลยี ClearBlack ที่ให้จอดำสนิท คอนทราสสูง และลดการสะท้อนทำให้มันสามารถมองเห็นได้ค่อนข้างถนัดตา แม้ภายใต้แสงธรรมชาติ PureMotion HD+ ยังให้คุณภาพเหนือกว่า HD ทั่วไป หากพูดเรื่องจอแล้ว Nokia Lumia 920 ดูเหมือนจะได้เปรียบทั้งในส่วนของความละเอียด และการแสดงผลที่เหนือกว่าชัดเจน

หน่วยความจำ และสตอเรจ: หากเปรียบเทียบในแง่ของสตอเรจ Galaxy Note 2 ดูเหมือนจะชนะใส โดยมีให้เลือก 16GB, 32GB และ 64GB ซึ่งทั้งสามรุ่นยังมาพร้อมกับหน่วยความจำ 2GB RAM แถมยังมีช่องใส่การ์ดหน่วยความจำ microSD ให้เพิ่มได้อีกต่างหาก ส่วนทางด้าน Nokia Lumia 920 จะมาพร้อมกับสตอเรจ 32GB และหน่วยความจำ 1GB RAM แถมยังไม่มีช่องใส่การ์ดหน่วยความจำ microSD ให้เพิ่มได้อีกต่างหาก
โพรเซสเซอร์ และชิปเซต: ความแรงของโพรเซสเซอร์เป็นอีกปัจจัยหนึ่งในการเลือกใช้สมาร์ทโฟนในปัจจุบัน ซึ่ง Galaxy Note 2 โดดเด่นด้วยโพรเซสเซอค์"ควอด-คอร์" Exynos ความเร็ว 1.6GHz ของ Samsung ในขณะที่ทำงานร่วมกับชิปเซ็ตที่ตอบโจทย์การทำหลายงานพร้อมกัน (multi-tasking) ซึ่งทำให้ Note 2 มีการทำงานที่ไหลลื่น คุณสามารถใช้งานง่ายๆ ตั้งแต่อีเมล์ไปจนถึงเล่นเกมส์ไฮเอ็นด์บน Galaxy Note 2 ได้อย่างสบายๆ ส่วนทางด้าน Nokia Lumia 920 จะพร้อมกับโพรเซสเซอร์ "ดูอัลคอร์" Snapdragon Krait ความเร็ว 1.5GHz ของ Qualcomm แม้ความเร็วจะต่ำกว่า เมื่อดูจากสเป็ก แต่การทำงานของมันลื่นไหลไม่แพ้กัน โดยเฉพาะเมื่อทำงานกับแอพพลิเคชันต่างๆ บน Windows Phone 8 การเปรียบเทียบในข้อนี้ ในแง่มุมของประสิทธิภาพความเร็วในการใช้งานถือว่าไม่ต่างกันมากนัก

ระบบปฏิบัติการ: Galaxy Note 2 มาพร้อมกับระบบปฏิบัติการ Android Jelly Bean ซึ่งเป็นโอเอสทีดีที่สุดในปัจจุบันของ Android โดยคำตอบทีได้จากโอเอสตัวนี้ก็คือ การทำงานของแอพฯ ที่ไหลลื่น รวมถึงการเล่นเกมส์ และเล่นมีเดียต่างๆ ในขณะที่ Lumia 920 ที่มาพร้อมกับ Windows Phone 8 ซึ่งเป็นโอเอสใหม่ล่าสุดเหมือนกัน แต่จะให้มีฟีเจอร์ลูกเล่นทันกับ Android เห็นทีจะยังไม่ได้ อย่างไรก็ตาม Windows Phone 8 บน Nokia Lumia 920 ก็มีลูกเล่นบางอย่างที่น่าสนใจ โดยเฉพาะอินเตอร์เฟซแบบ Live tiles และ Social Network ที่อัพเดทการเปลี่ยนเปลียนแปลงให้เห็นได้ทันทีที่เปิดหน้าจอขึ้นทำงาน ใครทีต้องการการอัพเดทตลอดเวลา แถมยังเข้าถึงบริการเหล่านี้ได้โดยง่าย คุณอาจจะชอบ Windows Phone 8 บน Lumia 920 ก็ได้

กล้อง: สำหรับการเปรียบทียบคุณสมบัติข้อนี้คงต้องยกให้ Nokia Lumia 920 เป็นผู้ชนะ เพราะเป็นฟีเจอร์ที่ทางผู้ผลิตเน้นมากๆ Lumia 920 มาพร้อมกับกล้อง 8MP เทคโนโลยี PureView ซึ่งทำให้ได้ภาพที่ชัด และสว่างกว่า ในขณะที่ให้คอนทราส และรายละเอียดของภาพถ่ายได้เหนือกว่า แม้จะเป็นการถ่ายภาพในที่แสงน้อยมากๆ ก็ตาม ส่วนกล้องหน้า 1.3 ล้านพิกเซล ทางด้าน Samsung Galaxy Note 2 กล้องหลัง 8MP เหมือนกัน สามารถบันทึกวิดีโอ 1080p ในขณะที่กล้องหน้าจัดเต็มกว่าคือ 1.9 ล้านพิกเซล ฟีเจอร์อื่นๆ ก็จะเหมือนกับกล้องบนสมาร์ทโฟนทั่วๆ ไปไม่ว่าจะเป็น ออโต้โฟกัส หรือสัมผัสตำแหน่งที่ต้องการให้โฟกัส แฟลช LED ระบบตรวจจับรอยยิ้ม และใบหน้า ฯลฯ

แล้วจะเลือกใครดีล่ะ? อ่านมาถึงตรงนี้ เชื่่อว่า คุณผู้อ่านที่กำลังชั่งใจอยู่ว่าจะเลือกสมาร์ทโฟนรุ่นไหนดีน่าจะได้คำตอบมากขึ้นแล้ว อย่างไรก็ตาม หากพิจารณาทางด้านของ Samsung Galaxy Note 2 ก็ต้องบอกว่า มันเป็นพัฒนาการที่เหนือกว่าสมาร์ทโฟนรุ่นก่อนหน้านี้อย่างเหนือชั้น โดยเฉพาะ S-Pen ที่ตอบสนองการขีดเขียนไอเดียสร้างสรรค์ได้อย่างคล่องแคล่ว แถมยังใช้ Android OS รุ่นใหม่ที่ทำงานบนโพรเซสเซอร์แรงสุดๆ ที่หมายถึงความเร็ว และการทำงานหลายงานได้พร้อมกันอย่างมีประสิทธิภาพ ในขณะที่ Nokia Lumia 920 มีโพรเซสเซอร์ที่ไม่แพ้กันมากนักในเรื่องความเร็ว กับฟีเจอร์ใหม่ที่ได้รับการพัฒนาให้ดีขึ้นสำหรับการทำงานบนสมาร์ทโฟนของ Windows Phone 8 ซึ่งสอบผ่านในแง่ของการใช้งาน แต่จุดตัดสินใจที่สำคัญของ Nokia Lumia 920 ก็คือ กล้องที่ดีกว่า และหน้าจอที่ละเอียดกว่า แถมยังให้ภาพ HD ที่เหนือกว่า (หน้าจอเล็กกว่า) สองอย่างนี้ Galaxy Note 2 ให้คุณไม่ได้ ชอบแบบไหนก็เลือกแบบนั้น :D
