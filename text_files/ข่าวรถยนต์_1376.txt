     งานมอเตอร์โชว์ 2014 (Motor Show 2014) รอบพิเศษเริ่มต้นขึ้นแล้วเมื่อวานนี้ (24 มี.ค.) โดยบรรดาค่ายรถชั้นนำ ต่างนำรถรุ่นใหม่มาแสดงอย่างคับคั่ง
     วันนี้ Sanook!Auto มีภาพงานมอเตอร์โชว์ 2014 มาให้คุณผู้อ่านได้ชมแบบ Exclusive กันเลยทีเดียว

BMW i8

      โดยในงานวันนี้ เป็นรอบของบรรดาเหล่าผู้ร่วมงานวีไอพี ในงานจึงมีค่ายรถหรูทยอยเปิดตัวกันไปก่อน ไม่ว่าจะเป็น Mercedes Benz ที่นำรถครอสโอเวอร์ล่าสุดอย่าง GLA Class มาจัดแสดง รวมไปถึง C Class เจเนอเรชั่นใหม่ล่าสุดอีกด้วย แถมเคาะราคาสุดยั่วใจ
      นอกจากนั้นยังมี BMW, Mini, Rolls-Royce, Aston Marton และ MG มาเปิดตัวในรอบนี้ด้วย
      สำหรับ Motor Show 2014 ปีนี้จัดระหว่างวันที่ 26 มีนาคม-6 เมษายน ณ อาคารชาเลนเจอร์ 1-3 อิมแพ็ค เมืองทองธานี ภายใต้แนวคิด "Beauty in the Drive" หรือ "ยานยนต์สะกดใจ" ความสวยงาม และสีสันของการออกแบบก่อให้เกิดความชื่นชมผ่านการเข้าใจ รับรู้ถึงความสมดุล สัดส่วน และแรงดึงดูดของศิลปะการออกแบบ หรือความคิดสร้างสรรค์ที่สะกดใจ สะกดสายตาผู้คน กับงานแสดงยนตรกรรมที่รวมทุกรูปแบบของความสดกว่า ใหม่กว่า และสวยงามอย่างลงตัว
โดยภายในงานจะมีรถยนต์จากค่ายต่าง ๆ รวมถึงการเปิดตัวรถใหม่ เช่นปิดตัวรุ่นใหม่ เช่น Mazda 3,  Nissan Livina, Toyota Altis ESport, Ford Ecosport, BMW Series 2, Benz C Class เป็นต้น
