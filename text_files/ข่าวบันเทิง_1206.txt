อีกเพียงไม่กี่วันก็จะเข้าพิธีฉลองมงคลสมรสของคู่รักมาราธอน "ลิเดีย ศรัณย์รัชต์" กับหวานใจหนุ่ม "แมทธิว ดีน" แล้ว แต่ก็ดันเกิดปัญหาดราม่าขึ้นมาเสียก่อน เพราะเมื่อวันปาร์ตี้สละโสดของทั้งคู่ ที่เพื่อนๆ ต่างเซอร์ไพรส์แอบซุ่มจัดให้นั้น สาวลิเดียกลับได้ของขวัญเป็นถุงยางอนามัยจำนวนมหาศาล ซึ่งเจ้าตัวได้โพสต์ลงในอินสตาแกรมส่วนตัวแบบขำๆ แต่ผลตอบรับที่ได้รับนั้นคือกระแสวิพากษ์วิจารณ์ที่ไม่เหมาะสมตามมาเพียบ
อีกทั้งยังเกิดปัญหาระหว่างช่างภาพในงานแต่ง ที่แฉพฤติกรรมการมาถ่ายภาพของอีกทีมทั้งๆ ที่ไม่ได้ถูกว่าจ้างมา ทำให้ทีมหลักต้องมีอุปสรรคในการทำงาน ล่าสุดพอมีโอกาสได้เจอทั้ง ลิเดีย และแมทธิว ควงคู่มางานThe Mall Shopping Center Summer Wild ทั้งคู่จึงได้ออกมาเคลียร์ทุกปัญหาให้ฟังว่า
ปาร์ตี้สละโสดที่เพื่อนๆ พร้อมใจกันจัดให้?ลิเดีย : ตั้งแต่วันนั้นถึงวันนี้ยังไม่ได้พักเลย เรื่องของเรื่องวันนั้นคิดว่าต้องไปทำงานไปร้องเพลง ขึ้นคอนเสิร์ตด้วยซ้ำ เพราะผจก.บอกเดียว่าต้องร้อง 5 เพลง พอเดินเข้าไปปุ๊บ เพื่อนๆ ทุกคนก็เซอร์ไพรส์จุดอะไรกันปุ้งปังไปหมดใส่ชุดสีแดงกันหมดเลย เราก็งงๆ เล็กน้อย
แต่เห็นลิเดียใส่ชุดขาว?ลิเดีย : ก็เขาหลอกเราให้ใส่สีขาว เหมือนเป็นธีมงานเลี้ยงบริษัทที่เขาจ้างเรามาประมานนั้น
ตอนไหนที่ทำให้เราน้ำตาแตก?ลิเดีย : น้ำตาแตกหลายรอบมาก เพราะเพื่อนๆ แต่ละคนเขาทำของขวัญมาให้เรา เป็นความตั้งใจที่เขาทำเพื่อเราเลยนะแมทธิว : เพื่อนบางคนก็มาจากต่างประเทศเลยลิเดีย : ใช่ค่ะ บางคนก็บินตรงมาจากต่างประเทศไม่ได้เจอกันมาหลายปีมาก ซึ่งเราก็ไม่คิดว่าเค้าจะมาเซอร์ไพรส์เราขนาดนี้ ร้องไปหลายรอบเลย มันเป็นการเซอร์ไพรส์ที่เดียไม่รู้เรื่องมาก่อนเลยค่ะ
ของขวัญเป็นถุงยาง?ลิเดีย : คือมันไม่ใช่ของขวัญนะ มันเหมือนเล่นกันเป็นเกมส์ เขาปิดตาเดียแล้วก็ให้ตีๆๆๆ ซึ่งปกติของในกล่อง ตีออกมาจะต้องเป็นลูกอมนะ แต่นี่เจอแต่คอนดอมเต็มไปหมดเลย ลูกอมไม่มีเลยค่ะ (หัวเราะ)
แต่คนค่อนข้างวิจารณ์เยอะ?ลิเดีย : เอาสนุกมากกว่าค่ะ อย่าไปซีเรียสเลยเนอะ
ได้อ่านคำวิจารณ์เหล่านั้นบ้างไหม?ลิเดีย : อ่านค่ะ แต่เดียอ่านแล้วก็ไม่ได้คิดอะไรนะ เดียว่าอย่างที่บอกคือก็อย่าไปอะไรมากดีกว่า เพื่อนๆ ก็ตั้งใจให้ขำ ทุกคนตั้งใจเล่นกันเพื่อความสนุกและขำๆ อยู่แล้วค่ะ
งานหมั้นที่ผ่านมาเป็นยังไง?ลิเดีย : เป็นอะไรที่ดีมากๆ ค่ะ มีความสุขมากๆ
มีคำพูดอะไรถึงกันไหม?แมทธิว : วันนั้นไม่ได้มีอะไรเลยครับ เป็นงานพูดคุยกันมากกว่า แขกผู้ใหญ่เราทั้งคู่ก็ค่อนข้างเยอะพอสมควร ก็ถือว่าผ่านพ้นไปด้วยดีนะครับ โดยส่วนตัวก็ถือว่าพอใจกับงานหมั้นที่ผ่านมานะ
ล่าสุดมีประเด็นดราม่าเกี่ยวกับช่างภาพในงานแต่งเรา?ลิเดีย : ได้ข่าวเหมือนกันค่ะ แต่เราก็ไม่รู้อะไรเลยนะ
เราติดต่อเขามาถ่ายภาพใช่ไหม?แมทธิว : ต้องอธิบายก่อนว่าเราจะมีออร์แกไนซ์ทำงานให้เราอยู่แล้ว ในส่วนเรื่องของการจ้างตากล้องมา เราจ้างมา 3 ทีม เพื่อเก็บภาพในงานเรา คือใน 3 ทีมนั้น ก็คงจะดึงใครมาช่วย มันก็แล้วแต่เขาครับ
Box wedding ถือเป็นทีมตากล้องงานเราไหม?ลิเดีย : คือบ็อกเวดดิ้งเป็น 1 ใน 3 ทีมค่ะ คือเราจ้าง 3 เจ้าอยู่แล้วค่ะ เราจะเป็นคนเลือกช่างภาพเองค่ะ ว่าเราอยากได้ใครบ้าง ซึ่งก็มีทั้งหมด 3 เจ้า
ตกใจไหม?แมทธิว : มันไม่ใช่เรื่องของเรานะ มันไม่ได้เกี่ยวกับเราเลย เพราะเขาอาจจะมีปัญหาของวงการเขาคือวงการช่างภาพ แต่มันไม่ได้เกี่ยวกับเราเลยนะ
เสียความรู้สึกไหมว่าทำไมต้องมาเกิดกับงานแต่งเรา?ลิเดีย : คือเก่งทุกเจ้านะคะ เราก็เลือกทุกเจ้าอยู่แล้ว และมี 2 เจ้าก็ไปถ่ายพรีเวดดิ้งให้เราที่ต่างประเทศด้วยค่ะ
คนเลยมองว่าเหมือนงานขรุขระ?ลิเดีย : เรื่องขรุขระในงานหมั้นคือเรื่องปกติอยู่แล้วค่ะ เพราะพื้นที่ค่อนข้างจำกัดพอสมควร แต่เดียถือว่าเดียวพอใจนะค่ะ
จะคุยกับคุณจักรวินทร์ (ช่างภาพจากบ็อกเวดดิ้งที่ถูกกล่าวหาว่ามาถ่ายฟรี) ไหม เพราะล่าสุดเขาไปฟ้องร้องฐานเสื่อมเสียชื่อเสียงแล้ว?แมทธิว : เขาไม่ได้ฟ้องเรานี่ครับ
เห็นว่าคุณจักรวินทร์เสียความรู้สึก อาจจะไม่ถ่ายงานแต่งดาราต่อไป?ลิเดีย : เราไม่ขอคอมเม้นท์ตรงนี้ดีกว่า ขอไม่ยุ่งดีกว่านะค่ะ คือ 3 เจ้าเขาจะทะเลาะอะไรกันมันเป็นเรื่องของเขานะ เดียกับพี่แมทคงไม่ยุ่งดีกว่าค่ะ
ถือว่าเป็นอุปสรรคไหม?ลิเดีย : เดียทองว่ามันไม่ใช่อุปสรรคนะคะ เพราะมันไม่ใช่เรื่องของเราเลยค่ะ เดียว่าที่ผ่านมาทุกอย่างก็ราบรื่นดีนะ สำหรับเดีย ในเรื่องช่างภาพก็ไม่ได้เกี่ยวกับเราอยู่แล้ว
งานฉลองวันที่ 21 พร้อมไหม?ลิเดีย : พร้อมค่ะ เราเตรียมตัวพร้อมแล้วนะ ตอนนี้ทุกอย่างน่าจะโอเคประมาณ 95%แล้วค่ะ
แว่วว่าแค่งานหมั้นที่ผ่านมาค่าใช้จ่ายก็สูงถึง 10 ล้านบาทแล้ว?แมทธิว : ยังไม่ได้รวมยอดเลยครับ
กลัวคนจะมองว่าเวอร์หรืออลังการเกินไปไหม?ลิเดีย : ไม่นะค่ะ เพราะเดียว่ามันก็โอเคแล้วนะกับงานแต่งงาน อีกอย่างเดียก็ชอบที่ให้งานออกมาอย่างนี้ด้วยค่ะ
ลองใช้ชีวิตคู่ด้วยกันแล้วเป็นอย่างไรบ้าง? แมทธิว : ตั้งแต่หลังจากงานหมั้น เรายังไม่ได้เจอกันเลยครับ (หัวเราะ)ลิเดีย : คือ 2-3 วันที่ผ่านมาพี่แมทรับงานพิธีกรตลอดเลยค่ะ (หัวเราะ) เดียก็มีงานตลอด ยังแทบจะไม่เจอกันเลยแมทธิว : หลังวันที่ 21 เราคงจะเป็นสามี-ภรรยาอย่างเป็นเรื่องเป็นราวนะ (หัวเราะ)
