
ขึ้นชื่อว่าเกมส์จาก Crytek แล้ว เกมส์เขาไม่มีคำว่าภาพกาก เพราะใช้เอนจิ้นระดับสุดยอดอยู่เสมอๆ ไล่มาตั้งแต่ Cryengine ตัวแรก มาจนถึงล่าสุด Cryengine3 ที่สมจริงมากยิ่งขึ้นไปอีก โดยเฉพาะอย่างยิ่งความละเอียดของกราฟิกของแสงไฟ และน้ำ ที่ทาง Crytek เน้นกันมากเป็นพิเศษในเกมส์ใหม่ Crysis 3 ของพวกเขา เพราะต้องเข้าไปรบกันใน Liberty Dome ที่กลายสภาพเป็นป่าในซากเมืองไปแล้ว แต่ไม่ว่าเอนจิ้นสร้างเกมส์ของ Cryengine 3 จะเทพแค่ไหน ถ้า PC ที่ผู้เล่นใช้มันห่วยเกินกว่าจะรันภาพสวยๆได้ก็เท่านั้น งั้นเรามาดูก่อนดีกว่าว่า สเปกเครื่องที่ต้องใช้เล่นเกมส์ Crysis 3 นั้นควรเป็นแบบไหน

สเปกเครื่อง PC ขั้นต่ำสำหรับเกมส์ Crysis 3

 ระบบปฏิบัติการณ์ Windows Vista, Windows 7 or Windows 8
 DirectX 11 graphics card with 1Gb Video RAM
 Dual core CPU
 2GB Memory (RAM) (3GB on Vista)
 Nvidia/Intel example setup: Nvidia GTS 450, Intel Core2 Duo 2.4 Ghz (E6600)
 AMD example setup: AMD Radeon HD5770, AMD Athlon64 X2 2.7 Ghz (5200+)

 
สเปกเครื่อง PC ที่แนะนำสำหรับเกมส์ Crysis 3

 ระบบปฏิบัติการณ์ Windows Vista, Windows 7 or Windows 8
 DirectX 11 graphics card with 1GB Video RAM
 Quad core CPU
 4GB Memory (RAM)
 Nvidia/Intel example setup: Nvidia GTX 560, Intel Core i3-530
 AMD example setup: AMD Radeon HD5870, AMD Phenom II X2 565

 
สเปกเครื่อง PC ที่ต้องการกราฟิกเทพๆสำหรับเกมส์ Crysis 3

 ระบบปฏิบัติการณ์ Windows Vista, Windows 7 or Windows 8 (64 bit ยิ่งดี)
 Latest DirectX 11 graphics card
 Latest quad core CPU
 8GB Memory
 Nvidia/Intel example setup: NVidia GTX 680, Intel Core i7-2600k
 AMD example setup: AMD Radeon HD7970, AMD Bulldozer FX4150

สเปกเครื่องที่ต้องการภาพแรงๆ เรียกการ์ดจอชิพเดี่ยวแบบแรงสุดกันเลยทีเดียว !!

Crysis 3 ภารกิจปกป้องโลกของ Prophet ยังคงไม่จบสิ้นกันง่ายๆ แม้ว่า Alcatraz จะสามารถกวาดล้างพวก Cept เอเลี่ยนต่างดาวที่ยึดครองนิวยอร์คได้แล้วก็ตาม แต่เจ้าพวก Cell Corporation (หน่วยงานที่อยู่ฝ่ายตรงข้ามกับ Marine) ก็ยังก่อเรื่องไม่เลิก พวก Cell ได้ทำการรวบรวมเทคโนโลยีของพวก Cept ที่หลงเหลืออยู่ในเมืองนิวยอร์ค แล้วสร้างขึ้นเป็นโดมขนาดใหญ่ ชื่อว่า Liberty Dome โดยมีเป้าหมายที่จะใช้เทคโนโลยีนี้ในการเข้ายึดอำนาจทั่วโลก
 
ในปี 2047 ทาง Prophet ได้กลับมายังเมืองนิวยอร์คอีกครั้ง (ทาง  Crytek บอกว่าเขาพื้นจากความตายในภาค 2 กลับมาได้อีกครั้งในภาคนี้)  เพื่อตามล่ากวาดล้างพวก Cell ให้สิ้นซาก  พร้อมกับสืบหาว่าใครเป็นคนฆ่าลูกน้องในหน่วยของเขา และจะล้างแค้นไปพร้อมกัน  โดยมีอาวุธใหม่คือธนูพิฆาตที่มีพลังในการทำลายล้างเชื้อของพวก Cept  ที่อยู่ในเทคโนโลยีของพวก Cell ตอนนี้ รวมถึงมีธนูแบบหัวระเบิดด้วย  กับพลังใหม่ของชุด Nanosuit ที่เพิ่มขึ้น ทำให้สามารถใช้พลังได้พร้อมกันแบบ multi-tasking และยังมีพลังใหม่ๆที่เพิ่มขึ้นจาก Nanosuit 2 อีกด้วย
