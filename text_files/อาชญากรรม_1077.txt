"ตี๋" เจ้าของปาร์ตี้บางบัวทอง เผชิญหน้า "อัจฉริยะ" ยันไม่ได้ล่วงละเมิด "พริตตี้เดียร์" คราบอสุจิบนผ้าห่มแค่ช่วยตัวเอง เจอสวนแจ้งความเพราะร้อนตัว
จากกกรณีเรื่องราวต่อเนื่องจากคดี "ลันลาเบล" พริตตี้สาวซึ่งไปรับงานเอ็นเตอร์เทนที่บ้านบางบัวทองก่อนเสียชีวิตมีเงื่อนงำ ต่อมามีพริตตี้อีกคนนามว่า "เดียร์" ร้อง "นายอัจฉริยะ เรืองรัตนพงษ์" ประธานชมรมช่วยเหลือเหยื่ออาชญากรรม อ้างว่าถูกมอมเหล้า และถูกล่วงละเมิด ขณะรับงานเอนฯ ที่บ้านหลังดังกล่าว จนมีการเข้าแจ้งความดำเนินคดี 6 ผู้ต้องหา
ด้าน "นายนที" หรือ "ตี๋" เจ้าของปาร์ตี้บางบัวทอง หนึ่งในผู้ต้องหา ได้เข้าแจ้งความพริตตี้เดียร์ ข้อหาลักทรัพย์ พร้อมปล่อยคลิปสั้นๆ พริตตี้เดียร์ขึ้นคร่อมตี๋ หนึ่งในเจ้าของบ้าน เพื่อยืนยันว่าไม่มีการล่วงละเมิด และให้ภาพเล่าเรื่อง
รายการโหนกระแสวันที่ 1 ต.ค. โดย "หนุ่ม กรรชัย กำเนิดพลอย" ในฐานะผู้ดำเนินรายการ ผลิตในนามบริษัท ดีคืนดีวัน จำกัด ประเดิมเวลาใหม่ ช่องใหม่ ออกอากาศทุกวันจันทร์-ศุกร์ เวลา 12.20 น. ทางช่อง 3 ได้เปิดใจสัมภาษณ์ "ตี๋" ที่มาเผชิญหน้า "อัจฉริยะ" กันจะจะ

คุณตี๋ฝากบอกคุณอัจฉริยะ อย่าปล่อยนางไว้กลางทาง ให้ช่วยไปเรื่อยๆ?
อัจฉริยะ : "ปล่อยได้ไง เดือดเต็มร้อย"
เหตุการณ์ที่เกิดขึ้น หลังอุ่นพาลันลาเบลไปแล้ว วันนั้นเดียร์มายังไง?
ตี๋ : "ประมาณสี่ทุ่ม ตอนแรกเราจ้างน้องเดียร์มาหนึ่งทุ่ม น้องมาเลทประมาณสี่ทุ่ม ก็ดื่มสุรากันปกติ แต่ตื่นเช้ามารู้ข่าวน้องเบลเสีย ก็จะไปให้ปากคำที่สน. เลยขึ้นไปจะอาบน้ำ พอเปิดประตูห้องเข้าไปเห็นน้องเดียร์นอนอยู่ที่ชั้นสอง"
เป็นห้องที่คุณนอนประจำ?
ตี๋ : "ไม่ใช่ครับ ห้องนั้นเป็นห้องแต่งตัว มีแค่เสื้อผ้าอยู่"
คุณเปิดไปเห็นเขานอนสภาพยังไง?
ตี๋ : "ใส่เสื้อผ้า มีผ้าคลุม ผมก็ปลุกเขาให้ตื่น"
แล้วตอนเดียร์ไปถึงบ้านบางบัวทอง คุณเห็นเดียร์มาในสภาพไหน?
ตี๋ : "มาในชุดเดิมที่กลับไป"
เขาว่าคุณท้าชนเหล้าเขา ให้กระดก?
ตี๋ ไม่ครับ อย่างนั้นผมกินกันเอง ไม่ได้ให้น้องกิน
ยืนยันไม่มีการล่วงละเมิด?
ตี๋ : "ผมไม่ได้มีการล่วงละเมิดเลย"
เดียร์ตื่นขึ้นมาเอาผ้าออกจากบ้านหลังนั้นมาหนึ่งผืน กางเกงข้างในของใคร?
ตี๋ : "ของเฟิร์ส ภรรยาคิวครับ"
เขาไปแจ้งความเลยมั้ย?
ตี๋ : "ยังครับ แจ้งความน่าจะผ่านมาสองสามวัน"
คุณเป็นผู้ต้องหาเบอร์หนึ่งเลย?
ตี๋ : "ก็มีทั้งหมด  6 คน ทุกคนก็ประกันตัวออกมา ใส่อีเอ็มที่ขาอยู่ครับ"
กรณีนี้พี่อัจเข้ามาดูแลยังไง?
อัจฉริยะ : "เขามาร้องผมว่ามีการล่วงละเมิด เขาก็เล่าว่าตื่นเช้ามา แล้วรู้สึกไม่มีเสื้อใน มีผ้าห่มคลุม และมีกางเกงขาสั้นเฟิร์สอยู่ข้างใน"
ตี๋ : "ช่วงห้าทุ่มน้องเดินไปขอกางเกงเฟิร์สเองเพราะว่ามันโป๊"
ฝั่งนี้บอกเฟิร์สเปลี่ยนให้ แต่ตี๋บอกว่าเขาใส่เอง?
ตี๋ : "เขาบอกว่าเขาโป๊ เขาไม่ได้ใส่ซับในมา เขาเลยไปขอกางเกงเฟิร์ส"
เฟิร์สบอกว่ากางเกงตัวนั้นเป็นกางเกงที่เฟิร์สยังไมได้ซัก ก็อาจเป็นไปได้ที่อาจพบร่องรอยสามีเขาติดอยู่?
อัจฉริยะ : "อันนั้นเป็นเรื่องในสำนวน ตอบไม่ได้ เพราะถามพี่เขาแล้วในฐานะพี่เป็นเจ้าของผ้าห่ม"
ผ้าห่มอยู่ไหน?
ตี๋ : "เป็นผ้าห่มของผม มันก็อยู่ห้องนั้นแหละ ผมตั้งใจเอาไปซัก เพราะมันนานมากแล้ว ไม่ได้ใช้"
มีคราบอะไรคุณมั้ย?
ตี๋ : "ผมใช้เช็ดตัว มันก็อาจจะมีพวกสารคัดหลั่งต่างๆ"
อัจฉริยะ : "ทำไมคนอื่นไม่เห็นออกมาดิ้นรนเรื่องผ้าเช็ดตัว มีแต่ตี๋คนเดียวที่ออกมาร้อนรนเรื่องผ้าเช็ดตัว มีอแต่ตี๋คนเดียวที่ร้อนรน ถึงขนาดไปแจ้งความน้องเดียร์ข้อหาลักทรัพย์ผ้าห่ม ถ้าหากไม่มีคราบอสุจิแล้วคุณจะเดือดร้อนอะไร ผมถามหน่อย"
ตี๋ : "เรื่องนี้เราไม่ได้เดือดร้อนอะไร แต่เรารู้สึกว่าน้องเขาเอาของผมไป ผมก็แจ้งได้"
เดียร์ถามทำไมไม่ไปแจ้งตั้งแต่แรก?
ตี๋ : "ตอนแรกผมก็ไม่รู้ ของในห้องก็รกแหละ ผมไม่ได้เอะใจ จนมาเห็นในสำนวน"
อัจฉริยะ : "ผ้าห่มถือว่าเป็นของกลาง มันก็มีคราบอสุจิอยู่ พี่ไม่ต้องไปกังวล ถ้าไม่ใช่คราบอสุจิพี่ในการล่วงละเมิด เขาก็ยอมรับว่ามีคราบอสุจิ"
ตี๋ : "เดี่ยวก่อนครับ ผ้าห่มผมแจ้งไปแต่ต้นถ้ามีคราบอสุจิผมอยู่ก็ไม่แปลก ผมใช้เช็ดตัว บางทีเสร็จกิจก็เอามาเช็ด เปื้อนโดนผ้า"
เสร็จกิจหมายถึงอะไร?
ตี๋ : "ช่วยตัวเอง"
มีประเด็นคลิปวิดีโอ คุณปล่อยออกมาหรือเปล่า?
ตี๋ : "ใช่ครับ"
กรณีที่เกิดขึ้น มีคลิปที่หลุดออกมา เป็นคลิปผู้หญิงคนหนึ่ง เขาทำอะไรคุณ?
ตี๋ : "เขาดึงผมไปหอมครับ"
ภาพเดียร์ขึ้นไปอยู่บนตัวคุณ แล้วมีคำพูดที่หลุดออกมา เดี๋ยวเดียร์ทำเอง มันเกิดอะไรขึ้นตอนนั้น?
ตี๋ : "ผมเมามาก จนจำไม่ได้ ดูจากในคลิปผมนอนแผ่ไปแล้ว"
ที่ปล่อยคลิปออกมาเจตนาอะไร?
ตี๋ : "ทางเราโดนกระแสสังคมโจมตีค่อนข้างเยอะ เราก็อยากแก้ต่างให้ตัวเราเองบ้าง รูปพวกนี้เราไม่เคยปล่อยเลย จนกระแสมันตีเข้ามา มันเหนื่อยจนรับไม่ไหว จนต้องขอออกมาปกป้องตัวเอง"
คำว่าแก้ต่างหมายถึงอะไร?
ตี๋ : "เราไม่ได้ทำอะไรเลย เราสลบ เมาน็อกอยู่"
อัจฉริยะ : "การคุณปล่อยคลิป 30  วิ 19 วิ คุณมีเจตนาอะไร ทำไมไม่ปล่อยคลิปเต็ม การที่คุณทำแบบนี้แสดงว่าคุณมีคลิปผู้หญิงอยู่ในมือ และเป็นคลิปที่ไม่น่าดู คุณจะทำอะไรกับใครก็ได้ ที่มีคลิปอยู่ในมือ ถ้าคุณจะแสดงความบริสุทธิ์คุณ ทำไมไม่เปิดคลิปทั้งหมดให้คนดูทั้งประเทศ ผมมองว่าเจตนารมณ์คุณถ่ายคลิปเพื่ออะไร คุณยอมรับมั้ยล่ะว่าคุณแอบถ่ายคลิปน้อง"
ตี๋ : "อันนี้ขอแจงนิดนึง คลิปพวกนี้ที่ถ่ายกัน เพื่อแกล้งผมนี่แหละ คนที่ถ่ายเป็นคนในงาน เพราะมันมากันหลายคน เขาถ่ายเพื่อแกล้งผม เพราะผมมีคนคุยๆ อยู่ เขาบอกว่าจะส่งให้คนนี้นะ แค่นั้นเอง มันเลยเป็นคลิปสั้นๆ แค่นี้"
มีถ่ายไว้ยาวๆ กว่านี้มั้ย?
ตี๋ : "ไม่มีครับ"
อัจฉริยะ : "โนบิถ่ายคลิปอะไรไว้และส่งคลิปมาให้พวกคุณ จะบอกเลยว่านั่นคือหลักฐานทั้งหมด"
โนบิตะถ่ายคลิปเหตุการณ์ในนั้น เห็นเบลด้วย และส่งคลิปนั้นให้ผู้หญิงคนนึงดู?
อัจฉริยะ : และคลิปนั้นอยู่กับผม คุณบอกว่าคุณอยู่หน้าบ้าน  เรื่องเวลานอนคิวกับเฟิร์สยังไม่ตรงกันเลย ผมยังสงสัยอยู่เลย"
ถ้าทุกอย่างพิสูจน์ออกมาแล้วกลุ่มบ้านนี้บริสุทธิ์ พี่จะทำยังไง
อัจฉริยะ : "ก็ฟ้องผมกลับได้เลย ทางกฎหมายทำได้แต่บอกได้เลย คนที่ออกรายการกับผมและออกรายการพี่หนุ่ม ติดคุกหมด รายการนี้มีอาถรรพ์"
กรณีมีการถ่ายคลิปในบ้าน เห็นช็อตเดียวเป็นช็อตๆ ไม่คิดถ่ายเบลเหรอ?
ตี๋ : "ของเบลมีคลิปเพิ่มเติมแต่ส่งเป็นหลักฐานให้ตร."
อัจฉริยะ : "ผมมองว่าเจตนามันเห็นชัดอยู่แล้ว  เวลาถ่ายคลิปเขาไม่ถ่ายสั้นๆ แต่นี่ถ่ายสั้นๆ ให้เห็นน้ำอุ่นนั่งอยู่กับลันลาเบล เจตนามันชัดเจนว่าคืออะไร ผมเชื่อว่าคุณมีคลิปเต็ม ก่อนมานั่งที่นี่ ผมก็ได้มาแล้วคลิปที่มีการถ่ายกัน สิ่งที่เกิดขึ้นกับน้องเบล ต้องมีกฎแห่งกรรม ช้าเร็วหนีความจริงไปไม่พ้น และมั่นใจว่าหลักฐานในคลิปวิดีโอจะทำให้คดีคลี่คลาย"
มั่นใจแค่ไหนว่าเขาทำ?
อัจฉริยะ : "เอาเป็นว่าทั้งกระบวนการนี้ไม่รอด"
โมเดลลิ่งถูกจับ พี่ไปแจ้งหรือเปล่า?
อัจฉริยะ : "ผมแจ้ง"
เขาบอกไม่ได้ส่งเด็กไปวี?
อัจฉริยะ : "แต่เขาเป็นธุระจัดหา ล่อไป พาไปเพื่อการอนาจาร ตร.เขาแถลงไปแล้ว คุณจ้างเขามาทำอะไร"
บ้านคุณกินเหล้าอย่างเดียว ไม่มียา?
ตี๋ : "ไม่มียา เราไม่ได้พลิ้ว แต่เราไม่ได้เก็บ"
คุณไม่ได้ล่วงละเมิดเดียร์?
ตี๋ : ไม่มีครับ"
ถ้าสุดท้ายไม่ใช่ จะฟ้องกลับมั้ย
ตี๋ : "มีสิทธิ์ฟ้องกลับครับ ไม่กลัวครับ แต่กังวลเรื่องเดียร์ โมเดลลิ่ง เพราะน้องโดนจับไม่มีใครไปช่วยเลย"
อัจฉริยะ : "ไม่ต้องห่วงเขาหรอกครับ เพราะเรากำลังขยายผลอยู่ เพราะเดียร์ไม่ได้ส่งเด็กให้คุณแค่คนสองคน อีกเดี๋ยวสังคมจะเข้าใจเรื่องทั้งหมด และเข้าใจว่าทำไมมีคลิปหลุดออกมาเรื่อยๆ การที่คุณทำแบบนี้คุณย่ำยีศักดิ์ศรีน้องที่เป็นพริตตี้ เพราะการที่เขามาบริการคุณที่บ้าน ไม่ใช่คุณไปถ่ายคลิปเขา คุณไปขอดูเสือเผ่นข้างหลัง"

แต่น้องเขาถอดเองนะ แล้วตอนน้องบอกเสื้อในหายไป แต่ที่ดูไม่มีเสื้อใน?
อัจฉริยะ : "มันเป็นเกาะอกพี่หนุ่ม เขาเรียกปีกนก"
ครั้งนี้ที่พี่เข้ามาเล่นเรื่องนี้เพราะต้องการจะกลับมา เหมือนถูกแบนออกไป?
อัจฉริยะ : "ไม่มี ทำไมต้องแบน ครั้งนี้ไม่มีปั่น ผมไม่จำเป็นต้องเรียกร้องอะไร ผมทำมาตลอดชีวิต ผมช่วยเหลือคนยากจน ผมถามว่าผมมีความผิดอะไร ผมไม่เคยถูกศาลลงโทษ ไม่เคยถูกตร.ดำเนินคดี แม้แต่คดีเดียว ถามว่าวันนี้ทำไมมานั่งตรงนี้ได้ แต่ผมมั่นใจว่ากระบวนการนี้ไม่เกิน 5 วัน รับรองมีผลต่อคดี"
ตี๋ : "ก็พร้อมรับ ให้มันเป็นไปตามรูปคดี มั่นใจไม่ได้ทำอะไร"
ถ้าคุณบริสุทธิ์ ?
ตี๋ : "มีแน่ครับ ฟ้องกลับครับ"
