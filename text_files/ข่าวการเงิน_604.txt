เสี่ยเมืองจันท์ เจ้าของกระบะ Mercedes Benz x class 350d 4 Matic V6 turbo คันแรกของเมืองไทยในราคาสุดคุ้มค่า
ใครๆ ก็ใฝ่ฝันอยากจะเป็นเจ้าของรถเบนซ์กันทั้งนั้น ยิ่งเป็นรุ่นพิเศษ และคันแรกของเมืองไทยหากได้สัมผัสก่อนใครรับรองว่าคนรอบข้างต้องร้องว้าวทันที! ล่าสุดเสี่ยเมืองจันท์ทุ่มเงินหลายล้านบาทซื้อกระบะเบนซ์สุดหรูใช้ขนทุเรียนคันแรกของเมืองไทย
>> ถอยกระบะเบนซ์ขนทุเรียน! เสี่ยเมืองจันท์สปอร์ตมาก โชว์รูมปลื้มลูกค้าคนแรกของประเทศ
ซึ่งรถกระบะ Mercedes Benz x class 350d 4 Matic V6 turbo ที่เสี่ยเมืองจันท์ซื้อเป็นคันแรกที่นำเข้าโดย Aey Auto Import ซึ่งเป็นผู้นำเข้ารถหรูของเมืองไทย อยากรู้หรือไม่กระบะเบนซ์รุ่นนี้ราคาเท่าไหร่? เรามีคำตอบมาฝากกัน
Sanook! Money ตรวจสอบรถกระบะ Mercedes Benz x class 350d 4 Matic V6 turbo ในเว็บไซต์ Mercedes Benz ประเทศอังกฤษ เนื่องจากพวงมาลัยรถอยู่ด้านขวาเหมือนกับไทย พบว่า ราคาขายเริ่มต้นอยู่ที่ 47,412 ปอนด์ คิดเป็นเงินไทยอยู่ที่ราว 1,859,393 บาท
หากนำเข้าประเทศไทยก็จะต้องบวกภาษีเพิ่มเติมอีก เช่น ภาษีนำเข้ารถยนต์ 200% สรรพสามิต และอื่นๆ ที่เกี่ยวข้อง ทำให้ราคาขายในไทยเพิ่มสูงขึ้นหลายเท่าตัว แต่แว่วๆ ว่าราคาในไทยอยู่ที่ 4-5 ล้านบาท
