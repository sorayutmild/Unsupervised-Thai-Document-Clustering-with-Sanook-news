ปกติการใช้งาน Facebook บนมือถือนอกจากเข้าผ่าน Application แล้วยังมีอีกวิธีคือใช้งานผ่าน Web browser บนมือถือเช่น Google Chrome แต่บางคนอยากได้ประสบการณ์เหมือนกับใช้ในคอมพิวเตอร์จะทำอย่างไรได้บ้างวันนี้ทีมงาน Sanook! Hitech จะมาแนะนำให้ลองทำดูกัน
วิธีนี้ใช้กับ Google Chrome ได้ทั้ง iOS และ Android

เริ่มแรกให้คุณเข้าเว็บไซต์ www.facebook.com โดยระบบจะแสดงผลเป็น m.facebook.com

จากนั้นให้กด 3 จุดข้างขวา เลือกเป็น Request desktop Site เท่านี้เว็บ Facebook จะเป็นเวอร์ชั่น Desktop เรียบร้อย

แต่ถ้าอยากกลับเป็นรูปแบบมือถือเดิมก็ให้เอา Request desktop Site ออกได้เช่นกัน
นอกจากนี้วิธีการดังกล่าวสามารถใช้ได้กับทุกเว็บไซต์ได้เช่นกันเลยทีเดียว ยังไงก็ลองปรับใช้ดูได้เพราะเชื่อว่าบางเว็บการแสดงผลหน้ามือถืออาจจะทำงานได้ไม่เต็มที่ก็เป็นได้
