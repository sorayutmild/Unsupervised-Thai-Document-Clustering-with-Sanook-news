PES Manager เกมผู้จัดการทีมฟุตบอลจาก PES
PES Manager
ข่าวดีสำหรับแฟนๆเกมฟุตบอล โดยเฉพาะแฟนๆของ PES ที่นับว่ายังมีเยอะในประเทศไทย ตอนนี้ทาง Konami ได้ลองทำเกม PES แนวผู้จัดการทีมออกมาสู่ตลาดกันดูบ้างแล้ว ชื่อว่า PES Manager  หลังจากที่ผ่านมามีแต่แนวเล่นเองมาโดยตลอด มีเล่นเป็นผู้จัดการในโหมด Manager เท่านั้น ซึ่ง PES Manager ภาคผู้จัดการทีมงานทำออกมาให้เล่นกันฟรีๆ ในเวอร์ชั่น iOS และ Android
ระบบการเล่นของ PES Manager โดยรวมแล้วก็จะคล้ายๆกับเกมผู้จัดทีมฟุตบอลทั่วไป คือคุมทีมและไต่อันดับจนเป็นแชมป์ให้ได้ แต่ว่าก็มีบางส่วนที่ไม่เหมือนเกมผู้จัดการทีมทั่วไป อย่างที่เราทราบกันดีว่า เกม PES ของ Konami นั้นมีปัญหาเรื่องที่ไม่ยอมซื้อลิขสิทธิ์ชื่อทีมและนักเตะของฟีฟ่าให้ครบทุกทีม ซึ่งในเกมนี้จะมีทีมและนักเตะที่ได้ลิขสิทธิ์ชื่ออยู่ประมาณ 1,500 คนเท่านั้น นอกนั้นจะกลายเป็นทีมนักเตะชื่ออะไรก็ไม่รู้ที่ตั้งให้คล้ายๆทีมจริงแทน แต่ยังดีที่มีโหมดดูการเล่นในแบบ 3D ให้อยู่
PES Manager
ส่วนนักเตะในทีมที่เราคุมนั้น ตัวเกมก็จะให้มาเพียงแค่ 14 คนเท่านั้นเอง ซึ่งน้อยมากหากเทียบกับเกมผู้จัดการฟุตบอลอื่นๆ แต่จะใช้ระบบการฝึกและพัฒนานักเตะให้เก่งขึ้นได้เรื่อยๆแทน โดยหากแข่งขันขนะผู้เล่นจะได้รับ Energy Balls มาเป็นรางวัล ซึ่งเจ้าลูกบอลพลังงานนี่แหละที่จะเอาไว้ใช้ฝึกนักเตะ โดยเลือกว่าจะฝึก Skill หรือฝึกความสามารถด้านต่างๆได้ ถ้าหากต้องการเพิ่มจำนวนสมาชิกในทีมจำเป็นต้องขยายทีมก่อนด้วยการใช้เงินอัพเกรดทีม โดยการจะได้เงินมาเยอะนั้นจะต้องเล่นชนะบ่อยๆก็จะได้เงินเพิ่มจากยอดคนดูในสนามนั่นเอง เพิ่มจำนวนนักเตะไปก็เท่านั้น เพราะเวลาลงสนาม นักเตะจะมีค่า Cost ด้วย นักเตะคนไหนเทพๆ ค่า Cost ก็จะสูง ทำให้เลือกแต่นักเตะเทพๆลงสนามครบทีมไม่ได้อยู่ดี
PES Manager
จะว่าไปแล้ว PES Manager เป็นเกมผู้จัดการทีมฟุตบอลที่มีระบบต่างจากทั่วไปพอสมควร เหมาะสำหรับคนที่อยากลองระบบการเล่นใหม่ๆ แต่ถ้าชอบการคุมทีมแบบสมจริงล่ะก็ แนะนำให้ไปเล่น Football Manager ของ SEGA น่ะแหละดีแล้ว
PES Manager จะให้โหลดเล่นฟรี ทั้งระบบ iOS และ Android โดยหาได้ใน App Store ประเทศต่อไปนี้เท่านั้น (ดีนะมีไทยด้วย) ยังไม่กำหนดวันออกมา แต่คาดว่าเร็วๆนี้
