เกมเศรษฐีออนไลน์  สวัสดีค่ะ  ข่าวดี!!! สำหรับผู้ที่อยากเล่นเกมเศรษฐี (Everybody’s Marble)   วันนี้...สามารถสมัครเล่นเกมเศรษฐีได้ทันที ไม่ต้องยืนยันอีเมล์ (Email  Activation)  ด้วยการใช้ Facebook ID ได้แล้วจ้าแค่เพียง มหาชนชาวเกมเศรษฐีมี ID ที่  Login ผ่าน Facebook ท่านก็สามารถนำมาลงทะเบียนเป็น Netmarble Id  ได้ง่ายๆโดยทำตามขั้นตอนดังนี้   ขั้นตอนการสมัครเล่นเกมเศรษฐี โดยใช้ Facebook ID  ขั้นที่1: เข้าสู่หน้าเว็บไซต์ www.ebm.in.th แล้วคลิกที่ปุ่ม Register ID  หรือเข้าไปที่หน้าสมัครตรงที่ URL : https://member.netmarble.in.th/join/index.asp  เกมเศรษฐีออนไลน์  ขั้นที่2:  คลิกที่ปุ่ม “Facebook” เพื่อทำการสมัครสมาชิกโดยใช้ ID ของFacebook    ขั้นที่3: ยืนยันข้อมูลบัญชี Facebook   3.1  Login เข้าสู่ระบบ Facebook โดยกรอก Email และ Password  Facebook ของคุณ     (หมายเหตุ: ในกรณีที่มีการ Login หรือใช้งาน Facebook ด้วย Internet Explorer อยู่จะข้ามขั้นตอนนี้ไป)  3.2 เมื่อมีป๊อบอัพปรากฎขึ้น ให้กด OK  เพื่อยืนยันการใช้ข้อมูล Facebook เพื่อเล่นเกมเศรษฐี          ขั้นที่4. กรอกข้อมูล  ID และรหัสผ่านที่ต้องการใช้เล่นเกมฯ ในหน้าลงทะเบียน และกดปุ่ม สมัคร    ขั้นที่5. การสมัครสมบูรณ์  มาถึงตรงนี้ ID ของคุณก็สามารถใช้เล่นเกมได้แล้วจ้า    ขั้นที่6. หลังจากนี้ ท่านสามารถใช้ ID และ Password ที่สมัครไว้ ในการเข้าเล่นเกม หรือเข้าสู่ระบบสมาชิกเข้าสู่ระบบเติมเงิน  หรือระบบกิจกรรมต่างๆ ที่หน้าเว็บไซต์ โดยไม่จำเป็นต้องทำการยืนยันอีเมล์ (Email Activation) อีกต่อไป               เท่านี้ ก็สนุกกับเกมเศรษฐี Everybody’s Marble กันได้แล้วจ้า การสมัคร Netmarble ID ผ่าน Facebook ง่ายๆ แบบนี้โดยไม่จำเป็นต้องทำการยืนยันอีเมล์ (Email Activation) ให้เสียเวลาแบบนี้ สะดวกสบายมากเลยล่ะ  หวังว่ามหาชนชาวเกมเศรษฐีจะถูกอกถูกใจในการสมัครNetmarble ID กันนะคะ เข้ามาเล่นกันเยอะๆนะเจ้าค่ะ ^^  หมายเหตุ: ทั้งนี้ หลังจากนี้เป็นต้นไป ในกรณีที่สมัครสมาชิกโดยวิธีปกติ (ที่ต้องใช้ Email ในการสมัคร และต้องผ่านการยืนยันอีเมล์) ทางระบบจะทำการลบข้อมูล ID ที่สมัครไม่สมบูรณ์ และไม่ทำการยืนยันอีเมล์ภายใน 7 วัน นับตั้งแต่วันที่สมัคร ออกจากระบบโดยอัตโนมัติ   ขอบคุณค่ะ ทีมงานเกมเศรษฐีเกมมหาชน
