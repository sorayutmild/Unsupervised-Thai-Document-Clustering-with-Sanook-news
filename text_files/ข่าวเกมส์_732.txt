Seal Online Return ขนทีเด็ดใหม่ๆมาเพียบ เอาใจแฟนซีลชาวไทย
Seal Online Return
Seal Online Return การกลับมาอีกครั้งของหนึ่งในเกมออนไลน์สุดอมตะที่คนไทยชื่นชอบมากที่สุด แม้ว่าตัวเกมจะเก่าถึงสิบกว่าปีแล้ว แต่ก็ยังมีแฟนๆติดตามเล่นอยู่ตลอด ถึงจะเปลี่ยนผู้ให้บริการไปแล้วหลายรอบก็ตาม และครั้งนี้นับว่าพิเศษกว่าครั้งก่อนๆ เพราะได้ผู้บริการที่เป็นเจ้าของเกม Seal Online อย่าง Playwith จากเกาหลีใต้ (ซื้อกิจการของ YNK เจ้าของเกม Seal Online เดิมมาอีกที) ตัดสินใจมาเปิดสาขาในประเทศไทย เพื่อเปิดให้บริการเกม Seal Online Return ด้วยตัวเอง ทาง Playwith จึงสามารถจัดเต็มๆ คอนเท้นต์ใหม่ๆทั้งคอนเท้นต์จากแพทช์ล่าสุด รวมไปถึงคอนเท้นต์ที่ทาง Playwith พัฒนาขึ้นเพื่อเซิร์ฟเวอร์ไทยโดยเฉพาะ
เกม Seal Online Return มีกำหนดเปิดให้เล่นในช่วง Open Beta ในวันที่ 28 มีนาคม 2560 นี้ เข้าไปดาวน์โหลดเกมรอกันได้แล้ววันนี้

3. คลาสอาชีพใหม่ Cook
คลาสใหม่สดซิง ที่เพิ่งเพิ่มเข้ามาใน Seal Online Return ไม่ใช่ยี่ห้อน้ำมันพืชในห้างสรรพสินค้า แต่เป็นคลาสอาชีพที่เชี่ยวชาญการทำอาหาร และใช้ท่าโจมตีที่เกี่ยวกับอาหาร เป็นอาชีพที่เล่นไม่ยากเพราะใช้การโจมตีระยะไกล แถมยังมีท่า Buff เพิ่มพลังด้วยการทำอาหารแบบต่างๆด้วย
Seal Online Return
เมื่อเพิ่มเลเวลแล้วสามารถเลือกเปลี่ยนคลาสสองได้ 2 อาชีพคือ
- Chef ก็คือกุ๊กระดับมืออาชีพ ที่เพิ่มความสามารถเข้าไปอีก อย่างการสร้างไอเทมอาหารต่างๆด้วยสกิล Chef Cooking เพื่อใช้ Buff แบบต่างๆได้ สามารถนำไปใช้หรือแจกให้สมาชิกในปาร์ตี้ได้ ส่วนท่าโจมตีก็ยังเป็นการโจมตีระยะไกลเหมือนกุ๊ก
Seal Online Return
- Food Fighter นักสู้พลังอาหาร เปลี่ยนท่าโจมตีจากระยะไกลมาเป็นระยะใกล้ ด้วยอาวุธวัตถุดิบอาหารที่ถือในมือทั้งสองข้าง มี Buff สองแบบให้เลือกเล่นว่าจะเล่นสายไหนคือ Binge บัพเพิ่มเลือดและพลังโจมตี แต่อัตราหลบหลีกลดลง สำหรับสายถึกตีแรง และ Diet บัพเพิ่มความเร็วโจมตี อัตราหลบหลีก แต่พลังป้องกันลดลง สำหรับสายพริ้ว
Seal Online Return
