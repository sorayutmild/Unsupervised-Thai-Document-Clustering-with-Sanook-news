เตรียมความพร้อมต้อนรับเพื่อนๆ !
เริ่มแรกด้วยการดาวน์โหลดโปรแกรม Hamachi Unturned เพื่อสร้าง Server ของตัวเองก่อน เมื่อดาวน์โหลดมาติดตั้งเสร็จสมบูรณ์แล้วให้ทำการคน Create new network เพื่อทำการสร้าง Server ของตัวเอง โดยใส่ Network ID และใส่รหัส 2 ครั้ง หลังจากนี้ก็จะสามารถส่งรหัสและ IP  Hamachi ของเราให้เพื่อนๆเข้ามาใน Server Hamachi กันได้เลย !

วิธีการเล่นแบบ Multiplayer เข้า Server เพื่อนๆ
Unturned
สิ่งแรกที่จะต้องมีคือ IP HAMACHI พร้อมรหัสผ่านของเพื่อน โดยเมื่อเปิดเข้าเกมมาให้คลิกที่ Play / Connect ใส่ Password พร้อม IP ส่วน Port แนะนำให้เป็น 25444 และอย่าลืมว่าเราต้องอยู่ในห้องของ Hamachi ที่เพื่อนเป็นคนสร้างด้วยนะครับ  ^^ เมื่อทำตามทุกอย่างครบหมดแล้วให้ทำการกด Connect ไปได้เลย เพียงเท่านี้ก็สามารถเล่นกับเพื่อนๆที่แสนน่ารักของเราได้แล้ววว"มาช่วยกันเอาชีวิตรอดในโลกที่โหดร้ายกันดีกว่า" ฮ่าๆ
วิธีการสร้าง Host 
ขั้นตอนการสร้าง Host ก็คล้ายกับการเข้าไปเล่นแบบ Multiplayer กับเพื่อนๆโดยการคลิกที่ Play / Host โดยสามารถปรับแต่งให้สามารถ PVP ได้ หรือไม่ได้ , ความยากง่ายของเกม , เลือกให้ซิงค์กับเซฟที่เราเล่นคนเดียวหรือไม่ , ใส่รหัสผ่านในการเข้าเล่น, จำนวนผู้เล่นสูงสุด และ Port โดยผู้เล่นจะต้องมี Host ของ Hamachi ที่สร้างไว้ก่อน เพื่อจะให้เพื่อนๆเข้ามาใน Hamachi ก่อนเริ่มทำการเข้ามาในเกม [แนะนำให้รหัสเข้า Hamachi กับเข้าเกมเป็นรหัสเดียวกัน] เมื่อทำการปรับแต่งสิ่งที่ต้องการทุกอย่างเสร็จแล้วให้กด Host เพื่อเริ่มเกม และรอให้เพื่อนๆเข้ามาเริ่มเล่นกันได้เลย
