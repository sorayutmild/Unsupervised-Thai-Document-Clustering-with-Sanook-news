FIFA Manager
ในตลาดเกมส์แนวฟุตบอลหรือ Soccer นั้นในอดีต FIFA ของ EA ความนิยมของเกมเมอร์เป็นรองคู่แข่งอย่าง Winning หรือ PES มาโดยตลอด แต่ด้วยการพัฒนาในด้านของระบบการเล่นและลิขสิทธิ์ทีมนักเตะอย่างจริงจัง ในที่สุด FIFA ก็สามารถเอาชนะครองใจคนเล่นเกมส์ฟุตบอลในยุคนี้แทนที่ PES ซึ่งย่ำอยู่กับที่ไม่ก็ถอยหลังลงคลองได้สำเร็จ นับเป็นความสำเร็จอย่างยิ่งใหญ่สำหรับ EA ที่สามารถขึ้นมาเป็นที่หนึ่งในวงการเกมส์ฟุตบอลจนได้ แต่ก็มีอีกอย่างหนึ่งซึ่งทาง EA ไม่สามารถเอาชนะได้สักที นั่นก็คือเกมส์ Football Manager จากค่าย SEGA จนสุดท้าย EA ก็ประกาศยอมแพ้ เลิกพัฒนา FIFA Manager ออกมาแข่งแล้ว
เกมส์ Football Manager พัฒนาโดยทีม Sports Interactive ผู้เชี่ยวชาญการทำเกมส์ผู้จัดการทีมฟุตบอลมายาวนานนับตั้งแต่สมัยปี 1992 พวกเขามีผลงานช่วงแรกชื่อว่า Championship Manager โดยตอนนั้นสังกัดค่าย Eidos จนกระทั่งหลายปีต่อมา เกิดการไม่ลงรอยกับ Eidos ทาง Sports Interactive จึงได้ยกเลิกสร้างเกมให้กับทาง Eidos และเปลี่ยนมาเข้าสังกัด SEGA และพัฒนาเกมส์ชื่อใหม่ว่า Football Manager 2005 ในปี 2004 ส่วน Eidos ได้หาทีมงานใหม่มาพัฒนา Championship Manager ต่อ แต่ว่าระบบการเล่นเทียบกับของทีม Sports Interactive ทำไว้ไม่ได้เลย สุดท้าย Eidos ก็ล้มละลายจนต้องเข้าร่วมกับ Square-Enix ในที่สุด ซึ่งค่ายเหลี่ยมก็ไม่เน้น Championship Manager แล้ว ทำออกมาไม่กี่ภาคเอง
หลังจากที่ Championship Manager ล้มเหลวไปแล้ว ก็เหลือเกมส์ผู้จัดการฟุตบอลในตลาดเพียงสองเจ้าก็คือ Football Manager ของ SEGA และ FIFA Manager ของ EA ซึ่งยอดขายและความนิยมนั้นเรียกว่า Football Manager ของ SEGA กินขาดทุกประการ สาเหตุเพราะระบบเกมส์ที่เล่นเข้าใจง่ายไม่ยุ่งยาก ต่างกับ FIFA Managerที่ระบบการเล่นเรื่องมากวุ่นวาย ซึ่ง FM 2013 มีข่าวว่าโดนโหลดเถื่อนไปเล่นมากถึง 10 ล้านคน

ภาคนี้ทำออกมาภาคสุดท้ายแล้ว
ด้วยความนิยมอย่างล้นหลามของ Football Manager ทำให้ทาง Gerald Köhler ผู้ให้กำเนิดเกม FIFA Manager ของ EA ถึงกับออกมาเปิดใจถึงสาเหตุที่ทาง FIFA Manager เอาชนะไม่ได้สักที และประกาศถอนตัวออกจากตลาดเกมส์ผู้จัดการทีมฟุตบอล และยังอวยพรให้ Football Manager ประสบความสำเร็จในเกมส์แนวนี้ต่อไปด้วย เพราะเหลือดังๆแค่เกมเดียวแล้วนี่ และคุณ Gerald Köhler ได้ขอบคุณแฟนๆเกมส์ทุกคนที่อุตหนุน FIFA Manager มาตลอด 13 ปี (ตั้งแต่ตอนแรกใช้ชื่อว่า Fussball Manager และ Total Club Manager)
สำหรับใครที่เป็นแฟนเกม Football Manager ก็พยายามอุดหนุนของแท้กันบ้างนะครับ ก่อนที่จะไม่มีเกมส์แนวผู้จัดการฟุตบอลดีๆให้เล่นไปตลอดกาล
