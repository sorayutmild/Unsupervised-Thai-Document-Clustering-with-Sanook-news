สูตรโกงหรือ Cheat Code เป็นตัวช่วยให้เหล่าเกมเมอร์ผู้เล่นไม่เก่ง หรือหัดเล่น ได้สามารถทดลองเล่นเกมส์ต่างๆได้ง่ายขึ้น ซึ่งส่วนมากเป็นสูตรที่ทางค่ายเกมส์ทำมาให้ไว้ในตัวเกมส์อยู่แล้ว โดยมักเป็นสูตรกดปุ่มหรือไม่ก็สูตรที่พิมพ์รหัสลับลงไป เพื่อปลดล็อคโหมดต่างๆนั่นเอง ซึ่งตัวเกมส์ในปัจจุบันนี้ก็ยังพอจะมีสูตรโกงในเกมส์ให้เห็นกันบ้าง แต่ก็น้อยลงไปมากเมื่อเทียบกับเกมส์เก่าๆ แถมสมัยนี้ยังมีสูตรโกงแถม Hack ข้อมูลเกมส์มาแทนที่มากขึ้น อย่างพวก Emu หรือ Action Replay ทั้งหลาย ก็เลยทำให้สูตรโกงแบบกดเองมีน้อยลงทุกทีๆ  ด้วยเหตุนี้เราจึงมารำลึกความหลังกันหน่อย ว่าสูตรโกงโคตรคลาสสิคจากเกมส์เก่าเหล่านี้ คุณรู้จักกันหรือเปล่า

1. The Konami Code
ผมเชื่อว่าเหล่าเกมเมอร์วัยดึกทั้งหลายเกือบร้อยเปอร์เซนต์ จะต้องรู้จักกับสูตรกดมือโคตรยอดนิยมสูตรนี้ บนๆ ล่างๆ ซ้ายขวาๆ B A Select Start (ถ้าจะเล่นคนเดียวก็ไม่ต้องกด Select ) ซึ่งสามารถนำไปใช้ในเกมส์ของค่าย Konami ได้หลายเกมส์มาก ไม่ว่าจะเป็นสูตร Contra 30 ตัว สูตรปลดออฟชั่นทุกอย่างในเกมส์ยานยิง Gradius แต่ส่วนมากคงใช้กับ Contra กันมากกว่า เพราะมันเป็นเกมส์แอคชั่นสุดฮิตตลอดกาล

2. God Mode สูตรอมตะ
ในเกมส์ PC สมัยก่อนๆ ที่โคตรยากกว่าเกมส์สมัยนี้เยอะ มีน้อยคนนักที่จะจบเกมส์ได้แบบไม่โกง ดังนั้นผู้ผลิตเกมส์ก็เลยมีสูตรโกงทำให้ตัวละครอมตะ หรือที่เรียกว่า God Mode ให้ใช้กัน สำหรับเกมส์ PC เก่าๆที่นิยมใช้ God Mode กันเยอะก็มี Doom ที่เมื่อพิมพ์คำว่า IDDQD ลงในช่องใส่สูตร ตัวละครของคุณก็จะเป็นอมตะดุจเทพ มีพลังเต็ม 100 และมีเกราะถึง 200 ตลอดเวลา ไม่ว่าหน้าไหนก็ฆ่าตูไม่ได้ นอกจากนี้ก็มี God Mode ยอดนิยมในเกมส์ตระกูล Warcraft อย่าง WhosYourDaddy แปลเถื่อนๆได้ว่า รู้มั่งใครพ่อมึง เพียงเท่านี้ฮีโร่เราก็จะอมตะไร้เทียมทาน ใครจำสูตรอมตะเกมส์เก่าๆได้อีกก็บอกเพิ่มได้เน้อ
