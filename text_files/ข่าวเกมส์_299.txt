ผลกระทบต่อเกมเมอร์ หากไทยใช้ Single Gateway
Single Gateway
ภาพจาก https://sc1.checkpoint.com
หัวข้อที่เป็น Talk of the town ในช่วงนี้ของประเทศไทย โดยเฉพาะกับชาวโลกออนไลน์ นั่นก็คือเรื่อง Single Gateway นโยบายประตูบานเดียวที่รัฐบาลวางแผนจะเปลี่ยนระบบอินเตอร์เน็ตของไทย ให้อยู่ภายใต้การควบคุมได้ง่ายขึ้น แน่นอนว่ามันมีข้อดีตรงรัฐบาลสามารถควบคุมการใช้งานอินเตอร์เน็ตได้มากขึ้น และสามารถจับอาชญากรในโลกไซเบอร์ได้สะดวก แต่มันก็มาพร้อมกับข้อเสียมากมายเช่นกัน ไม่อย่างนั้นชาวเน็ตทั่วไทยคงไม่ประท้วงกันอยู่ตอนนี้
แต่ที่เราจะมาพูดกันวันนี้จะเกี่ยวข้องกับ "เกม" ล้วนๆ ว่าหากสมมติว่าไทยเราเปลี่ยนไปใช้ Single Gateway จริงๆ มันจะมีผลกระทบอะไรกับการเล่นเกมบ้าง ตรงนี้ต้องขอบอกตรงๆเลยว่า มีผลกระทบ "มากกกก" เพราะการเล่นเกมสมัยนี้มันต้องอาศัยการเชื่อมต่อออนไลน์เยอะ เราไม่ได้เล่นมอญซ่อนผ้า หรือแม่งูเอ๋ย แบบสมัยก่อนๆ แต่มันกระทบอย่างไรบ้างล่ะ จะมาแยกเป็นข้อๆให้ดูง่ายๆ
Single Gateway
1. กระทบต่อเกมมือถือ
ตลาดเกมมือถือมาแรงมากในปัจจุบัน จากปัจจัยที่เข้าถึงง่าย เล่นสะดวก และเล่นฟรี แต่อย่าลืมว่าเกมเล่นฟรีเหล่านั้นก็แฝงเงื่อนไขบางอย่าง นั่นก็คือการบังคับให้เราเชื่อมต่อออนไลน์เล่นเกมตลอดเวลา ทั้งตอนเข้าเกม ตอนแชร์ ตอนจะซื้อของ ฯลฯ ขนาดตอนยังไม่เปลี่ยนระบบ คนเล่นมือถือยังบ่นตลอดว่า "เล่นแล้วหลุด" นั่นก็เพราะเกมฟรีพวกนี้ หากเน็ตไม่เสถียรหรือไม่แรงพอก็จะทำให้หลุดจากเกม
แล้วถ้าเกิดไทยเปลี่ยนไปใช้ Single Gateway คิดหรือว่าประตูบานเดียว มันเพียงพอต่อการใช้งานคนเล่นมือถือของไทย ที่นับวันจะเพิ่มมากขึ้นๆ ยิ่งเกมมือถือส่วนมากเป็นเกมต่างประเทศที่เราโหลดจาก App Store ก็ทำให้มีโอกาสเล่นแล้วหลุดมากขึ้น หาก Single Gateway ใหญ่ไม่พอใช้งาน จะมีก็เพียงเกมมือถือที่เซิร์ฟเวอร์อยู่ในไทยเท่านั้น ที่ไม่มีผลกระทบเท่าไหร่

2. กระทบข้อมูลส่วนตัว
อย่างที่กล่าวไปว่า การใช้ Single Gateway ทำให้ทางรัฐบาลหรือหน่วยงานที่ควบคุม สามารถตรวจสอบข้อมูลการใช้งานอินเตอร์เน็ตได้ทั้งหมด นั่นหมายความว่าข้อมูลส่วนตัวของคุณก็ถูกตรวจสอบและมีคนอื่นล่วงรู้อยู่ด้วย คนเล่นเกมก็ไม่ใช่ว่าจะเป็นคนเล่นเกมฟรีไปซะทุกคน เพราะบางเกมก็มีการซื้อของในเกม หรือจะซื้อเกมจาก App Store หรือ Steam ก็ต้องใช้การผูกบัตรเครดิต หรือใส่ข้อมูลการเงินส่วนตัวเข้าไป
ทีนี้เราจะมั่นใจได้อย่างไร ว่าคนดูแลข้อมูลพวกนี้ไว้ใจได้? ข้อมูลการเงินประชาชนจะไม่รั่วไหล ขนาดเรื่องอื่นๆยังมีคอรัปชั่นกันกระจาย ถ้าเกิดให้ FBI หรือ CIA มาคุม อันนี้ยังพอไว้ใจได้

3. กระทบคนเล่นเกมออนไลน์
Single Gateway คือประตูการใช้งานเน็ตทางเดียว หากล่มก็ล่มกันหมดทั้งประเทศ ไม่มีช่องทางสำรอง ซึ่งประตูบานเดียวที่รัฐบาลจะใช้ ก็ไม่มีอะไรรับรองว่ามันจะคุณภาพดีแค่ไหน เพราะ Single Gateway ที่คุณภาพดีมากๆ ต้องใช้งบประมาณมหาศาล หากใช้ Single Gateway ห่วยๆ เน็ตก็จะล่มบ่อย คนเล่นเกมออนไลน์จำเป็นต้องเชื่อมต่อกับเกมตลอดเวลา หากเน็ตล่ม เกมหลุด ตอนบอสตาย แรร์ออก แต่อดเก็บ ใครจะรับผิดชอบ?
และที่แย่สุดๆก็คือ คนที่เล่นเกมออนไลน์ต่างประเทศ ที่จำเป็นต้องใช้เน็ตแรงๆ เนื่องจาก Single Gateway ทำให้ความเร็วและความมั่นคงของเน็ตออกต่างประเทศลดลง จึงทำให้เล่นเกมออนไลน์เซิร์ฟเวอร์ต่างประเทศได้ไม่ดี

4. กระทบเกม Rate 18+
เกม Rate 18+ ในที่นี้ เราไม่ได้หมายถึงเกมโป๊อนาจาร หรือ H-Game แค่อย่างเดียว แต่หมายถึงเกมที่มี Rate ความรุนแรง ไม่เหมาะสมกับเด็กด้วย โดยเมืองนอกจะมีเรตแบ่งความเหมาะสมกับอายุของผู้เล่นไว้ ตัวอย่างเกมรุนแรง ที่ 18+ ก็อย่างเช่น Grand Theft Auto ที่เคยมีข่าวก่อคดีแล้วโทษเกมไปเมื่อหลายปีก่อน กรณีหากรัฐบาลใช้ Single Gateway ก็จะสามารถบล็อคเกมต่างๆได้สะดวกขึ้น ทำให้เราอาจไม่สามารถเข้าเล่นเกมบางเกมได้ หากรัฐบาลทำการบล็อคเกมนั้นๆไป โดยให้เหตุผลว่า "เนื้อหาเกมไม่เหมาะสม"
