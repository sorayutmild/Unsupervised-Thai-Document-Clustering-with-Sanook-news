WeChat-โปรแกรมแชทใหม่สุดฮิต!
     ช่วงสองวันที่ผ่านมานั้นทีมงาน Sanook Hi-Tech เห็นกระแสใหม่ที่กำลังเป็นที่สนใจของเหล่า ดารา ดีเจ เซเล็บฯ ชื่อดังหลายๆคนหันมาเล่น app ใหม่ที่ชื่อว่า WeChat กันหลายต่อหลายคน เลยต้องขอนำมาแนะนำให้รู้จักเจ้า WeChat กันว่ามีดียังไงบ้าง


     ความพิเศษของ WeChat ที่เค้ายืนยันว่าเป็นแอพพลิเคชั่นที่มีฟังชั่นเหนือกว่า Whatsapp,Line ,voxer,cubie,path และอื่นๆอีกมากมาย ถือได้ว่าเป็น Social App ที่ฟังก์ชั่นครบครันที่สุดในปัจจุบัน อีกทั้งยังเป็นที่นิยมในกลุ่มดาราและเหล่าเซเลบฯอีกในขณะนี้ ตั้งแต่ดีเจนุ้ย เนโกะจั้ม แนนนี่เกอลลี่เบอร์รี่ และอีกหลายต่อหลายคนก็มาเล่น WeChat กันอย่างมากมาย

ความสามารถของ WeChat นอกจากสามารถส่งข้อความตัวอักษร,ข้อความเสียง,รูปภาพ,วิดิโอ และ Location ของคุณได้แล้วนั้น ยังมีอีกหลายฟังก์ชั่นที่เป็นเอกลักษณ์เฉพาะของWeChat อีกมากมาย อย่างเช่น สามารถแบ่งปันภาพถ่ายไปยังกลุ่มเพื่อนในรายชื่อ หรือเขย่าโทรศัพท์หาว่ามีเพื่อนคนไหนอยู่บริเวณใกล้เคียงเราบ้าง เรามาลองดูดีกว่าว่ามันสนุกขนาดไหน! 
 มาดูวิธีการใช้งาน WeChat แบบง่ายๆ กัน
 
      หลังจากโหลดเจ้า WeChat มาแล้วก็เริ่มใช้งานกันเลย  โดยทั่วไปจะปรากฏหน้าให้เห็นรายชื่อเพื่อนที่คุณกำลังติดต่ออยู่ในขณะนี้
       แต่ถ้าหากว่านี่คือการเข้าสู่ระบบครั้งแรก คุณจะเห็นWechat teamเข้ามาทำการทักทายกับคุณ คุณสามารถติชมหรือให้คำแนะนำเกี่ยวกับผลิตภัณฑ์ตัวนี้กับWechat Teamได้ เพื่อนของคุณที่เล่น WeChat สามารถ add กันได้ง่ายๆ และอย่าลืมชวนคนอื่นๆของคุณที่อยู่ในสมุดรายชื่อของคุณจะได้รับการเชิญจากคุณผ่านข้อความแนะนำ หรือจะเลือกชวนเพื่อนจากใน Facebook ที่เล่น WeChat เหมือนกันก็ยังได้
 การคุยกับเพื่อนนอกจากพิมพ์ข้อความแล้วยังสามารถส่งเสียงได้ด้วย

วิธีการส่งเสียงไปหาเพื่อนของคุณไม่ต่างกับการใช้ Walkie Talkie มันช่างแสนง่ายและสะดวก  เพียงแค่คุณเลือกเพื่อนที่คุณต้องการจะสนทนาด้วยด้วย เพียงเท่านั้นก็คุณก็ได้เข้าสู่การสนทนาแล้ว หากคุณต้องการสนทนาด้วยเสียง คุณเพียงกดปุ่ม”กดเพื่อพูด”ค้างไว้ เมื่อพูดจบจากนั้นก็ปล่อย เพียงเท่านี้เพื่อนของคุณก็จะได้รับข้อความเสียงและตอบกลับได้ทันทีประหยัด ค่าโทรไปหลายตังเลย


นอกจากจะรองรับการสนทนาด้วยเสียงแล้ว  คุณยังสามารถส่งภาพ วิดิโอ Locationของคุณ และภาพถ่ายน่ารักๆอีกมากมาย  ข้างล่างคือภาพตัวอย่างการส่งภาพและ Location ที่ทำได้ง่ายมากๆ 



ถ้าตกลงกับเพื่อนไม่ได้ว่าใครจะเป็นคนตัดสินใจ ก็เป่ายิ้งฉุบกันไปเลย



กำหนดเพิ่มอีโมติคอนด้วยตัวคุณเอง-Wechatคุณสามารถกำหนดอิโมติคอนได้ด้วยตัวคุณเอง—เพียงเท่านี้คุณก็จะมีอิโมติคอนที่ไม่ซ้ำใครแล้ว!



นอกจากนี้ ฟังชั่น "กลุ่มเพื่อน" ที่ทำงานคล้ายๆกับ Path สามารถทำให้คุณสามารถถ่ายรูปและอัพโหลดรูปหรือจะตกแต่งภาพก่อนโพสก็สามารถทำได้ เพื่อนๆ ของคุณเท่านั้น ที่จะเข้ามาดูได้



คุณสามารถแบ่งปันรูปภาพอาหาร หรือภาพเก๋ๆ สวยๆ กับเพื่อนของคุณได้ 



การเพิ่มเพื่อนทำได้ง่ายๆ เพียงแค่คุณเข้าไปที่ “Social” คุณจะสามารถเลือกเพิ่มเพื่อนจาก Contacts, และหาเพื่อนจากคนที่อยู่ใกล้ๆได้ด้วย



 คุณสามารถเห็นเพื่อนที่อยู่ใกล้ๆคุณปรากฏอยู่



สำหรับการเขย่าเพื่อเพิ่มเพื่ือน ถ้าคุณและกลุ่มเพื่อน โหลด wechat และอยู๋ใกล้ๆ กัน สามารถเขย่าพร้อมกันและทุกคนที่เขย่าโทรศัพท์ของคุณ คุณก็จะสามารถ เพิ่มเพื่อนได้เช่นกัน



เขียนความรู้สึกของคุณใส่ลงในขวดจากนั้นโยนลงไปในทะเล  บางทีอาจจะมีใครเห็นเข้าก็ได้นะ



ในการตั้งค่าคุณยังสามารถเปลี่ยนภาพและสเตตัสของคุณได้ อีกทั้งยังสามารถสร้างมีอัลบัมภาพสวยๆของคุณเองอีกด้วย

WeChat ในปัจจุบันสามารถรองรับระบบปฏิบัติการ IOS, Andriod และ Windows Phone แต่เพื่อการตอบสนองความต้องการของผู้ใช้บริการ  ในปลายเดือนพฤษภาคม 2555 ทางTencentจะเปิดให้บริการสำหรับโทรศัพท์มือถือที่เป็นระบบปฎิบัติการSymbian อย่าลืมติดตามกันน่ะ Plug-in API  เป็นการประสิทธิภาพ การใช้ WeChat ให้สนุกยิ่งขึ้น WeChat เป็นแอพพลิแคชั่นแรกของโลกที่เปิดให้ผู้ให้บริการ Application อื่นๆ มาผูกกับ WeChat ได้สามารถเพิ่ม  API ด้วย 

ซึ่งในอนาคต WeChat ยังสามารถแบ่งบัน ข่าว, ดูดวง และจะสามารถเล่นเกมส์ร่วมกันได้อีกด้วย ดังตัวอย่าง หากคุณดาวน์โหลด QQ Image Appลงบนโทรศัพท์มือถือ คุณก็สามารถใช้ WeChat ร่วมกับ QQ Imageได้   นั่นคือคุณสามารถนำภาพที่ตกแต่งจาก QQ image มาแบ่งปันกับเพื่อนของคุณใน WeChat ได้  ซึ่งAPIในขณะนี้รองรับแอพลิเคชั่น QQ Image เท่านั้น แต่อีกไม่นานจะมีแอพพลิเคชั่นอีกมากมายเพิ่มเข้ามาให้คุณได้ใช้ร่วมกันเราลองมาดูฟังก์ชั่นรายชื่อ WeChat ซึ่งไม่เพียงแต่สามารถสนทนาได้กับรายชื่อเพื่อนที่อยู่ในโทรศัพท์มือถือหรือช่องทางที่กล่าวมาแล้วได้เท่านั้น คุณยังสามารถเพิ่มรายชื่อเพื่อนเพื่อเป็นการคุยหลายๆคน หรือ สนทนากลุ่ม ได้ทันทีเพียงแค่คลิ๊กเพิ่มเพื่อนที่คุณต้องการดึงเข้าสู่วงสนทนาเท่านั้นเอง
