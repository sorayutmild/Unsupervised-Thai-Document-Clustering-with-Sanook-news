Temple Run 2
Temple Run เกมส์วิ่งๆๆ แนวสปีดรัน ที่ต้องวิ่งไปถึงเป้าหมายให้ทันเวลา โดยที่ระหว่างทางจะต้องกระโดดหลบสิ่งกีดขวางไปด้วย เหมือนกับเกมส์ Pepsi Man สมัยเครื่อง PS1 นั่นแหละ ว่ากันว่า Pepsi Man เป็นเกมส์แรกๆ ที่นำเสนอความสนุกของเกมส์แนวสปีดรัน จนมาเป็น Temple Run ในปี 2011 ที่มียอดดาวน์โหลดไปเล่นสูงสุดๆถึง 170 ล้านครั้ง และวันนี้ Temple Run 2 ก็ออกมาให้คุณได้ท้าทายความเร็วในการวิ่งต่อแล้ว
 
Temple Run 2 มีการปรับเปลี่ยนกราฟิกให้ดูดีขึ้น จากภาคแรกที่กราฟิกค่อนข้างหยาบ ก็ทำให้ดูสวยและเรียบเนียนกว่าเดิม พร้อมทั้งเพิ่มรายละเอียดฉากรอบๆ มี power up แบบต่างให้เลือกมากขึ้น ตามแต่ละตัวละครที่เราเลือกเล่น รวมไปถึง achievements ใหม่ๆ และเจ้าลิงที่ตามล่าตัวเอกในภาคนี้ ก็ดูโหดมากกว่าเดิมมากด้วย
