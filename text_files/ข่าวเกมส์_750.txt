สวัสดีเพื่อน ๆ นักโดดร่ม ช่วงนี้เกม PUBG MOBILE ของเรากำลัง Hot สุด ๆ กิจกรรมมาเพียบ ไม่ว่าจะเป็นการอัปเดตแพทช์ใหม่ กิจกรรมฉลองครบรอบ 3 ปี แถมยังมีการแข่งขัน PMPL SS3 หรือ PUBG MOBILE THAILAND PRO LEAGUE 2021 อีกด้วย และสิ่งหนึ่งที่เพื่อน ๆ นักโดดร่มจะได้รับกันทุกคนเลยก็คือ บรรดา Item Code แจกของรางวัลต่าง ๆ ที่เพื่อนๆจะหาได้จากหลายช่องทาง ไม่ว่าจะเป็นในเพจของเกม PUBG MOBILE เอง หรือจากกิจกรรมนอกอื่น ๆ อย่างเช่นกิจกรรมสงกรานต์เร็ว ๆ นี้
ขั้นตอนการใช้งาน Item Code ของเกม PUBG MOBILE
1) เมื่อได้รับ Item Code มาแล้วให้ไปที่เว็บไซต์ www.pubgmobile.com/redeem/global

2) กรอก Character ID โดยเข้าไปดูได้จากหน้าจอในเกม (ดังภาพ)

3) นำ Item Code ที่ได้กรอกในช่อง Redemption code
4) กรอก Verification code เป็นตัวเลข หรือตัวอักษร 4 หลัก แล้วกด REDEEM
5) จากนั้นระบบจะทำการยืนยัน Character ID และไอเทมที่จะได้รับ หากถูกต้องก็ให้กดยืนยัน และเข้าเกมไปรับของจากในกล่องจดหมายได้เลย 
หมายเหตุ: บทควาทนี้เป็นวิธีการใช้ Item Code ของเกม PUBG MOBILE เอง ไม่ใช่การใช้งานโค้ดจากบริษัทเติมเกมต่างๆ
