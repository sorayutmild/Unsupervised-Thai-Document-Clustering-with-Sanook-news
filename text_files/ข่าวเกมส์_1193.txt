Need for Speed ภาคใหม่ขอ'ย้อนรอย' กลับไปแบบเก่าๆ
Need for Speed
EA (Electronic Arts) ประกาศเกม Need for Speed ภาคใหม่สำหรับชาว PC และคอนโซล PS4 และ Xbox one อย่างเป็นทางการในปีนี้ หลังจากปีที่แล้วงดสร้างไปหนึ่งปี เพื่อไปทำลงภาคมือถือแทน ภาคใหม่นี้ ทาง EA เผยว่าเป็นการ Reboot คิดใหม่ทำใหม่ทั้งหมด โดยมีแนวโน้มว่าจะกลับไปทำเป็นแนวภาคเก่าๆอย่าง Need For Speed Underground จากตัวอย่าง Teaser แรก จะเห็นได้ว่าเป็นการแข่งรถใต้ดินตอนกลางคืน โดยมีตำรวจไล่จับ แต่เพิ่มฉากที่เป็นแบบ Open World เข้าไปด้วย

ตอนนี้ทาง EA ยังไม่เปิดเผยข้อมูลของตัวเกม Need for Speed ภาคใหม่นี้ออกมามากนัก เพราะจะไปเปิดตัวอย่างเป็นทางการอีกทีในงาน E3 2015 กลางเดือนมิถุนายนนี้ ตอนนี้ทราบเพียงว่าภาคนี้พัฒนาโดยทีม Ghost Game เป็นทีมงานที่เคยช่วยทำภาค Need for Speed Rivals กับทีม Criterion Games นั่นเอง
Need for Speed
Need for Speed ภาคใหม่ ทำลงเครื่อง PC, PS4 และ Xbox one กำหนดวางจำหน่าย ประมาณเดือนพฤศจิกายน 2015
