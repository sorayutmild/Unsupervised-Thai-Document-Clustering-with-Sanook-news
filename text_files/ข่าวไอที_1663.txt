นักพัฒนา พบวิธีเปลี่ยนไอคอน Facebook Reactions ให้เป็นรูปอื่นๆ หรือใบหน้าบุคคลที่ชื่นชอบได้แล้ว!

หลังจากที่ทาง Facebook ได้เปิดตัวฟีเจอร์ Facebook Reactions กับปุ่มแสดงความรู้สึกแบบใหม่ ที่มีให้เลือกกันถึง 5 แบบ นอกเหนือจากการกด Like แบบปกติ ล่าสุด ได้มีนักพัฒนา แกะโค้ด แล้วเปลี่ยนเป็นไอคอนอื่น หรือใบหน้าของบุคคลที่ชื่นชอบได้แล้ว

         โดยนักพัฒนาคนดังกล่าว มีชื่อว่า François Grante ได้ลองแกะโค้ดบนหน้าเว็บ Facebook แล้วเปลี่ยน Emoji บน Facebook Reactions ให้เป็นใบหน้าของ Donald Trump ผู้ที่เสนอชื่อเข้ารับการคัดเลือกเป็นประธานาธิบดีของสหรัฐฯ นั่นเอง
         ซึ่งไอคอนนี้ เป็นส่วนเสริมบน Google Chrome (Chrome extension) ที่ทาง François Grante ได้ปล่อยให้ดาวน์โหลดบน Chrome Web Store นั่นหมายความว่า ผู้ใช้ท่านใดที่ไม่ได้ดาวน์โหลดส่วนเสริมดังกล่าวนี้ ก็จะมองเห็นเป็นไอคอนแบบปกติ แต่ถ้าหากดาวน์โหลดไปใช้งาน ก็จะมองเห็นเป็นไอคอนรูปใบหน้าคนนั่นเอง

         นอกจาก Donald Trump แล้ว ยังมีรูปของ Mark Zuckerberg ผู้ก่อตั้ง Facebook อีกด้วย ส่วนท่านใดอยากจะลองแก้ไขเป็นไอคอนของตัวเอง ด้าน François Grante ได้แนะวิธีการทำ สามารถดูได้จาก Github ครับ
