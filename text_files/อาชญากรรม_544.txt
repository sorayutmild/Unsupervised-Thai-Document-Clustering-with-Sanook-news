ครอบครัวผู้เสียชีวิตจากเหตุระเบิดบริเวณสี่แยกราชประสงค์ เข้ามาติดต่อรับศพผู้เสียชีวิตและสอบถามรายละเอียดลักษณะของผู้เสียชีวิต โดยบรรยากาศเป็นไปอย่างโศกเศร้า
18 ส.ค. - พลตำรวจตรี นายแพทย์พรชัย สุธีรคุณ ผู้บังคับการสถาบันนิติเวชวิทยา โรงพยาบาลตำรวจ ระบุว่า ขณะนี้ศพที่อยู่ที่สถาบันนิติเวช โรงพยาบาลตำรวจ มี ทั้งหมด 17ราย และอยู่ที่โรงพยาบาลจุฬาฯ จำนวน 3 ราย และหนึ่งในนั้นเป็นเด็กหญิงยังไม่ทราบสัญชาติ ขณะเดียวกัน ญาติของผู้เสียชีวิต ได้เข้ามายืนยันชื่อบางส่วนแล้ว และในจำนวนนี้ที่สามารถยืนยันสัญชาติได้แล้วนั้น พบว่า มีคนไทยจำนวน 5คน ส่วนที่เหลือเป็นชาวต่างชาติ
นอกจากนี้ ยังพบชิ้นส่วนศพอีก 2 รายที่ยังไม่สามารถยืนยันได้ ส่วนญาติหรือครอบครัวที่จะเข้ามาติดตาม ขอให้นำข้อมูลอาทิ ประวัติการทำฟัน รูปร่างลักษณะการแต่งกายในวันเกิดเหตุเพื่อมายืนยันตัวบุคคล ซึ่งสาเหตุการเสียชีวิตส่วนใหญ่ผู้ตายเสียชีวิตจากแรงอัดของระเบิด
นอกจากนี้ พลตำรวจตรี นายแพทย์พรชัย ยังกล่าวอีกว่า ทางนิติเวชได้ระดมแพทย์นิติเวชทั้งหมดมาทำการตรวจสอบศพของผู้เสียชีวิต ซึ่งจะเร่งตรวจพิสูจน์ทราบสาเหตุการเสียชีวิต โดยคาดว่าจะแล้วเสร็จภายในวันนี้ และหากการตรวจเสร็จสิ้นก็สามารถส่งศพให้ญาติไปบำเพ็ญกุศลทางศาสนาได้ทันที
สำหรับรายชื่อผู้เสียชีวิตทั้ง 20 ราย นั้น ประกอบด้วย นางสาวสุดชาดา นิสีดา ชาวไทย / mr.neoh jaijun ชายชาวมาเลเซีย/ mrs.lioe lie tging ชาวอินโดนีเซีย / ชายไม่ทราบชื่อ / miss.pang,wan chee arcadia ชาวฮ่องกง / หญิงไม่ทราบชื่อ/ mra.melisa liu rui chum ชาวสิงค์โปร์ /ชายไม่ทราบชื่อ / นางสาววราภรณ์ ช่างทำ ชาวไทย/ นายยุทธณรงค์ สิงห์รอ ชาวไทย / ชิ้นส่วนลำตัวบน / กะโหลกศีรษะ /mr.diei chengi ต่างชาติ / นายสุวรรณ สัตย์มั่น ชาวไทย / gao yuzhu ชาวจีน / miss lim saw sek มาเลเซีย / miss chan wingyan vivian ชางฮ่องกง / หญิงไม่ทราบชื่อ / หญิงไม่ทราบชื่อ และ เด็กหญิงไม่ทราบชื่อ /
ขณะที่ นางสาวกฤษสุดา ณรงเปลี่ยน ภรรยาของนายยุทธณรงค์ สิงห์รอ หนึ่งในผู้เสียชีวิตยังคงอยู่ในความโศกเศร้า ต่อจากนี้ไปตนก็ไม่รู้จะใช้ชีวิตอยู่ได้อย่างไร
