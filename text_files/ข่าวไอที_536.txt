Apple เปิดตัว iPhone 5 แล้วจ้า...ใน Source Code ของ iOS 5.1 เวอร์ชันล่าสุด! iPhone 5 - อัพเดทข่าวล่าสุด เพียงไม่นานหลังจากที่ Apple ปล่อยระบบปฏิบัติการ iOS 5.1 Beta ให้กับเหล่านักพัฒนาได้สัมผัสก่อนใครก็ถือเป็นโอกาสอันดีที่พวกเขาเหล่านี้จะได้ทำการล้วงลับตับแตก Source Code ภายในเพื่อค้นหาหลักฐานเกี่ยวกับอุปกรณ์ใหม่ๆจาก Apple ซึ่งล่าสุดมีรายงานเข้ามาแล้วว่าใน Source Code ภายใน iOS 5.1 Beta ปรากฏรายชื่อของผลิตภัณฑ์รุ่นใหม่จาก Apple อย่าง iPhone 5 ที่ใครหลายคนกำลังเฝ้าฝันถึงกันแล้ว

โดย Source Code ของ iOS 5.1 Beta มีการเปิดเผยรหัสผลิตภัณฑ์ iPhone 5, 1 ซึ่งไม่น่าจะหนีไปจาก iPhone 5 ที่ Apple กำลังซุ่มพัฒนาอยู่นั่นเอง โดย iPhone 5 นั้นจะมาพร้อมกับหน้าจอขนาด 4 นิ้วที่ใหญ่ขึ้นกว่าเดิม รวมถึงชิปเซ็ตประมวลผลภายในเครื่องที่ดีขึ้นกว่าเดิม (คาดว่าน่าจะเป็น Apple A6 แบบ Quad Core CPU)

ทั้งนี้สำหรับลำดับตัวเลขประจำผลิตภัณฑ์ iPhone รุ่นก่อนหน้า iPhone 5 ที่เคยปรากฏอยู่ใน Source Code ของ iOS เวอร์ชันก่อนๆประกอบด้วย

iPhone1,1 - iPhone รุ่นแรก
iPhone1,2 - iPhone 3G 
iPhone2,1 - iPhone 3GS 
iPhone3,1 - iPhone 4 
iPhone4,1 - iPhone 4S
