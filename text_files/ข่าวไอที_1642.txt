    ปัญหาใหญ่สำหรับคนใช้งาน iOS หรือ iPhone ที่คุณรักทั้งหลายว่า ถ้าเกิดต้องเปลี่ยนเครื่องไปแล้วคุณเกิดลืม Apple ID ซึ่งเรียกได้ว่าเป็นชีวิตส่วนหนึ่งของคุณเลย แล้วคุณจะทำอย่างไร เรื่องนี้ใหญ่ยิ่งกว่าลืม Password ซะอีก แต่วันนี้ Sanook! Hitech มีคำแนะนำและวิธีแก้ไขสถานการณ์เมื่อคุณลืม Apple ID ที่แสนง่ายดายมาฝากกัน

    แค่คุณเข้าไปที่เว็บไซต์ appleid.apple.com/th แล้วกดคำว่า ลืม Apple ID หรือ รหัสผ่าน


    กรอกข้อมูลให้ครบทั้งชื่อ นามสกุล และอีเมล์ ที่ไม่จะเป็นต้องเป็นอีเมล์ที่สมัคร Apple ID หลังจากนั้นระบบจะส่งอีเมล์กลับมาให้คุณ ว่า Apple ID ของคุณคือ เมล์อะไร
    หมายเหตุ กรณีลืมรหัสก็สามารถใช้เว็บไซต์นี้แก้ไขได้เช่นเดียวกัน

    กรณีที่คุณมี iOS หลายเครื่อง ให้เครื่องอื่นช่วยกู้ Apple ID ได้เพียงแค่ เข้าไปที่ ตั้งค่า (Setting) > iTunes & Apps Store จะปรากฏ Apple ID ของคุณให้ชัดเจน
    แค่นี้คุณก็สามารถได้ Apple ID สำหรับไว้ลงมือถือเครื่องใหม่ แต่ข้อมูลทุกอย่างยังอยู่ครบแล้วครับ
