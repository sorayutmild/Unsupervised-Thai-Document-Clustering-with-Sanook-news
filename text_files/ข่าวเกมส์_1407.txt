ย้อน กลับไปในสมัยยังเด็กๆ  ที่ยังไม่สามารถหารายได้ด้วยตนเองและที่บ้านอาจจะฐานะการเงินไม่ค่อยจะสู้ดี   กว่าจะมีเครื่องเกมเล่นซักเครื่องต้องใช้ความเพียรพยายามมากในการหามาเป็น เจ้าของ แต่ไอ้พวกผู้ผลิตก็ดั๊นนนนน มาผลิตอุปกรณ์เสริมล่อตายั่วใจกันอีก  ด้วยภูมิปัญญาของเจ้าแห่งเทคโนโลยีอย่างญี่ปุ่น  ก็สร้างนวัตกรรมชวนให้อดข้าวกระเป๋าแฟ่บเหลือเกิน  เราไปดูกันว่าเจ้าอุปกรณ์เสริมเหล่านั้นมีอะไรกันบ้าง   1. Multitap
เกมส์
      ในยุคสมัยนั้นการเล่นเกมได้พร้อมกัน 4  คนถือว่าเป็นเรื่องที่พระเจ้าจอร์จมาก ดังนั้นก็เกิดกิเลศทนแรงยั่วยุต่อ  Multitap ไม่ได้ ซื้อมาก็ใช่ว่าจะจบ ต้องซื้อจอยเพิ่มอีก แล้วสมัยเด็กๆ  จะไร้คำว่าอเมริกันแชร์โดยสิ้นเชิง  เหมือนกับกฏอะไรอย่างหนึ่งว่าเจ้าของเครื่องต้องเป็นคนซื้อ  อูยยย  อ่วมกันเลยทีเดียว   2. Memory Unit
เกมส์
      นับว่าเป็นนวัตกรรมแสนชอกชํ้าของ Sega เวลามีเกมบางเกมออกมา  ผู้เล่นต้องทำการซื้อ Memory Unit มาเสียบที่ตัวเครื่องเพิ่ม  ไม่เช่นนั้นจะไม่สามารถเล่นได้  ซึ่งบางเกมก็น่าเล่นเสียเหลือกันจนอดใจไม่ไหว  งานนี้ก็ต้องอดข้าวเคล้านํ้าตากันอีกครั้ง   3. Nintendo Super Scope
เกมส์
สมัย เด็กๆ เวลาที่ได้ดูหนังแอคชั่นยิงล้างผลาญ  มักจะรู้สึกว่าใครที่ถือบาซูก้ายิงนี่มันโคตรเท่เลย  อาจจะเป็นเพราะเหตุนี้เลยทำให้ Nintendo ทำการวางจำหน่าย Super Scope  จอยปืนสุดลํ้าที่ใช้ระบบ Infarade  ไม่มีสายมากวนใจแถมยั่วนํ้าลายผู้เล่นอีกต่างหาก  ชูโรงเกมที่ซัพพอร์ตด้วยเกม Yoshi's Safari น่าเล่นเป็นที่สุด  แต่พอซื้อมาแล้วดันมีเกมรองรับแสนจะน้อย  แถมบอบบางทำตกทีเดียวก็เสด็จขึ้นสวรรคตซะแล้ว หยอดประปุกมาตั้งนาน  พังในพริบตาเดียว เฮ้อ   4. Sony PocketStation และ Dreamcast Memory Unit
เกมส์
      หลังจากที่กระแส Tamagochi โด่งดังเป็นพลุแตกทั่วโลก  ก็ได้เกิดนวัตกรรมบนเครื่องคอนโซลต่อมาก็คือ PocketStation  ที่สามารถดึงข้อมูลในเกม มาเล่นเป็นเกมบนเครื่อง PocketStation แถมเป็น  Memory Card ได้ด้วย ทั้งๆ ที่เกมก็เป็นมินิเกมขำๆ ไม่ได้มีอะไรมากมาย  แต่ก็อยากได้กันอยู่ดี  (แต่บางเกมถ้าดึงข้อมูลกลับก็จะปลดล็อคอะไรบางอย่างในเกม) จากนั้น  Dreamcast ก็มีการผลิต Memory Unit  ที่มีลักษณะคล้ายๆ PocketStation ออกมา  แต่ใส่ลูกเล่นไว้แพรวพราว และราคาก็แพงตาม   5. Light Boy
เกมส์
      เป็นความอนาถจิตของจอภาพ Dot Matrix  ในเกมบอยที่ไม่สามารถเล่นในเวลากลางคืนได้เพราะว่าไม่มีไฟ จึงได้ก่อนเกิด  Light Boy ขึ้นมา เพียงแค่ใส่ถ่าน เอามาครอบหัว  ก็มีไฟส่องสว่างให้เล่นในเวลากลางคืนแล้ว  แถมมากับแว่นขยายก๊อกแก๊กที่ทำให้จอภาพใหญ่กว่าเดิม  ในสมัยก่อนนั้นเป็นสิ่งที่ผู้เล่นเกมบอยแทบทุกคนต้องอยากได้กันสุดๆ   6.หัวโปร
เกมส์
      หัวโปรเป็นอุปกรณ์เสริมที่ไม่ใช่เป็นของ Official จากโรงงานผู้ผลิต  แต่ดันเป็นสิ่งที่ขาดไม่ได้เลยทีเดียว เพราะว่ามันทำให้เครื่อง Super  Famicom สามารถเล่นแผ่นผีได้นั่นเองโดยการ Rip เกมจากตลับมาปั๊มใส่แผ่น  Floppy 3.5  ดังนั้นใครที่บ้านไม่ได้มีอันจะกินก็ต้องเก็บตังเพื่อหัวโปรอันเป็นที่รัก ตัวนี้ ใครซื้อรุ่นแรกๆ จะซวยหน่อยเพราะไม่สามารถเล่นเกมที่มีความจุสูงได้  แถมไม่มี DSP ทำให้เล่น Mario Kart ไม่ได้อีก   7. Arcade Stick
เกมส์
      ในสมัยก่อนที่วงการ Arcade Game บ้านเรารุ่งเรือง  เหล่าผู้คนที่เป็นสาวกก็จะรู้สึกชื่นชอบ Arcade Stick กันมาก  ปล่อยฮาโดเคนได้สบายโดยนิ้วไม่พอง ทางผู้ผลิตจึงได้ทำ Arcade Stick  ให้ผู้เล่นได้รับความรู้สึกเหมือนกับเล่นเกม Arcade กันอย่างเต็มที่  ใครอยากได้ต้องใจแข็งกันซักนิด เพราะว่าราคามันโหดเหลือเกิน  และในปัจจุบันก็ยังเป็นสิ่งที่ขายดิบขายดีกันอยู่   8. สายลิ้งเกมบอย
เกมส์
      ในยุคโปเกมอนรุ่งเรืองที่ทำเอาเกมบอยขายดีเป็นเทนํ้าเทท่า  ใครที่ไม่มีสายลิ้งนี่อย่าหวังว่าจะเก็บโปเกมอนได้ครบถ้าไม่แน่จริง  ทุกคนจึงต้องมีสาย data link  ติดตัวไว้ตลอดเวลาเพื่อทำการเล่นเกมด้วยกันหรือแชร์ไอเทมให้กันและกัน   9. Power Glove
เกมส์
      ในยุคสมัย Famicom ครองเมือง ไอ้เจ้านี่ถือว่าเป็นอุปกรณ์โคตรลํ้าสุดๆ  เลยทีเดียว สำหรับ Power Glove  ที่สามารถจับความเคลื่อนไหวของผู้ใช้เพื่อใช้บังคับเกม ผ่านเซ็นเซอร์ทั้ง 3  มุม แถมยังมีปุ่มยุ่บยับเต็มไปหมด ถึงแม้ว่าในยุคสมัยหลัง Power Glove  จะโดนหาว่าเป็นอุปกรณ์ไร้สาระ แต่ในสมัยก่อนสามารถขายได้ถึง 100,000  ชุดเลยทีเดียว
10. หม้อแปลง Game Gear 
เกมส์
      ถึง แม้ว่าจะเป็นของที่อยากได้มาก แต่ก็เป็นของที่ต้องซื้อโดยไม่เต็มใจที่สุด  สำหรับ หม้อแปลงของ Game Gear เชื่อว่าแฟนๆ เกมเกียร์ในสมัยก่อน  ต้องส่งมันเสด็จขึ้นสวรรคตด้วยนํ้ามือตัวเองไม่ตํ่ากว่า 2  ตัวทำให้ต้องซื้อใหม่ตลอดเวลา ด้วยความที่มันสูบถ่านอย่างกับสูบนํ้า  จะให้ซื้อถ่านเล่นตลอดก็คงจะไม่ไหว
