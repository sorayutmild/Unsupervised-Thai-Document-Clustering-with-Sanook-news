           หลังจากมีภาพ Official ของ 2018 Honda Civic Type R Prototype ใหม่ เผยแพร่มาก่อนหน้านี้ ล่าสุดเรามีภาพจากงานเปิดตัวที่ Paris Motor Show 2016 มาฝากกันครับ
     Honda Civic Type R ถูกพัฒนาต่อมาจาก Civic Hatchback ที่เพิ่งวางจำหน่ายในอเมริกาเหนือไปก่อนหน้านี้ โดยวางจำหน่ายเป็นเวอร์ชั่นสมรรถนะสูงสำหรับแฟนฮอนด้าที่ชื่นชอบความแรงโดยเฉพาะ
2017 - 2018 Honda Civic Type R Prototype
     Civic Type R Prototype มาพร้อมตัวถังสีเทาด้านเคลือบเงาพิเศษ ติดตั้งชุดแต่งเสริมแอโร่ไดนามิครอบคัน ทั้งลิ้นกันชนหน้า, สปอยเลอร์ด้านข้าง และดิฟฟิวเซอร์ท้าย ติดตั้งไฟหน้าแบบ LED พร้อมกระจังหน้าสีดำเงาแปะโลโก้ 'Type R', ฝากระโปรงหน้ามาพร้อมช่องดักลม
     ด้านท้ายติดตั้งสปอยเลอร์ และปลายท่อไอเสียแบบ 3 ท่อวางกลาง เสริมโป่งล้อทั้ง 4 ข้างให้ยื่นออกมามากกว่าปกติ ติดตั้งตั้งครีบระบายความร้อนบริเวณซุ้มล้อด้านหน้า พร้อมล้ออัลลอยสีดำ-แดงขนาด 20 นิ้ว
2017 - 2018 Honda Civic Type R Prototype
     รายละเอียดขุมพลังของ Civic Type R ใหม่ ยังไม่ถูกเปิดเผยออกมา แต่คาดว่าจะใช้เครื่องยนต์เบนซินเทอร์โบชาร์จความจุ 2.0 ลิตร ที่เค้นกำลังได้สูงสุด 306 แรงม้า (BHP)
     ส่วนการทำตลาดเวอร์ชั่น Type R ในบ้านเราคงมีหวังเพียงริบหรี่ แต่สำหรับ Civic Hatchback ปกติก็อาจไม่แน่เหมือนกัน
