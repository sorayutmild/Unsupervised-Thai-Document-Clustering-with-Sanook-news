             เมอร์เซเดส เปิดตัวรถยนต์รุ่นใหม่  Mercedes Benz A- Class  อย่างเป็นทาง ชงขาย  2   รุ่นราคาเริ่ม  1.89  ล้านบาท เน้นกลุ่มคนรุ่นใหม่ที่ประสบความสำเร็จในชีวิต 
                บริษัท เมอร์เซเดส เบนซ์ (ประเทศไทย) เปิดตัว รถยนต์ รุ่นใหม่  Mercedes Benz A- Class  อย่างเป็นทางการ  ส่งท้ายปีนี้  โดย New! A- Class  ใหม่ นี้เป็นรถยนต์ คอมแพ็คคาร์ ที่จะมีให้เลือก  2  รุ่น คือ  Mercedes Benz A- Class   A180 และ  Mercedes Benz A - Class A250  ที่สอดคล้องกับไลฟสไตล์ของชีวิตเจนเนอร์ชั่นใหม่ ที่มีสไตล์เป็นของตัวเองในความสปอร์ต ปราดเปรียว และคล่องแคล่ว  รวมถึงยังตอบสนองในความเพลิดเพลินในการขับขี่ด้วย
 Mercedes BenZ A- Class
                ดร. อเล็กซานเดอร์ เพาฟเลอร์ ประธานบริหาร  เมอร์เซเดส-เบนซ์  (ประเทศไทย) จำกัด เปิดเผยว่า  the New A- Class  นับเป็นการสร้างปรากฏการณ์ สำคัญในด้านกลยุทธ์การออกแบบของ  Mercedes  Benz  ต่อไปในอนาคต  รวมถึงยังเป็นมาตรฐานใหม่ ของการออกแบบในอนาคตด้วย
                ทั้งนี้ Mercedes Benz A- Class  ใหม่ มาพร้อมการออแบบ ที่สะท้อนในความเป็นตามปรัชญา  Two-Box  ที่สื่อถึงความคล่องแคล่ว และปราดเปรียวในการออกแบบ ถยนต์ และ โดดเด่น ด้วยเส้นสายที่สะดุดตา พริ้วไหว  โดยมาพร้อมเอกลักษณ์ใหม่ล่าสุด  Diamond Grill พร้อมโคมไฟหน้าที่โฉบเฉี่ยว
                ด้านในห้องโดยสาร Mercedes Benz A- Class  ใหม่ปลุกอารมณ์ ที่ให้ความสปอร์ต ของตัวรถเข้ากับความหรูหรามากยิ่งขึ้น ตอบสนองต่อผู้ใช้ด้วยฟังชั่นความสามารถในการเข้าถึงสังคมออนไลน์ แต่ไม่ทิ้งความหรูหราที่ดูมีเสน่ห์ในการตอบสนองมากยิ่งขึ้น เช่นเดียวกับ การเลือกสรรวัสดุที่มีคุณภาพและลงตัวในการใช้งาน   เริ่มจาก ช่องปรับอากาศ  5  ช่อง  พวงมาลุยหุ้มหนัง เป็นต้น
 Mercedes BenZ A- Class
                สำหรับใน Mercedes Benz A- Class  ใหม่ มาพร้อม  2  รุ่น ให้เลือก คือ  A180  ที่มาพร้อมกับเครื่องยนต์   1.6 ลิตรเทอร์โบ ให้กำลังสูงสุด 122 แรงม้า ที่  5000  รอบต่อนาที และ รุ่น A250  AMG Sport  ที่มาพร้อมกับ ขุมพลังขนาด 2.0 ลิตร เทอร์โบชาร์จ  ให้กำลังสูงสุด 211 แรงม้า ที่ 5,500  รอบต่อนาที และสามารถเร่ง  0-100  ก.ม./ช.ม. ได้ใน   6.6 วินาที เท่านั้น
                ด้านราคาจำหน่าย  Mercedes Benz A- class  ใหม่ เปิดราคาจำหน่าย ในรุ่น เริ่มต้น Mercedes Benz A- class A180  ที่ 1.89 ล้านบาท ส่วนในรุ่น  Mercedes Benz A- Class A250 AMG sport  จะวางจำหน่ายในราคา  2.49 ล้านบาท  โดยยังมาพร้อมการรับประกันยาวนานถึง  3  ปี และ  Star Assist  โปรแกรม ดูแลรักษาเป็นเพื่อทุกเส้นทางตลอด 24 ชั่วโมง
