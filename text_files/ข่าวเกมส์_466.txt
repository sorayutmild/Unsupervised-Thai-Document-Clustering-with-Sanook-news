จะว่าไปแล้ว ในวงการไหนๆ ก็ตาม มักจะมีศัพท์เฉพาะเป็นของตัวเองทั้งนั้น  ไม่ว่าทางการแพทย์, การช่าง, อุตสาหกรรม หรือเศรษฐกิจฯลฯ  ที่เวลาพวกเขาพูดกันเราอาจไม่เข้าใจ และแน่นอน  วงการเกมเองก็มีศัพท์เฉพาะอยู่มากมายเช่นกันที่อาจทำให้คนทั่วไปงงได้  ลองมาดูกันดีกว่าครับว่ามีอะไรบ้าง
ศัพท์เฉพาะเกมส์
Hit Point (HP) - เกมเมอร์ - พลังชีวิตของตัวละครในเกม - คนทั่วไป - จุดต่างๆ ในร่างกายของเหยื่อถูกคนร้ายโจมตี ในคดีฆาตกรรม 
ศัพท์เฉพาะเกมส์
Energy Bar (EN) - เกมเมอร์ - แถบพลังงาน สำหรับใช้สกิลในเกมต่างๆ บางเกมจะมาในรูปแบบของมานา - คนทั่วไป - ขนมที่ช่วยให้พลังงาน
ศัพท์เฉพาะเกมส์
XP System (EXP) - เกมเมอร์ - ระบบเก็บค่าประสบการณ์ในเกม RPG เพื่อพัฒนาตัวละคร เมื่อถึงระดับที่กำหนด - คนทั่วไป - ระบบปฏิบัติการณ์วินโดนว์รุ่นเมื่อเกือบๆ 10 ปีที่แล้ว
ศัพท์เฉพาะเกมส์
Party - เกมเมอร์ - กลุ่มคนที่รวมตัวกันในเกมออนไลน์ เพื่อจุดประสงค์อย่างใดอย่างนึง เช่นล่าบอส ลุยดันเจี้ยน - คนทั่วไป - งานสังสรรค์, งานรื่นเริง
ศัพท์เฉพาะเกมส์
Level Grinding - เกมเมอร์ - การปั๊มเลเวลหรือวิ่งเควสต์เดิมๆ ในเกม RPG เพื่อให้ตัวละครเก่งขึ้น - คนทั่วไป - งานเจียรหิน หรืออิฐ ที่ออกมาเรียบเนียนสุดๆ
ศัพท์เฉพาะเกมส์
Re-Spec - เกมเมอร์ - ระบบการรีเซ็ตการอัพสกิลใหม่ หรือที่คนไทยเรียกว่า "รีสกิล" - คนทั่วไป - การวัดสายตา
ศัพท์เฉพาะเกมส์
Death Match - เกมเมอร์ - การต่อสู้ตัดสินกันแบบตายไปข้างหนึ่งของเกมที่มีการแบ่งฝ่ายสู้กัน - คนทั่วไป - จุดต้นเพลิงเวลาเกิดเหตุเพลิงไหม้
ศัพท์เฉพาะเกมส์
Head Shot - เกมเมอร์ - มักใช้กันในเกมแนว First / Third Person Shooting ที่หมายถึงการยิงปืนเข้าหัวศัตรูถึงแก่ความตายในทันที - คนทั่วไป - รูปถ่ายหน้าตรงที่มักใช้ในการสมัครงาน
ศัพท์เฉพาะเกมส์
Cover System - เกมเมอร์ - ระบบที่สามารถทำให้ผู้เล่นหลบหลังที่กำบังได้ - คนทั่วไป - ผ้าใบกันน้ำ...
ศัพท์เฉพาะเกมส์
Game Mechanic - เกมเมอร์ - ระบบเด่นหรือรูปแบบการบังคับเกมนั้นๆ - คนทั่วไป - ช่างซ่อมเครื่องเกม...
ศัพท์เฉพาะเกมส์
God Mode - เกมเมอร์ - พวกที่โกงเกม ใส่สูตรอมตะ อาวุธเต็ม ต่อยทีเดียวตาย บลาๆๆ ฯลฯ - คนทั่วไป - คนที่เพิ่งจะเริ่มนับถือศาสนา หรือค้นพบตัวเองว่านับถือศาสนาอะไร
ศัพท์เฉพาะเกมส์
Clipping Error - เกมเมอร์ - Bug ของเกมหรือ Glitch ที่ทำให้ร่างกายตัวละครในเกมบิดเบี้ยว - คนทั่วไป - ทรงผมพิลึกๆ
ศัพท์เฉพาะเกมส์
Co-Op Campaign - เกมเมอร์ - โหมดหนึ่งของเกมที่สามารถทำให้ผู้เล่นหลายคนสามารถร่วมลุยผ่านด่านในเกมไปด้วยกันได้ - คนทั่วไป - โปรโมฃั่นของร้านค้า  ในรูปแบบการซื้อจากร้านหนึ่ง  เพื่อได้สิทธิ์ลดราคาเวลาซื้อของจากอีกร้านหนึ่ง  หรือได้สิทธิ์ลดราคาในการซื้อสินค้าร้านเดิมครั้งต่อไป
ศัพท์เฉพาะเกมส์
Multiplayer - เกมเมอร์ - โหมดของเกมที่จะทำให้ผู้เล่นสามารถเล่นเกมร่วมกับผู้อื่นได้มากกว่า 2 คนขึ้นไป - คนทั่วไป - เครื่องเล่นแผ่น CD หรือ DVD ที่สามารถใส่ได้ทีละหลายๆ แผ่น แล้วเลือกสลับได้โดยไม่ต้องเปลี่ยนแผ่น
ศัพท์เฉพาะเกมส์
Ghost Data - เกมเมอร์ - ข้อมูลสถิติของผู้เล่นเกม หรือรูปแบบการเล่น เพื่อใช้ในการฝึกผู้เล่นใหม่ - คนทั่วไป - คล้ายๆ กับช่วง "ล่าท้าผี" ในรายการคนอวดผีแหละครับ
ศัพท์เฉพาะเกมส์
Shoulder Button - เกมเมอร์ - ปุ่มบนคอนโทรลเลอร์จำพวก L2-R2 ใน Playstation หรือ LT-RT ใน XBOX360 - คนทั่วไป - กระดุมบนปกเสื้อ
ศัพท์เฉพาะเกมส์
Cloud Storage - เกมเมอร์ - ระบบของการสำรองข้อมูลของผู้เล่นเกม ที่เก็บไว้ยังเซิร์ฟเวอร์กลางเพื่อป้องกันความเสียหายจากเครื่องเกมของผู้เล่นเอง - คนทั่วไป - ถังเก็บกักน้ำฝน
ศัพท์เฉพาะเกมส์
Battery Backup - เกมเมอร์ - ระบบการสร้างเกมตลับบางเกม ที่สามารถทำให้เซฟเกมได้ - คนทั่วไป - ถ่าน / แบตเตอร์รี่สำรอง
ศัพท์เฉพาะเกมส์
WoW - เกมเมอร์ - ชื่อเรียกย่อๆ ของเกม World of WarCraft - คนทั่วไป - เสียงอุทานเวลาตื่นเต้นดีใจ, ร้านฟิตเนสเจ้าหนึ่ง
ศัพท์เฉพาะเกมส์
CoD - เกมเมอร์ - ชื่อเรียกย่อๆ ของเกม Call of Duty - คนทั่วไป - ปลาชนิดหนึ่ง...
ศัพท์เฉพาะเกมส์
Character Model - เกมเมอร์ - รูปแบบ ความละเอียด ความเรียบของการสร้างตัวละครในเกม - คนทั่วไป - นางแบบ / นายแบบ ตามนิตยสาร
ศัพท์เฉพาะเกมส์
Splash Damage - เกมเมอร์ - การโจมตีที่สร้างความเสียหายเป็นวงกว้าง  - คนทั่วไป - จะเข้าใจว่าน้ำหรือกาแฟ หรือของเหลวหกใส่งาน หรือไม่ก็ตั้งแก้วน้ำเย็นไว้พอยกออกแล้วจะเป็นคราบน้ำตามรอยของก้นแก้ว
ศัพท์เฉพาะเกมส์
Super Combo - เกมเมอร์ - การจู่โจมอย่างต่อเนื่องจนศัตรูตายโดยที่ไม่มีโอกาสโต้ตอบได้ - คนทั่วไป - วงดนตรีวงหนึ่งเมื่อนานมาแล้ว
ศัพท์เฉพาะเกมส์
Spam - เกมเมอร์ - การทำอะไรก็ตาม ในรูปแบบเดิมๆ ซ้ำๆ เช่นจู่โจมในรูปแบบเดิมๆ (อารมณ์บลังก้านั่งช็อตไฟฟ้ารัวๆ) - คนทั่วไป - เนื้อบดอัดกระป๋อง..
ศัพท์เฉพาะเกมส์
ตบบอส - เกมเมอร์ - ต่อสู้กับมอนสเตอร์ หรือศัตรูที่เป็นหัวหน้าในฉากนั้นๆ - คนทั่วไป - ตบหัวหน้าตัวเอง ?
ศัพท์เฉพาะเกมส์
บวก / เข้า - เกมเมอร์ - โจมตี, ต่อสู้, ตะลุมบอน - คนทั่วไป - การเพิ่มจำนวน, ดีขึ้น / เดินไปในสถานที่หนึ่ง
ศัพท์เฉพาะเกมส์
เกมเถื่อน - เกมเมอร์ - เกมผิดกฏหมาย (โหลดฟรีไม่เสียเงินซื้อ) - คนทั่วไป - เกมผิดกฏหมาย (เกมใช้ความรุนแรง เกมที่เด็กไม่ควรเล่น)
ศัพท์เฉพาะเกมส์
เกรียน - เกมเมอร์ - คำด่าคนที่ชอบแกล้งหรือก่อกวนสร้างความรำคาญให้คนอื่นในเกม (ไม่จำเป็นว่าจะเด็กหรือผู้ใหญ่) - คนทั่วไป - คำด่าชนิดหนึ่งสำหรับคนที่นิสัยกวนๆ หรือด่าเฮฮากันในกลุ่มเพื่อนฝูง - คนทั่วไป ของทั่วไป - คำด่าชนิดหนึ่งสำหรับเด็กไม่บรรลุนิติภาวะ - คนทั่วไป ของทั่วไป ของทั่วไป - ผมทรงนักเรียน ของเด็กในโรงเรียนรัฐบาล (น่าจะเป็นที่รู้จักกันในวงกว้างแล้วล่ะนะ)
หมดเท่านี้แหละครับ สำหรับศัพท์เฉพาะสไตล์เกมเมอร์  ที่คนไม่เล่นเกมอาจงงกันได้ จริงๆ แล้วศัพท์เฉพาะพวกนี้มีอยู่เยอะมาก  ที่ผมนำมาก็เป็นเพียงแค่ส่วนหนึ่งเท่านั้น หากเพื่อนๆ  มีคำไหนอีกบ้างก็ลองเอามาแนะนำดูได้ครับ
