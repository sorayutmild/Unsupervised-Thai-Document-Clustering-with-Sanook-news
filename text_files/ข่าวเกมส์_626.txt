The Evil Within
The Evil Within เกมแนวผจญภัยสยองขวัญเอาตัวรอด ผลงานใหม่จากผู้สร้าง Resident Evil ตอนนี้ก็ออกมาให้เล่นกันแล้วทั้งของคอนโซลและพีซี สำหรับคนที่ชอบเล่น RE มาก่อนก็สามารถเล่นเกมนี้ได้ เพราะแนวการเล่นไม่ค่อยต่างกันเลย แต่ที่ต่างกันพอสมควรก็คือศัตรู เพราะนอกเหนือจากซอมบี้ศพคืนชีพแล้ว ศัตรูของ The Evil Within จะออกแนวสัตว์ประหลาดโหดๆมากกว่า RE แถมอาวุธก็ค่อนข้างมีจำกัดด้วย อาจทำให้ผู้เล่นมีปัญหาในการต่อสู้พอสมควร
สำหรับชาว PC ที่อยากเล่น The Evil Within แบบสะใจกว่าเดิม ทาง Bethesda Softworks ก็มีสูตรโกงให้ใช้กันอีกเหมือนอย่างเคย (เกมค่ายนี้ PC มีสูตรโกงให้ตลอด) แต่ใครที่ใช้สูตร ทางค่ายจะไม่รับผิดชอบปัญหาเกี่ยกับตัวเกมที่ตามมาในทุกกรณีนะ ถ้าตั้งใจว่าจะโกงแน่ก็ลองมาดูกัน
สูตรโกงนี้จะใช้ได้สำหรับคนที่เล่นเกม The Evil Within ของ PC เท่านั้น โดยการใช้คำสั่ง Debug Console Command เหมือนเกมอื่นๆของ Bethesda Softworks (เช่น Skyrim) แต่ว่าในเกม The Evil Within จะมีวิธีเปิดใช้งานเล็กน้อย คือต้องไปเปิดใน Steam (หากติดตั้งเกมแบบอื่นๆยังไม่มีแจ้งวิธ๊เปิดใช้) คลิกขวาที่ตัวเกมใน Steam แล้วเลือก properties ที่ Tab General คลิกที่ set launch options จากนั้นให้พิมพ์คำว่า +com_allowconsole 1 ลงไปในช่องเพื่อปลดล็อคการใช้งาน แล้วก็เข้าเกมตามปกติ และให้กดปุ่ม Insert เมื่อต้องการใช้งานแถบคำสั่ง Debug Console Command โดยมีสูตรปลดทั้งหมดดังนี้
 
R_swapinterval 
สูตรปลดเฟรมเรตของเกม โดยให้ใส่ตัวเลขตามหลังสูตร เช่น R_swapinterval -2 จะเป็นการตั้งให้มีเฟรมเรต 30 FPS (ค่ามาตรฐาน) หากเป็น -1 คือ 60 FPS และถ้า 0 คือไม่ล็อคค่าเฟรมเรต
 
God
สูตรอมตะ ทำให้ผู้เล่นไม่โดนศัตรูตบตาย และศัตรูจะไม่ใช้ท่าโจมตีแบบตายทันทีด้วย
 
Noclip
เดิมทะลุกำแพงได้ ไม่แนะนำให้ใช้ เพราะอาจเกิดบั๊กง่ายมาก
 
G_infiniteammo
กระสุนไม่มีวันหมด แต่ว่าก่อนใช้งานสูตรนี้ ปืนของผู้เล่นจะต้องมีกระสุนอย่างน้อย 1 นัดในปืน
 
G_stoptime 1
หยุดเวลา ระวังเวลาใช้สูตรนี้จะเกิดบั๊กกับ User Interface หากพิมพ์ G_stoptime 0 จะยกเลิกการหยุดเวลา
 
Toggledebugcamera
ปลดล็อคมุมกล้อง ทำให้ขยับมุมกล้องไปที่อื่นๆได้ นอกเหนือจากที่ตัวเอก
 
R_forceaspectratio 
สูตรปลดตำแหน่งของหน้าจอ ให้ใส่ตัวเลขที่ต้องการ เดิมจะเป็น 2.5
 
Com_showFPS
โชว์ตัวเลขเฟรมเรตในขณะที่เล่นเกม
 
Startslowmotion2
ทำให้การเคลื่อนไหวช้าลง ยกเลิกด้วยคำสั่ง Stopslowmotion
