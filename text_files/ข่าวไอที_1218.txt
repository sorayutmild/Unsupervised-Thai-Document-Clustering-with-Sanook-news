Google ปล่อยอัปเดต Chrome บน Android เป็นเวอร์ชั่น 74 ซึ่งเวอร์ชั่นนี้สามารถเปิดใช้งาน Dark mode ได้แล้ว ดูวิธีารเปิดใช้งานได้ด้านล่างนี้เลยครับ
เปิดแอป Google Chrome ในช่อง URL ให้พิมพ์ว่า chrome://flags เปิดใช้งานในช่อง Android Chrome UI dark mode ปิด Google Chrome แล้วเปิดใหม่
เมื่อทำส่วนนี้เสร็จแล้ว ให้เปิดแอปพลิเคชั่น Chrome ใหม่อีกครั้ง ไปที่ Settings จะมีหัวข้อ Dark mode ปรากฏขึ้นมา ให้เลื่อนเป็น On เพียงเท่านี้หน้าเว็บก็จะกลายเป็นสีดำแล้วครับ อย่างไรก็ตาม Dark mode ในเวอร์ชั่นนี้ยังไม่ใช่เวอร์ชั่นสมบูรณ์ เพราะจะเปลี่ยนเป็นสีดำเฉพาะหน้า UI ของแอปเท่านั้น ส่วนการแสดงผลหน้าเว็บไซต์จะยังเป็นแบบปกติ
