 

ยิ่งใกล้วันวางขายเกมส์เท่าไหร่ Capcom ก็จะโปรโมทเกมนั้นๆถี่ยิบเป็นพิเศษเสมอ ล่าสุดอัพเดตระบบใหม่ของ Resident Evil 6 เพิ่มเติมกันอีกแล้ว หลังจากล่าสุดเพิ่งอัพเดตระบบเว็บ ResidentEvil.net ไปหมาดๆ มาวันนี้เป็นคิวของระบบ Skill เพิ่มความสามารถให้ตัวละคร และโหมด Split Screen Co-Op ให้สามารถแบ่งจอออกครึ่งๆ ช่วยกันเล่นกับเพื่อนๆพร้อมกันได้
 
สำหรับใครที่ได้ทดลองเล่น Demo ของ Resident Evil 6 อาจจะรู้จักระบบ Skill ของภาคนี้กันไปบ้างแล้ว โดยทุกครั้งที่ตัวละครที่คุณเล่น สามารถจัดการศัตรูได้ จะเห็นว่าจะได้คะแนน Skill Point ให้ทุกๆครั้ง ที่ด้านล่างซ้ายของหน้าจอ และเมื่อเล่นจนจบ Chapter ระบบเกมส์จะนำคะแนน Skill Point ที่ทำได้มาเพิ่มกับโบนัสตอนจบฉาก และก็ออกมาเป็นจำนวน Skill Point ทั้งหมดที่เราได้รับ ผู้เล่นสามารถเอาแต้ม Skill Point นี้ไปเรียนรู้ Skill ใหม่ๆมาใช้งานได้ และยังเอาแต้มนี้ไปอัพเกรดสคิลที่เรียนแล้วก็ได้เช่นกัน สคิลที่อัพเกรดแล้วก็จะมีระดับเลเวล LV ที่เพิ่มขึ้น แต่ว่าจะเลือกเรียนหรือเลือกอัพเกรดสคิลไหนก็ต้องคิดไว้ดีๆหน่อย เพราะตัวละครหนึ่งๆ สามารถติดตั้ง Skill ไปใช้งานได้พร้อมกันแค่ 3 Skill เท่านั้น แต่ว่าจะมี Set ให้เลือกติดตั้งได้ทั้งหมด 8 Set สามารถเลือกเปลี่ยน Skill ที่อยากใช้ได้ตลอด ระหว่างที่เล่น โดยการกดเข้าเมนูเกม (สมาร์ทโฟน) แล้วเลือก Set Skill ที่ต้องการ ซึ่งตอนเลือก Skill นั้นเกมไม่หยุดนะ ต้องหาโอกาสเปลี่ยนดีๆ ไม่งั้นอาจโดนซอมบี้ตุ๋ยหลังได้










 
 
ตัวอย่างความสามารถ Skill (เท่าที่เห็นในภาพ)
- Firearms เพิ่มพลังโจมตีของอาวุธปืนทุกชนิด
- Battle Attack เพิ่มพลังโจมตีของท่าโจมตีประชิดทุกชนิด
- Defence ลดความเสียหาย เวลาโดนโจมตี
- Best Partner กดปุ่ม Help ขอความช่วยเหลือแล้วพาร์ทเนอร์จะมาช่วยอย่างรวดเร็ว
- Brave Partner พาร์ทเนอร์จะลุยแหลกมากขึ้น
- Do not help ปิดการช่วยเหลือของพาร์ทเนอร์ (สคิลพาร์ทเนอร์ จะสำหรับเล่นคนเดียวเท่านั้น)
- Recovery ใช้ไอเทมฟื้นพลังแล้วได้ผลมากขึ้น
- Ammo Up เพิ่มจำนวนกระสุน (สคิลประเภทนี้จะแยกชนิดปืน)
- Sturdy Aim ทำให้ไม่โดนขัดจังหวะ หากโดนโจมตีขณะเล็งปืน
- Concentrate เพิ่มความแม่นยำของปืน (ลดอาการปืนส่าย หรือมือสั่น)
- Critical Rate เพิ่มโอกาสโจมตีติด คริติคอล
- Penetration ทำให้ยิงปืนแล้วกระสุนเจาะทะลุไปโดนศัตรูด้านหลัง
- Hand Gun Master เพิ่มความสามารถเมื่อใช้ปืนพก (สคิลแยกชนิดปืน)
- Magnum Master เพิ่มความสามารถเมื่อใช้ปืนแมกนั่ม (สคิลแยกชนิดปืน)
- Item Drop Up เพิ่มโอกาสตกของไอเทม จากศัตรู
- Quick Reload บรรจุกระสุนเร็วขึ้น
- J'avo Killer เพิ่มพลังโจมตี เมื่อสู้กับ J'avo
- Zombies Killer เพิ่มพลังโจมตี เมื่อสู้กับ Zombies
 
ระบบอีกอย่างที่อัพเดตวันนี้ก็คือระบบ Split Screen Co-Op ไม่ใช่ระบบใหม่อะไร ก็เป็นระบบเดียวกับภาค Resident Evil 5 นั่นเอง ผู้เล่นสามารถใช้เครื่องเกมและทีวีเครื่องเดียว แบ่งจอเล่นพร้อมกันกับเพื่อนสองคนได้ เพิ่มความสนุกมากขึ้น ดีกว่านั่งเล่นคนเดียวกับ AI งี่เง่าๆอีกตัว
