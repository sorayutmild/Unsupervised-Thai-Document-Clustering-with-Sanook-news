พาไปชมคลิปเด็ดของ "นุศรา ต้อมคำ" มือเซตสุดสวยทีมชาติไทย ที่ต้องบอกว่า "สุดยอด" ไม่น้อยหน้าใครในวงการวอลเล่ย์บอลโลกและนี่คือลูกเก่งของ "ซาร่า" นั่นคือการเซ็ตบอลเร็วให้เพื่อนขึ้นตบทำคะแนน ที่แต่ละช็อตในคลิป ต้องเรียกว่าเป๊ะทั้งจังหวะ-ความเร็ว แสดงให้เห็นว่านอกจากต้องซ้อมมาอย่างหนักแล้ว ประสบการณ์ก็เป็นสิ่งสำคัญสำหรับการเล่นลูกลักษณะนี้ให้สมบูรณ์แบบได้ย้ำกันอีกทีครับ ทีมวอลเลย์บอลหญิงไทย มีโปรแกรมใหญ่ประจำปีรออยู่ นั่นคือศึก "เวิลด์ กรังด์ปรีซ์ 2016" ที่สนามแรกจะจัดขึ้นที่ประเทศจีน และรอบสุดท้ายไทยเราได้เป็นเจ้าภาพด้วย
 
โปรแกรมการแข่งขันเฉพาะทีมชาติไทย ในศึกวอลเลย์บอลหญิง "เวิลด์กรังด์ปรีซ์ 2016" รอบแรก  
โดยรายการนี้ ช่อง 7 สี และ เว็บไซต์ Bugaboo.tv ถ่ายทอดสดทุกนัด ที่ทีมชาติไทยลงสนาม สนามแรก ณ ประเทศจีน  ระหว่างวันที่ 10-12 มิถุนายน 2559 วันศุกร์ที่ 10 มิถุนายน 2559 // เวลา 18.30 น.- ไทย พบ จีน  วันเสาร์ที่ 11 มิถุนายน 2559 // เวลา 14.00 น.  - ไทย พบ สหรัฐอเมริกาวันอาทิตย์ที่ 12 มิถุนายน 2559 // เวลา 14.00 น.  - ไทย พบ เยอรมนี------------------------------------------------------------------------สนามที่ 2 ณ ประเทศอิตาลี ระหว่าง 17-19 มิถุนายน 2559  คืนวันศุกร์ที่ 17 มิถุนายน 2559 // เวลา 01.30 น.- ไทย พบ อิตาลีวันเสาร์ที่ 18 มิถุนายน 2559 // เวลา 22.30 น.- ไทย พบ รัสเซียวันอาทิตย์ที่ 19 มิถุนายน 2559 // เวลา 22.30 น.- ไทย พบ เนเธอร์แลนด์------------------------------------------------------------------------
สนามที่ 3 ณ ประเทศญี่ปุ่น  ระหว่าง 24-26 มิถุนายน 2559 วันศุกร์ที่ 24 มิถุนายน 2559 // เวลา 17.10 น.- ไทย พบ ญี่ปุ่นวันเสาร์ที่ 25 มิถุนายน 2559 //เวลา 14.10 น.- ไทย พบ รัสเซียวันอาทิตย์ที่ 26 มิถุนายน 2559 // เวลา 14.10 น.- ไทย พบ เซอร์เบีย ส่วนการแข่งขันรอบสุดท้ายที่ประเทศไทยได้รับสิทธิ์เป็นเจ้าภาพ จะแข่งขันใน ระหว่างวันที่ 6-10 กรกฎาคม 2559 ที่ สนามอินดอร์ สเตเดี้ยม หัวหมาก กรุงเทพมหานคร โดยทีมชาติไทย จะพบกับทีมที่มีคะแนนดีที่สุด 5 ทีมจากการแข่งขันใน 3 สนาม โดย ช่อง 7 สี และ เว็บไซต์ Bugaboo.tv  ถ่ายทอดสดทุกนัด ที่ทีมชาติไทยลงสนามเช่นเดิม
