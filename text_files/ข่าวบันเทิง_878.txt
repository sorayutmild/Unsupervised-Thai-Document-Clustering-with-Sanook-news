ดุเด็ดเผ็ดมันส์ไม่น้อยเมื่อขวานผ่าซากอย่าง "เอ็กซ์ จักรกฤษณ์ พณิชย์ผาติกรรม" นักกีฬายิงปืนทีมชาติไทยออกมาแฉแหลกนักแสดงสาวนางหนึ่งในวงการบันเทิงสูบเงินหลายล้านบาทจากลุง วัย 65 ปี ซึ่งตนนับถือเป็นพ่อบุญธรรม โดยก่อนหน้านี้นักแสดงสาวคนดังกล่าวเคยมาออกรายการล้วงลับอัพโหลดแล้ว แต่ได้พูดบิดเบือนความจริง ซึ่งทำให้หนุ่มเอ็กซ์ทนไม่ไหวถึงกับโวยวายในรายการแฉแต่เช้า โดยมี มดดำ คชาภาและดีเจ.กฤษณ์ ศรีภูมิเศรษฐ์ ทำหน้าที่จัดรายการ
โดยหนุ่มเอ็กซ์เผยกลางรายการว่า ทำไมเอาคนมาออกรายการแล้วให้โกหกประชาชน และเผลอหลุดว่าเป็นดาราหญิงช่อง 3 เคยเล่นหนัง น้ำตาลแดง ดารามีหลายคนไม่ได้เจาะจงว่าเป็นใคร แต่วันนี้อยากพูดถึงด้วยความคับแค้นใจเกี่ยวกับเรื่องทรัพย์สมบัติที่ผู้หญิงคนนั้นพูดในรายการว่าหามาด้วยตนเอง ที่แท้เป็นของลุงตนเองต่างหาก ซึ่งลุงของตนเลี้ยงดูสาวนางนี้ตั้งแต่ยังไม่เข้าวงการ กระทั่งถึงตอนนี้ ลุงอายุ 65 ปีแล้วหวังจะเอาเงินก้อนสุดท้ายที่มีอยู่ซื้อบ้านด้วยกันเพื่อเป็นรังรักในบั้นปลายชีวิต มูลค่ากว่า 5 ล้านบาทและตกแต่งอีกกว่า 2 ล้านบาทพอโอนกรรมสิทธิ์เสร็จปุ๊บ ลุงตนเองก็ถูกทอดทิ้งทันที ซึ่งไม่มีความเป็นธรรม
ถึงตรงนี้เองที่ มดดำ แย้งว่าจะฟ้องได้ยังไงในเมื่อซื้อและยกให้ด้วยความเสน่หาไปแล้ว หนุ่มจักรกฤษณ์ กล่าวต่อว่า ตนเองก็ไม่ได้คิดจะฟ้องร้องเป็นเรื่องคดีความ เพราะตอนที่ซื้อบ้านนั้นได้มีการสัญญาเบื้องหลังว่าจะครองคู่กัน แค่อยากให้ดาราสาวนางนั้นโอนสมบัติคืนเจ้าของเท่านั้นเอง พร้อมมีหลักฐานยืนยันการโอนซื้อบ้านย่านพหลโยธิน ซอย 32
เขาบอกว่าเขาทำมาหากินด้วยตัวเอง มันไม่ใช่ เขาได้มาโดยไม่บริสุทธิ์ใจ แล้วคุณพูดออกมา ภาพคุณดีอยู่แล้ว คุณกล้ามาถกไหม ตุ้มหู แหวนนาฬิกา ที่ได้ยินมาแบมือขอเงินตลอดเวลา แค่เงินจัดงานวันเกิดยังต้องตามไปจ่ายตังค์ให้ตั้ง 7-8 หมื่นบาท มีแมสเสจทุกอย่างในโทรศัพท์ ผมยืนยันได้แล้วบ้านหลังที่เป็นเรือนตายของลุงผม แต่พอโอนถ่ายกรรมสิทธิ์แล้วเหมือนทรยศ เหมือนพ่อแม่ให้มรดกแล้วไล่พ่อแม่ออกจากบ้าน เขาเรียกว่าเนรคุณ คนอย่างนี้มีหน้ามายืนในสังคมได้ไงนักยิงปืนเล่าอย่างดุเดือด จากนั้นยังกล่าวต่ออีกว่าถ้าอีกฝ่ายฟ้องร้องตนก็พร้อมเพราะว่าอยากให้เรื่องมันแดงอยู่แล้ว
 ผมมีรูปที่ใครจะเชื่อ เอาว่าสนั่นกว่าแอนนาแล้วกันคนทั้งประเทศต้องขอดู บ้านผมบันไดไม่แห้งก็แล้วกัน เออชัดเจน
หนุ่มเอ็กซ์กล่าวต่อว่าสมบัติที่คุณลุงเลี้ยงดูปูเสื่อมา ทั้งนาฬิกาโรเล็กซ์ รถแอคคอร์สีขาว ที่อยากให้ไปถามว่าใครซื้อให้ ที่ตนเองออกมาเพราะสงสารคุณลุงที่มากอดตนร้องไห้ ตอนนี้ตนจึงสั่งให้ช่างรับเหมาล็อกประตูบ้านแล้ว หลังมีหนังสือแจ้งค่าตกแต่งมาเพิ่มอีก 5 แสนบาท ขอให้จ่ายค่าตกแต่งก่อนค่อยมาว่ากัน
โอนบ้านคืนก่อน อย่าให้ผมแฉ ถ้าผมแฉ ผมบอกเลยว่าผมเหนื่อกว่าแอนนาอีกขั้วโลกหนึ่ง ถ้าคุณฟังอยู่ขณะนี้อยากจะให้คุณเอาอะไรที่ไม่ใช่ของคุณกรุณาโอนคืนด้วย ถ้าคุณจะรักจะชอบพอกับคนอื่น เชิญตามสบาย ณ เวลานี้ทุกคนพร้อมใจครับ และเข้าใจคุณ เพราะคุณมีความต้องการสูง
 ล่าสุดในรายการล้วงลับอัพโหลด เมื่อคืน (15 ก.ย. 2554) ที่ผ่านมา หนุ่มเอ็กซ์ได้มีโอกาสออกมาเปิดใจเกี่ยวกับกรณีดังกล่าวนี้อีกครั้ง โดยมี หนุ่ม กรรชัย และ มดดำ คชาภา ทำหน้าที่คอยซักถาม 
กรรชัย : คุณมีพ่อบุญธรรมคอยดูแลคุณอยู่จริงหรือไม่เอ็กซ์ : "ไม่จริงในเรื่องดูแล แต่เป็นพ่อบุญธรรมที่ผมให้ความรักและเคารพครับ"
กรรชัย : จริงหรือไม่ที่ครั้งนึงพ่อบุญธรรม เคยเล่าให้คุณฟังว่า มีผู้หญิงคนนึงมาเสนอขายตัวให้เอ็กซ์ : "อย่าใช้คำว่าขายตัว ใช้คำว่าซื้อบริการดีกว่า ไม่ได้ระบุนะครับว่าบริการด้านไหน ก็ใช้วิจารณญาณ ตรงนีถือว่าเป็นเรื่องจริงครับ"
กรรชัย : ในราคา 3 หมื่นบาท ต่อ 1 ครั้ง จริงหรือไม่เอ็กซ์ : "เป็นเรื่องจริงครับ"
กรรชัย : จริงหรือไม่ที่พ่อบุญธรรมไม่ซื้อบริการ แต่ขอเลี้ยงดูผู้หญิงคนนั้นเอ็กซ์ : "จริงครับ เพราะมันกลายเป็นความรักที่เกิดขึ้นระหว่างคนต่างวัย ภายหลังเกษียณจากการทำงาน เงินก้อนสุดท้ายเขาก็ได้ไปซื้อบ้าน"
กรรชัย : จริงหรือไม่ที่พ่อบุญธรรมส่งเสียเล่าเรียนจนถึงปริญญาโทเอ็กซ์ : "ตอนนี้ถึงขั้นด็อกเตอร์แล้ว เรียนที่มหาวิทยาลัยกลางกรุง"
กรรชัย : หลังจากนั้นได้ซื้อรถให้ผู้หญิงคนนี้ด้วย จริงหรือไม่เอ็กซ์ : "ไม่ได้ซื้อ แต่ส่งเงินให้ผ่อนทุกเดือน 3 ปีติดกัน แม้กระทั่งบ้าน"
กรรชัย : ภายหลังผู้หญิงคนนี้ได้เข้าวงการบันเทิงเอ็กซ์ : "ถูกต้อง"
กรรชัย : หลังจากนั้นไปขอให้พ่อบุญธรรม โอนบ้านให้ในราคาหลายล้านบาทเอ็กซ์ : "ตรงนี้เป็นความตั้งใจที่เขามีร่วมกันมา จะใช้เป็นเรือนรักเรือนตาย ผมมีหลักฐานเยอะแยะ เอาเงิน 3 ล้านกว่าโอนเข้าโครงการที่ซื้อบ้านเลย และโอนบ้านเป็นชื่อผู้หญิงคนนี้"
กรรชัย : มีหลักฐานมาให้ดูไหมเอ็กซ์ : "มี แต่ให้ดูไม่ได้"
มดดำ : ว่าด้วยตามตัวบทกฎหมาย เขาให้ด้วยความเสน่หา คุณไม่มีสิทธิ์ออกมาขนาดนี้ มันเป็นการทำลายชื่อเสียงดาราไทย คุณรู้สึกอย่างไรกรรชัย : "ผมไม่รู้สึกอะไร ถ้าคนคนนั้นเป็นคนดี ถ้าเกิดคนนั้นมีความสุจริตใจและมีความจริงใจ ก็ไปอยุ่ด้วยกันกับลุงเลย บ้านนี้เปิดประตูตลอด"
กรรชัย : พอพ่อบุญธรรมมอบบ้าน รถ ให้ทุกอย่าง ผู้หญิงคนนี้ก็หนีไปเลยเอ็กซ์ : "สบายไปแล้ว"
กรรชัย : เขาได้ไปเยอะไหมเอ็กซ์ : "ไม่รู้ ไม่สามารถบอกได้"
กรรชัย : บ้านราคาเท่าไหร่เอ็กซ์ : "5 ล้านกว่า รวมตกแต่งภายใน ก็ตก 7-8 ล้าน แต่ตอนนี้ผมกลับจะต้องเป็นคนมานั่งรับผิดชอบค่าตกแต่งภายในอีก 5 แสนกว่า ฉะนั้นก็เลยอยากบอกเขาให้ออกมาแสดงความรับผิดชอบหน่อย จ่ายเงินเขาไปซะ ไม่งั้นก็เอาบ้านคืนลุงไปซะ ของพวกนี้ผมไม่อยากได้หรอก"
กรรชัย : ถ้าผู้หญิงคนนี้เดินออกมาบอกว่าไม่จริง ไม่ได้มีความสัมพันธ์ลึกซึ้งกับพ่อบุญธรรมของคุณ จริงหรือไม่ที่ทางคุณมีหลักฐานเด็ดที่จะยืนยันว่าเขาทั้ง 2 คนมีความสัมพันธ์กันจริงๆเอ็กซ์ : "ผมไม่เคยโกหกนะ ผมมีหลักฐานอยู่จริง ถ้าผมเอาออกมาเปิดเนี่ยถึงตาย ก้าวออกจากประเทศไทยนี้ไปเลย แต่ผมไม่ทำ ประสงค์ของผมไม่ต้องการทำลายชื่อเสียงใคร ต้องการเรียกร้องความยุติธรรมและความถูกต้อง"
มดดำ : จริงหรือไม่ที่หลักฐานนั้นคือคลิปวิดีโอเอ็กซ์ : "ไม่สามารถตอบได้ครับ"
กรรชัย : จริงหรือไม่ที่คุณต้องการให้ผู้หญิงคนนี้เดินมาขอโทษพ่อบุญธรรมของคุณเอ็กซ์ : "ไม่ต้องเดินเข้ามา พวกเราไม่ต้องการคำขอโทษ แต่เราให้อภัยไปแล้ว ให้สังคมเป็นผู้ตัดสินเอานะครับ ให้เขาเดินมาขอโทษกับที่ไหนก็ได้ว่า กราบขอโทษ ตอนนี้หนูยืนได้ด้วยลำแข้งของหนูแล้ว หนูไม่อยากอยู่ลักษณะรูปแบบเดิม เราแยกจากกันด้วยดี แต่สิ่งที่ดูแลมาทั้งหมด หนูกราบขอบพระคุณ อะไรก็ว่าไป"
กรรชัย : คุณต้องการให้ผู้หญิงคนนี้ออกมายอมรับว่าครั้งนี้เคยคบกับพ่อบุญธรรมคุณจริงเอ็กซ์ : "ใช่ ไม่ใช่ว่าเขาไม่มีตัวตน ที่ออกมาโกหกประชาชน ผมไม่ชอบ แต่ก็ขอบคุณที่ครั้งนึงเขาเคยทำให้พ่อบุญธรรมผมมีความสุข ผมดีใจนะในสิ่งที่คุณได้รับไปได้สร้างอนาคต ความมั่งคั่งให้กับตัวคุณเอง แต่อย่าไปหลอกคน"
