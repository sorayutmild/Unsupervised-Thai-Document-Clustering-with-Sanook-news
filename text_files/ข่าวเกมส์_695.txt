FM 2014
แฟนๆทีมฟุตบอลบางทีม คงอาจจะกำลังหงุดหงิดที่ผู้จัดการระดับเทพคนเก่าวางมือใหม่ ผู้จัดการคนใหม่ไม่เทพดังใจ อยากจะลงไปคุมทีมแทนให้มันรู้แล้วรู้รอด ซึ่งจะไปคุมทีมจริงๆแทนก็คงทำไม่ได้ ดังนั้นเรามาคุมทีมในเกมส์กันดีกว่า ซึ่ง FM 2014 หรือ Football Manager 2014 ก็กำลังจะออกมาให้เหล่าผู้จัดการมือเซียนทุกท่านได้คุมทีมกันอีกครั้งแล้ว กำหนดเปิดฤดูกาลใหม่วันที่ 31 ตุลาคม 2013 เรียกว่ามาวันเดียวกับฮัลโลวีนเลย
ในภาคใหม่นี้ ทาง Sports Interactive ผู้พัฒนาเกมส์ได้เพิ่มให้ FM 2014 สามารถเล่นในระบบปฏิบัติการณ์ Linux ได้เป็นภาคแรก เพื่อขยายกลุ่มผู้เล่นเกมส์นี้ไปใน Linux ด้วย นอกจากนี้ก็ยังมีระบบ cross-save ที่ทำให้ผู้เล่นสามารถโอนเซฟไฟล์การเล่นของเครื่อง PC ข้ามไปเล่นในเวอร์ชั่นพกพาสำหรับเครื่อง PS VITA ได้อีก เรียกว่าถ้าไม่ได้อยู่บ้าน ก็โอนเซฟไปเล่นกันต่อใน PS VITA ได้นั่นเอง

สรุปฟีเจอร์ทั้งหมดของ FM 2014

ZiZo หรือ Zoom In / Zoom Out ฟีเจอร์สำหรับคนมีปัญหาทางสายตาหรือเรื่องจอภาพ สามารถปรับขยายหรือย่อหน้าจอ ตัวอักษร และอื่นๆทั้งหมดได้ตามใจ
YouTube highlights package ภาคนี้ปรับปรุงการ Export คลิปการแข่งไปลง Youtube มากขึ้น มีเพิ่มมุมกล้อง และปรับแต่งได้
Retina support รองรับระบบ Retina 
Mac full screen mode support เล่นเต็มจอใน Mac ได้แล้ว
Linux Compatibillity เล่นใน Linux ได้เป็นภาคแรก
Cloud Saving เซฟเกมส์ด้วยระบบ Cloud ได้
Steam Workshop หากเล่นผ่าน Steam สามารถสร้างและแชร์คอนเท้นในเกมส์ได้
Cross-save with PS Vita ถ่ายโอนเซฟเกมลง PS VITA ไปเล่นต่อนอกบ้านได้

Football Manager 2014 ลง PC ทั้ง 3 ระบบคือ Windows, Mac OS และ Linux นอกจากนี้ก็ยังมีเวอร์ชั่น PS VITA ไว้เล่นนอกบ้านด้วย
