ดูกันอีกรอบ คลิปเกมรถแข่งที่สวยที่สุดในโลก
Project CARS
Project CARS เกมรถแข่งออนไลน์ตัวใหม่ สร้างโดยทีมงาน Slightly Mad Studios (ผู้สร้าง Need for Speed Shift) ที่ยังคงสร้างเกมรถแข่งออกมาในแบบคอนเซปต์เดิม คือเน้นความสมจริงในการขับขี่ให้มากที่สุดเท่าที่จะทำให้ พวกเขาจึงตัดสินใจพัฒนา Project CARS นี้เป็นเกมแข่งรถตัวใหม่ในอุดมคติของพวกเขา ด้วยเอนจิ้น Madness ตัวใหม่ล่าสุด ด้วยพลังของมันทำให้ Project CARS เป็นเกมรถแข่งที่มีการใส่ระบบฟิสิกส์เข้าไปโดยละเอียดที่สุดเท่าที่จะสามารถทำได้
นอกจากเรื่องความสมจริงทั้งเรื่องภาพ เสียง และการขับขี่แล้ว Project CARS ก็ยังเน้นเรื่องของรถแข่ง ที่ต้องเป็นรถที่ได้รับลิขสิทธิ์แบบสมจริงที่สุด โดยมีรถแข่งแบบต่างๆให้เลือกมากมาย ไม่ว่าจะเป็นรถสปอร์ท รถแรลลี่  รถสูตรหนึ่ง และรถโบราณต่างๆ รวมแล้วกว่า 60 รุ่น  และมีสนามแข่งจากสถานที่จริงอีกกว่า 35 สนามแข่ง ซึ่งมากกว่า Gran Turismo  ที่มีสนามแข่งเพียง 27 สนามเท่านั้น แถมยังเพิ่มความสมจริง  ด้วยการให้นักแข่งรถชื่อดังจากวงการต่างๆ ทั้งนักแข่ง Top Gear  นักแข่งแรลลี่ และนักแข่งรถสูตรหนึ่ง  มาร่วมงานทดสอบการขับในเกมด้วยว่าสมจริงหรือเปล่า
คุณสามารถชมความสมจริงของ Project CARS ได้จาก Trailer ตัวล่าสุด ซึ่งไม่ได้เป็นภาพ CG แต่เป็นภาพในเกมที่ผ่านการเรนเดอร์มาแล้ว
