Kazuhisa Hashimoto คือผู้ให้กำเนิดสูตรเกมที่ทุก ๆ ต้องเคยได้ยิน ณ ตอนนี้เขาได้เสียชีวิตลงอย่างสงบ จากการประกาศของ Konami
Konami ได้โพสต์ผ่าน Twitter แสดงความเสียใจต่อ Hashimoto ผู้ให้กำเนิดสูตร ขึ้น-ขึ้น-ลง-ลง-ซ้าย-ขวา-ซ้าย-ขาว-B-A-Start หลังจบชีวิตลงด้วยวัย 61 ปี เมื่อวันอังคารที่ผ่านมา

Hashimoto เข้ามาร่วมงานกับ Konami ช่วงปี 80 เขามีผลงานมากมาย ไม่ว่าจะเป็น Contra, Gradius, Castlevania และอีกมามาย โดยสูตรดังกล่าวถูกใช้ครั้งแรกกับ Gradius เครื่อง Famicom ด้วยเหตุผลที่ว่า เกมมีความยากเกินไป หลังจากนั้น Konami Code ก็เป็นสูตรลับของ Konami ที่ถูกใส่เข้าไปหลายเกม และมาโด่งดังกับ Contra ที่จะทำให้เรามี 30 ชีวิต

Konami Code ไม่ได้เป็นเพียงแค่สูตรลับที่อยู่ในเกมเท่านั้น Konami Code ยังเป็นเหมือน Easter egg ที่แทรกอยู่ในหลาย ๆ อย่างด้วยกัน และ Konami Code จะอยู่ตลอดไป
