เกมส์ PES 2014
PES 2014 เปิดตัวอย่างไม่เป็นทางการไปเมื่อช่วงเดือนมีนาคมที่ผ่านมา โดยตอนนั้นทาง Konami จำเป็นต้องปล่อยข่าว PES 2014 ออกมาทั้งๆที่ยังไม่มีภาพและข้อมูลเกมส์ใดๆ เพื่อนำมางัดกระแสกับ FIFA 14 ที่เปิดตัวในช่วงนั้น สิ่งที่ทราบเกี่ยวกับ PES 2014 ในช่วงนั้นก็มีเพียงว่าจะใช้ Fox Engine ตัวใหม่ที่พัฒนาโดยทีม Hideo Kojima ผู้สร้างเกมส์ Metal Gear Solid เท่านั้น ซึ่งการใช้เอนจิ้นใหม่สร้างแบบนี้ จะทำให้ได้เกมส์ PES รูปแบบใหม่ ที่ไม่เหมือนกับภาคที่แล้วมา
แต่มันก็ไม่ง่ายนักในการใช้เอนจิ้นตัวใหม่ เพราะทีมงาน PES ต้องโยนงานเก่าๆทิ้งไป และเริ่มต้นกันใหม่จากศูนย์ แต่พวกเขาก็ยินดีเพื่อที่จะให้ได้ PES ยุคใหม่ ที่ไม่จำเจเหมือนภาคที่สร้างด้วยเอนจิ้นเก่าๆที่ผ่านมา และด้วยพลังของ Fox Engine ทำให้สร้าง PES เป็นเกมส์ฟุตบอลที่สมบูรณ์แบบมากขึ้น ไม่ว่าจะเป็นเรื่องของกราฟฟิก โมเดลรูปร่างและใบหน้านักเตะที่ดูสมจริง บรรยากาศสนามแข่งที่ชวนข่มขวัญคู่ต่อสู้ รวมไปถึงการเคลื่อนไหวของนักเตะที่ทำได้ราวกับเป็นนักเตะในสนามจริงกันเลย ซึ่งฟีเจอร์ใหม่ๆ ที่ทางทีม PES 2014 ได้สร้างขึ้นจาก Fox Engine มีดังนี้

True Ball Tech
เพื่อให้ดูเป็นเกมส์ที่จำลองการแข่งขันฟุตบอลจริงๆ สิ่งแรกที่ PES 2014 เน้นก็คือความสมจริงของลูกฟุตบอล ทุกสิ่งในสนามแข่งจะส่งผลต่อการเคลื่อนที่ของลูกฟุตบอลอย่างสมจริง ไม่ว่าจะแรงปะทะ มุมของแรงปะทะ พื้นสนาม น้ำหนักและส่วนสูงของนักเตะ ทุกอย่างล้วนเอามาคำนวนผลการเคลื่อนที่ของลูกฟุตบอลทั้งหมด รวมไปถึงระบบการส่งลูกที่ให้อิสระในการควบคุมลูกส่งมากขึ้น โดยการใช้อนาล็อคสติ๊ก และยังใช้รูปแบบเดียวกันในการตัดลูกส่งของคู่แข่งด้วย ซึ่งทำได้แบบ 360 องศา
Motion Animation Stability System (M.A.S.S.)
เพิ่มความสมจริงให้กับร่างกายของนักเตะทุกคนด้วยระบบฟิสิกส์ ซึ่งส่งผลให้รูปร่างของนักเตะมีผลต่อการเล่นทุกจังหวะ ไม่ว่าจะเป็นการยิง การส่ง การกระโดดโหม่้ง การเข้าบอล บังบอล และการแทคเกิล(สไลด์) ข้อมูลเหล่านี้จะส่งผลต่อการเล่นทั้งหมด โดยเฉพาะการแทคเกิลนั้นเป็นระบบที่ PES 2014 เน้นค่อนข้างมาก โดยจะเพิ่มรูปแบบการแทคเกิลหลายๆแบบเอาไว้ ซึ่งนักเตะจะใช้การแทคเกิลต่างกันออกไปตามสถานการณ์ และแรงของการแทคเกิลก็มาจากรูปร่างและพลังของนักเตะนั้นๆด้วย แถมผู้ถูกแทคเกิลก็จะมีผลจากความแรงนั้นเช่นกัน ถ้าเข้าโดนบอลก็แล้วไป แต่ถ้าโดนคนไม่โดนบอล ก็จะฟาวด์และมีท่าตอบสนองหลายๆแบบ ผลของระบบ M.A.S.S. ยังมีอีกอย่างคือการกดดันการบุก โดยฝ่ายรับจะใช้ร่างกายเข้าประชิดเพื่อแย่งบอลจากฝ่ายรุก ซึ่งแรงกดดันและแรงปะทะจากฝ่ายรับจะทำให้ฝ่ายรุกทำอะไรไม่สะดวก เมื่อถูกเข้าประชิด จึงต้องหาพื้นที่ลากบอลหนี หรือส่งบอลไปยังที่ว่างเพื่อให้เล่นบอลง่ายขึ้น ทั้งที้ก็ขึ้นอยู่กับความสามารถเฉพาะตัวนักเตะด้วย

Heart
การทำเกมส์ฟุตบอลให้สมจริงนั้นไม่ใช่ง่ายๆ สิ่งหนึ่งที่ PES 2014 เล็งเห็นว่าจะช่วยเพิ่มความสมจริงให้นั้น ไม่ใช่่เรื่องของเทคนิคการเล่น แต่เป็นบรรยากาศ บรรยากาศของสนามแข่ง เสียงเชียร์ของผู้ชม ที่ข่มขวัญทีมเยือนนั้นเอง ที่จะช่วยให้การแข่งดูสมจริงขึ้น ด้วยการนี้ PES 2014 ได้เพิ่มสถานะใหม่เข้าไปในนักเตะทุกคน นั่นคือ mental attributes หรือสภาพจิตใจนักเตะ ซึ่งมันจะส่งผลลดลงเมื่อต้องไปเตะในฐานะทีมเยือน หรือว่าทีมกำลังอยู่ในสถานะที่เป็นรอง กำลังใจนักเตะก็จะลดลง ส่งผลต่อความสามารถในการแข่ง หากนักเตะคนไหนหมดกำลังใจ ก็ต้องให้เพื่อนร่วมทีมเรียกกำลังใจมา หรือไม่ก็เปลี่ยนตัวสำรองที่กำลังใจดีกว่าลงไปแทน
PES ID
เป็นระบบ AI ที่หยิบมาใช้ต่อจาก PES 2013 ที่จะช่วยบันทึกและจดจำสไตล์การเล่นของนักเตะชื่อดังแต่ละคน ว่าชอบเล่นแบบไหน ทำอย่างไรในการแข่งจริง และนำมาเป็นสไตล์การเล่นในเกมส์ให้เหมือนตัวจริงด้วย ซึ่งในภาค PES 2013 มีนักเตะชื่อดัง 50 คน ที่มีระบบนี้ ส่วน PES 2014 ก็จะขยายไปยังนักเตะคนอื่นๆมากขึ้น

Team Play
เพราะฟุตบอลไม่ได้เล่นคนเดียว การวางแผนจึงเป็นสิ่งสำคัญต่อชัยชนะ นอกจากนักเตะที่ผู้เล่นบังคับเองแล้ว ยังสามารถสั่งการนักเตะที่ไม่ได้บังคับได้ด้วย สามารถให้นักเตะที่อยู่ใกล้ๆมาช่วยแย่งบอล ช่วยบังบอล คุมพื้นที่ วิ่งหาที่ว่าง และอื่นๆได้ ทั้งตอนเป้นฝ่ายรุกหรือฝ่ายรับ
The Core
สุดท้ายกับสิ่งที่เน้นมากที่สุดในการใช้เอนจิ้นใหม่ ก็คือเรื่องของกราฟฟิก ด้วยพลังของ Fox Engine ทำให้พวกเขาสามารถสร้างใบหน้า การขยับของใบหน้า รูปร่างและการเคลื่อนไหวนักเตะได้สมจริงกว่าเก่า ราวกับเป็นใบหน้าของนักเตะตัวจริง

นอกจากฟีเจอร์เหล่านี้แล้ว ทาง PES 2014 ยังมีการปรับปรุงการเล่นลูกตั้งเตะ ทั้งฟรีคิกและลูกโทษ เพิ่มลูกล่อในการใช้นักเตะอีกคนเป็นตัวล่อ ในระหว่างที่ยังไม่ได้เตะฟรีคิก ฝ่ายรับก็สั่งการกำแพงได้เช่นกัน ส่วนลูกโทษก็เพิ่มอิสระในการยิง และผู้รักษาประตูก็เคลื่อนที่ได้อิสระขึ้น
ปิดท้ายกันที่ PES 2014 ได้ลิขสิทธิ์ทีมแข่งใน Asia Champions League อย่างเป็นทางการ ซึ่งได้มีการเซ็นสัญญาไปเมื่อช่วงวันที่ 22 เมษายน 2013
PES 2014 กำหนดออกภายในปี 2013 (ประมาณช่วงเปิดฤดูกาล 2013-2014) ลง PlayStation 3, Xbox 360, Windows PC และ PSP (ยังไม่ลง PS4 และ Xbox One)
