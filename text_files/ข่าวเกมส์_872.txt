พิสูจน์กันให้เห็นกับตา ATK หรือ INT อัพสายไหน เก่งไวกว่ากัน!!

ในตอนที่เกม TS Online เปิดใหม่ๆ เพื่อนๆ หลายคนคงทราบดีว่า ผู้เล่นส่วนใหญ่จะอัพสเตตัสเน้นไปทางสาย ATK ซะเป็นส่วนใหญ่ เนื่องจากสามารถเก็บเลเวลได้ง่าย และยังไม่ต้องเปลือง SP อีกด้วย ซึ่งปัจจุบันนี้ ปัญหา SP ไม่พอใช้ ก็ถูกแก้ไขไปด้วยการตั้ง เสนา ผู้เล่นส่วนใหญ่จึงหันมาเล่นสาย INT เนื่องจากค่า INT นั้นยังสามารถทำให้หลบสกิลจับต่างๆ ได้อีกด้วย
วันนี้เราจะมาพิสูจน์กันดูว่า ทั้งสาย ATK และ INT นั้น หากเปรียบเทียบกันสเตตัสต่อสเตตัสแล้ว สายไหนกันแน่ ที่จะทำให้เพื่อนๆ เก่งกว่าคนอื่น!!!

ทำการทดสอบโดย ตัวละครธาตุลม เลเวล 200

Status หลัง  คือค่า Status ที่ได้จากการใส่ยุทธภัณฑ์ต่างๆ เสริมเพิ่มเข้ามา



สำหรับค่า Status หลังนั้นผู้เล่นสามารถเพิ่มหรือลดได้ตามต้องการ เพราะฉะนั้นเรามาดูกันดีกว่าว่าค่า Status หน้า หรือ Status หลัง จะสำคัญกว่ากัน???

วิเคราะห์ จุดต่อจุด


จากภาพ การอัพ Status สาย INT เพียง 2 point นั้น ทำให้ Damage เพิ่มมาเพียงแค่ 3 point เท่านั้น ซึ่งทำให้เห็นว่ากว่าสาย INT จะเพิ่มพลังทำลายให้สูงขึ้นได้นั้น จำต้องใช้ระยะเวลาหรือเลเวลที่สูงกว่าสาย ATK



จากภาพด้านบน จะเห็นว่าไม่ว่าจะเพิ่มค่า Status หน้าหรือหลัง สาย INT สามารถทำ Damage ได้เท่าๆ กันเลยทีเดียว

สรุปเปรียบเทียบระหว่าง สาย ATK และสาย INT

จากภาพ ในกรณีที่ไม่มีค่า Status ใดๆ เลย สาย INT นั้น มีพลังทำลายมากกว่าสาย ATK อยู่มากทีเดียว



แต่เมื่อใส่อาวุธ (+50) เข้าไปแล้ว จะเห็นได้ว่าความรุนแรงที่ได้นั้น แตกต่างกันโดยสิ้นเชิง สาย ATK กลับทำ Damage ได้มากกว่าสาย INT อย่างเห็นได้ชัด

สรุป

สาย ATK  เน้นค่า Status หลังมากกว่า เพราะฉะนั้นหากผู้เล่นสามารถหาของ + ATK มาใส่ได้มากเท่าไหร่ ตัวละครก็สามารถมีพลังทำลายที่สูงมากขึ้นเท่านั้น ซึ่งในเวอร์ชั่นหลังๆ นี้เกม TS Online ก็ได้เพิ่มไอเท็มยุทธภัณฑ์ ที่เพิ่มค่า ATK ในส่วนที่ไม่เคยมีเข้ามาเช่น หมวก รองเท้า ชุดเกราะ เป็นต้น ทำให้สาย ATK นั้น ไม่ต้องน้อยใจว่าไม่มีของใส่ตัวที่ + ATK อีกต่อไป
สาย INT  ไม่ได้เน้นค่า Status ใดเป็นพิเศษ เมื่อดูโดยรวมอาจจะเห็นว่าสาย INT นั้นมีความด้อยกว่าในด้านของการทำลาย แต่ในเวอร์ชั่นใหม่ๆ นั้น ตัวเกมได้เพิ่มสกิลที่เพิ่มพลังทำลายให้สาย INT มากยิ่งขึ้น รวมถึงสกิลหมู่ต่างๆ ที่สามารถทำให้ผ่านด่าน หรือกิจกรรมต่างๆ ได้เร็วขึ้นกว่าสาย ATK อีกด้วย แต่แน่นอน ท่าสาย INT บางท่านั้น ไม่สามารถคอมโบได้
ทั้งสาย ATK และ INT นั้น ต่างก็มีข้อดี ข้อเสียต่างกันไป ไม่ว่าจะเลือกเล่นสายใด ก็สามารถเก่งได้ไม่ยาก หากมีทักษะการเล่นที่เข้าขากับคนอื่นๆ ในปาตี้ได้ดี จะเก่งบู๊ หรือบุ๋น ก็แล้วแต่เพื่อนๆ เลือกกันนะจ๊ะ  ^_^
