GTX Titan
การแข่งขันทำตลาดขายการ์ดจอในปี 2013 ดูท่าจะดุเดือดไม่น้อย เมื่อทาง AMD อุบการ์ดจอซีรี่ย์ HD8000 ยังไม่เปิดตัวออกมา แต่ในขณะเดียวกันทาง NVIDIA ได้เปิดการ์ดจอรุ่นใหม่ GTX Titan ยักษ์ใหญ่ออกมาสยบความแรงของทุกการ์ดจอที่ผ่านๆมาจนแบนติดดินไปหมด  แต่ก็ต้องแลกมาด้วยขนาดของการ์ดจอที่ใหญ่สมชื่อ โดยนอกจากจะแรงแล้ว ยังติดตั้งระบบระบายความร้อนที่มีประสิทธิภาพสูง และทำงานเงียบด้วย ภายในประกอบไปด้วย CUDA Core 2,688 ตัว รันที่ความเร็ว 837MHz แรม  GDDR5  6GB ที่ความกว้างแบนวิธ 384 บิต แต่กินไฟเพียง 250 วัตต์เท่านั้น ราคาก็แรงเวอร์ $999
 
ในเมื่อทาง NVIDIA เสนอการ์ดจอเทพมาแล้ว ก็เลยมีสนองด้วยการนำไปทดสอบพลังกันหน่อย ด้วยโปรแกรม Benchmark ต่างๆ โดยเฉพาะของเกม Crysis3 ที่ GTX Titan ทำออกมาเพื่อสยบความเทพของเกมนี้โดยเฉพาะ แต่ขั้นแรก เรามาลองดูภาพรายละเอียดของ GTX Titan เมื่อรัน Tools ปรับการ์ดจอกันก่อน ว่ามันมีคุณสมบัติเท่าไหร่ ซึ่ง Tools นี่สามารถปรับให้การ์ดจอทำงาน Overclock ได้แบบไม่มีปัญหา สามารถปรับ Hz ความถี่หน้าจอได้ และใช้ VSync ได้ไม่มีปัญหา แต่สำหรับการทดสอบในวันนี้ จะใช้ค่าเดิมๆที่ default settings ไม่มีการปรับเปลี่ยนใดๆ

เริ่มเทสกันที่โปรแกรม Unigine กันก่อน ซึ่งเป็นโปรแกรมทดสอบการ์ดจอ ในการสร้างภาพ 3D แบบสมจริง ซึ่งมีกราฟิกแบบต่างๆให้เลือกมากมาย ไม่ว่าจะเป็นฉากป่าเขา ฉากท้องฟ้า ฉากที่ขุดเจาะน้ำมันกลางทะเล ฯลฯ เราลองไปดูคะแนนของ GTX Titan หลังจากทดสอบฉากท้องฟ้า Heaven ก่อน ตามด้วยฉากป่าเขา Valley ที่เป็นฉากล่าสุด เฟรมเรตของฉากท้องฟ้า ทำคะแนนได้เป้นที่น่าพอใจมาก ส่วนฉาหกป่าเขาที่รายละเอียดค่อนข้างเยอะจะได้เฟรมเรตที่น้อยลงมานิด แต่คะแนนการทดสอบก็ยังน่าพอใจ

เกมส์ที่ทดสอบแล้วได้เฟรมเรตสูงสุดก็คือ Battlefield 3 จากการทดสอบโหมดเนื้อเรื่องนั้นการ์ดจอทั่วไปทำได้ประมาณ 30-50 เฟรมเรตต่อวินาที แต่ GTX Titan กลับทำได้สูงมาก มาดูทางเกมส์ Batman Arkham City ทำเฟรมเรตได้มาตรฐานไม่ต่ำหรือสูงเกินไป ส่วนหนึ่งเพราะมีการเปิดใช้ PhysX ด้วย ดังนั้นตัวเลขที่ออกมาจึงอยู่ในระดับโอเค ส่วนโคตรเกมอย่าง Crysis 3 ก็สามารถเล่นกราฟิกสูงได้โอเค แต่่ว่าทาง Far Cry 3 กลับทำเฟรมเรตได้ต่ำสุด ต่ำกว่าโคตรเกมกราฟิกอย่าง Crysis 3 อีก นั่นก็เพราะ Far Cry 3 มีฉากที่ค่อนข้างกว้าง เป็น Open-world ก็เลยทำเฟรมเรตได้น้อย แต่ก็สามารถเล่นได้ไม่มีปัญหาอะไร
 
บทสรุปการทดสอบ GTX Titan เป็นการ์ดจอประสิทธิภาพสูง ที่รองรับการเล่นเกมทุกเกมได้หมดในยุคนี้ แม้แต่ Crysis 3 ที่ว่าแน่ ก็เล่นได้สบายๆ แต่กับราคา $999 นี่ก็ถือว่าสูงเอาเรื่องไปหน่อยนะ
