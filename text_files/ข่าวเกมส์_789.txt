โครงการพัฒนาบ็อต Pokemon Go ทยอยปิดตัว หลัง Niantic เอาจริง
Pokemon Go
ท่าทีของ Niantic Labs จริงจังกับการโกงเกม Pokemon Go มากขึ้น หลังประกาศแบนผู้เล่นที่โกงเกม ก็มีข่าวว่าโครงการบ็อต Pokemon Go หลายโครงการยุติการพัฒนาแล้วเช่นกัน

PokeMobBot หน้าเว็บบน GitHub ถูกระงับการใช้งาน เพราะ GitHub ได้รับจดหมายแจ้งเตือนปัญหาละเมิดลิขสิทธิ์เกม
HaxtonBot ประกาศหยุดพัฒนา แต่ซอร์สโค้ดยังเข้าถึงได้จากหน้า GitHub
NecroBot ประกาศหยุดพัฒนา  เหตุผลคือทีมผู้สร้างเกรงจะเกิดปัญหาทางกฎหมายเหมือนกับที่โครงการอื่นเจอ  ตอนนี้ ไฟล์และซอร์สโค้ดทั้งหมดของ NecroBot ถูกเอาออกจากหน้าเพจ GitHub  แล้ว

ก่อนหน้านี้ Twitch เว็บสตรีมเกมชื่อดังก็ประกาศห้ามถ่ายทอดการเล่น  Pokemon Go แบบที่ใช้ซอฟต์แวร์โกงเกม เช่น ตัวหลอกพิกัด GPS  หรือเล่นแบบหลายบัญชี
