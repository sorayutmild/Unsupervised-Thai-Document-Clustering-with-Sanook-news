วิธีแก้ปัญหา แอพแต่งรูปการ์ตูน 魔漫相机 (แอพจีน) เซฟภาพบน Android ไม่ได้
ใช้ Android แต่เซฟภาพไม่ได้ ทำอย่างไร ?
เป็นปัญหาที่ผู้ใช้ Android บ่นกันเข้ามามากครับ อยากเล่น 魔漫相机 แต่ ไม่มีปุ่มให้เซฟภาพเหมือนบน iPhone ไม่รู้จะแก้ไขอย่างไร ตัดสินใจ save screenshot ไปก่อน แล้วค่อย crop เอาทีหลัง วันนี้ ทีมงาน techmoblog มีวิธีแก้ครับ

** อุปกรณ์ที่ใช้ทดสอบ : Samsung Galaxy Note 10.1 (2014 Edition)

อันดับแรก ให้เข้าไปที่ Play Store ค้นหาไฟล์ที่มีชื่อว่า ES File Explorer File Manager มาติดตั้งในเครื่องเสียก่อน ซึ่งแอพฯ นี้ ดาวน์โหลดมาใช้งานได้ฟรีครับ
ข้อมูลเพิ่มเติม ES File Explorer File Manager : https://play.google.com/store/apps/details?id=com.estrongs.android.pop&hl=th

เมื่อดาวน์โหลดเสร็จแล้ว ให้เปิดโปรแกรมขึ้นมาครับ จะพบกับโฟลเดอร์ที่มีชื่อว่า MoManCamera ให้คลิ๊กเข้าไป จากนั้นให้เลือกโฟลเดอร์ album

เมื่อคลิ๊กไปแล้ว จะเห็นว่า มีภาพอยู่ แต่ปัญหาก็คือ มองไม่เห็นภาพเหล่านั้น การแก้ปัญหาก็คือ ให้เปลี่ยนนามสกุลไฟล์ภาพ ด้วยการแตะที่ภาพๆ นั้นค้างไว้ และติ๊กเครื่องหมายถูกภาพที่ต้องการจะเปลี่ยนนามสกุล มองที่เมนูด้านล่าง เลือก เปลี่ยนชื่อ

เปลี่ยนจากนามสกุลเดิม .bshn เป็น .jpeg

จะเห็นได้ว่า ภาพที่ถูกเปลี่ยนนามสกุล ปรากฏรูปขึ้นมาแล้ว

จากนั้น คลิ๊กที่ไปภาพครับ จะมีเมนูปรากฏขึ้นมา ให้คลิ๊กที่ แกลลอรี่

ไปที่แอพฯ แกลอรี่ครับ จะเห็นว่า มีโฟลเดอร์ album (2) ปรากฏขึ้นมา ซึ่งข้างในจะเป็นภาพจาก แอพฯ ภาพการ์ตูน 魔漫相机 จากนั้น แชร์ผ่าน Facebook หรือ Twitter ได้ตามสะดวกครับ
อย่างไรก็ดี ข้อเสียของแอพฯ 魔漫相机 สำหรับคนไทยอย่างเราๆ ก็คือ เป็นภาษาจีนล้วนครับ แต่ก็ใช้งานได้ไม่ยาก เพราะทุกอย่างสามารถเดาได้หมด ลองดาวน์โหลดมาเล่นกันดูนะครับ
