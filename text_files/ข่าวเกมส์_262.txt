Project Cars
Project Cars เกมรถแข่งธรรมดาที่ไม่ธรรมดา เพราะเป็นเกมรถแข่งที่มีกราฟิกอลังการที่สุดในขณะนี้ ด้วยพลังของเอนจิ้น Madness ตัวใหม่จากทีมงาน Slightly Mad Studios (ผู้สร้าง Need for Speed Shift) ทำให้ Project Cars กลายเป็นเกมรถแข่งที่มีกราฟิกสมจริงที่สุด โดยไม่ใช่แค่ความสมจริงของโมเดลตัวรถเท่านั้น แต่ฉากสนามแข่ง เอฟเฟคหยดน้ำ ฝุ่นควัน พื้นดินพื้นหญ้า ทุกอย่างล้วนทำออกมาสวยสมจริง ราวกับภาพถ่ายจากโลกจริงเลยก็ว่าได้ ซึ่งเกม Project Cars ก็ปล่อยภาพตัวอย่างออกมาให้ดูกันบ่อยมาก แต่ครั้งนี้ดูจะสมจริงกว่าที่ผ่านๆมา ย้ำอีกครั้ง! ภาพที่เห็นนี้ไม่ใช่ของจริง! แค่เป็นภาพเกมรถแข่งเกมหนึ่ง
Project Cars
จุดมุ่งหมายของเกม Project CARS ทำออกมาเพื่อแข่งกับเกมจำลองการแข่งรถแบบสมจริงอย่าง Gran Turismo ของโซนี่  และ Forza Motorsport ของไมโครซอฟท์โดยเฉพาะ จนได้ภาพกราฟฟิกที่สวยงามอลังการ  ที่ไม่มีเกมรถแข่งที่ไหนทำได้ขนาดนี้มาก่อน และยังใช้ระบบฟิสิกส์ระดับ  "hardcore" ที่ให้ความสมจริงแบบสุดๆ  ทั้งเรื่องของการตอบสนองการขับขี่ของรถแข่งแต่ละรุ่น  รวมไปถึงการตอบสนองของสภาพแวดล้อมในสนามแข่งด้วย  โดยมีรถแข่งแบบต่างๆให้เลือกมากมาย ไม่ว่าจะเป็นรถสปอร์ท รถแรลลี่  รถสูตรหนึ่ง และรถโบราณต่างๆ รวมแล้วกว่า 60 รุ่น  และมีสนามแข่งจากสถานที่จริงอีกกว่า 35 สนามแข่ง ซึ่งมากกว่า Gran Turismo  ที่มีสนามแข่งเพียง 27 สนามเท่านั้น แถมยังเพิ่มความสมจริง  ด้วยการให้นักแข่งรถชื่อดังจากวงการต่างๆ ทั้งนักแข่ง Top Gear  นักแข่งแรลลี่ และนักแข่งรถสูตรหนึ่ง  มาร่วมงานทดสอบการขับในเกมด้วยว่าสมจริงหรือเปล่า
Project Cars หรือ Community Assisted Racing Simulator กำหนดออกในเครื่อง PS4, Xbox one และ PC Windows ในเดือนพฤศจิกายน 2014
