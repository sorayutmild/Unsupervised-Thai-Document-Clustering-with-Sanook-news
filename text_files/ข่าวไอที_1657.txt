     UnRead เป็น Android Application เพื่อใช้รองรับสถานการณ์ที่ไม่อยากให้คู่สนทนาเห็นว่าเราได้อ่านข้อความของเขาแล้ว อยากหลีกเลี่ยงการถูกกดดันให้ตอบคำถาม หรือ อยากอ่านข้อความแต่ยังไม่อยากตอบ โดย UnRead จะแสดงข้อความจาก Chat Application ต่างๆ เช่น LINE, facebook Messenger, WhatsApp, BeeTalk และอีกมากมาย โดยที่เราสามารถแอบอ่านข้อความจาก UnRead แทนการอ่านจาก Chat Application เหล่านี้โดยตรง




     การใช้งาน UnRead สะดวกเรียบง่าย เช่น เมื่อผู้ใช้งานได้รับการแจ้งเตือนจาก LINE ว่ามีข้อความใหม่ และเห็นเฉพาะข้อความสุดท้ายที่มีความน่าสงสัย เกิดความอยากอ่านแต่ไม่อยากให้เขารู้ว่าเราอ่าน ให้เปิด UnRead เพื่ออ่านข้อความทั้งหมดที่ได้รับแทนการเข้าไปอ่านจาก LINE โดยตรง ซึ่งจะไม่ทำให้สถานะของการอ่านข้อความใน LINE ถูกเปลี่ยนเป็น Read
     UnRead จะทำงานได้ต่อเมื่อผู้ใช้งานอนุญาตให้ UnRead สามารถเข้าถึงข้อมูลจาก Notification bar ดังนั้น ข้อความสนทนาภายใน UnRead จึงเป็นข้อความที่ Chat Application แจ้งเตือนเราผ่าน Notification bar เท่านั้น
     หากไม่พบข้อความใดๆ ต้องตรวจสอบว่า Chat Application ได้ตั้งค่าการแจ้งเตือนไว้ถูกต้องหรือไม่ หรือหาก UnRead ยังไม่รองรับ Chat Application ที่เราใช้งานอยู่ ก็สามารถแจ้งรายละเอียดมาที่ app.near@gmail.com เพื่อปรับปรุงได้
