ดูภาพเต็มๆของ GTX Titan X การ์ดจอใหญ่สุดแรงสุดจากค่ายเขียว
GTX Titan X
เทคโนโลยีการ์ดจอค่อนข้างเดินหน้าไปอย่างรวดเร็ว จากปัจจัยสำคัญที่มีผู้ใช้งานมากทั้งในมือถือและคอมพิวเตอร์ต่างๆ แถมยังมีการแข่งขันกันอย่างรุนแรงของสองบริษัทที่ไม่มีใครยอมใคร และล่าสุดในงาน GDC 2015 ทางค่ายเขียว nvidia ก็กลับมาทวงตำแหน่งสุดยอดการ์ดจอของโลกได้อีกครั้ง เมื่อเปิดตัว GTX Titan X การ์ดจอสุดแรงตัวใหม่ ที่เขาว่าทั้งแรงและความจุใหญ่ที่สุดในโลก คือมีเมโมรี่ถึง 12GB และมี transistors ถึง 8 พันล้านตัว
GTX Titan X เป็นการ์ดจอที่ทางค่ายเขียวทำมาเพื่อรองรับการเล่นเกมโดยเฉพาะ อย่างแรกเลยคือมันเป็นการ์ดจอเดี่ยว ไม่ใช่การ์ดจอคู่สองหัวเหมือนอย่าง Titan Z (dual gpu) เนื่องจากการ์ดจอชนิดเดี่ยวนั้นสามารถรองรับการใช้งานได้มากกว่าการ์ดจอสองหัว ที่บางเกมบางโปรแกรมไม่รองรับนั่นเอง โดยที่ว่าไม่รองรับคือบางเกมนั้นจะเปิดให้งานการ์ดจอแค่หัวเดียว ส่วนอีกหัวจะไม่ได้ร่วมทำงานด้วยนั่นเอง ดังนั้น GTX Titan X แม้จะมี GPU ตัวเดียวแต่ก็ทำงานได้เต็มประสิทธิภาพมากกว่า
GTX Titan X
GTX Titan X นับเป็นการ์ดจอที่มีความจุเมโมรีใหญ่ที่สุดตอนนี้ โดยมีชิป GDDR5 ถึง 12 ตัวรวมกันที่ 12GB มี CUDA Core มีสูงถึง 3072 Cores ขนาด 28 นาโนเมตร ที่ชื่อว่า GM200 จึงทำให้มันเป็นการ์ดจอที่ทั้งแรงและมีความจุขนาดใหญ่สุดในตัวเดียวกัน เห็นแบบนี้บางคนอาจจะคิดว่ามันกินไฟแหงๆ แต่ GTX Titan X ใช้สถาปัตยกรรม Maxwell ใส่เข้าไปด้วยทำให้มันกินไฟน้อยกว่าปกติ ซึ่งทาง Epic Games ได้ลองนำมันไปใช้เปิด Tech Demo ของ Unreal Engine 4 และทาง Oculus ก็ได้ลองเอาไปใช้เล่นเกมจากจอภาพเสมือน VR Headset ด้วย แถมยังปรับความละเอียดกราฟิกสูงสุดที่จอภาพ 4K ได้ด้วย จึงได้เสียงตอบรับที่ดีมาก
ในงาน GDC 2015 ไม่ได้ประกาศราคาของ GTX Titan X ไว้ว่าเท่าไหร่ ลือๆกันว่ามันอาจจะแพงถึงประมาณ 50,000 บาท กำหนดออกภายในปี 2015 นี้
